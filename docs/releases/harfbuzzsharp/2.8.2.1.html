API changes · HarfBuzzSharp API diff · Ships with SkiaSharp 2.88.1

Highlights

This release focuses on project consolidation and modern .NET support, with nullable annotations, newer target frameworks, and Apple-platform follow-up work.

Breaking Changes

None in this release.

New Features

  • Project consolidation — Merges the classic project layout, simplifying how HarfBuzzSharp packages are produced and maintained. (#2092)
  • Generator refresh — Updates the generator used to produce the interop surface. (#2087)
  • Nullable and newer TFMs — Adds nullable support and expands target framework coverage. (#2120)

Platform Support

  • Apple follow-up work — Implements the missing macios feature needed by this release line. (#2198)

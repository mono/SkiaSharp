# SkiaSharp Guides

Learn SkiaSharp with step-by-step tutorials. Start with the basics and work your way up to advanced effects.

## [Drawing Basics](basics/index.md)

Learn the fundamentals: canvases, paint objects, coordinates, animation, text, and bitmaps.

## [Lines and Paths](paths/index.md)

Draw lines, create paths, explore fill types, work with polylines, and add dashes and dots.

## [Transforms](transforms/index.md)

Translate, scale, rotate, and skew graphics. Work with matrix transforms and 3D rotations.

## [Curves and Paths](curves/index.md)

Draw arcs and Bézier curves, use SVG path data, clip with paths, and apply path effects.

## [Bitmaps](bitmaps/index.md)

Load, display, create, crop, save, and animate bitmaps. Access pixel data directly.

## [Effects](effects/index.md)

Apply shaders, blend modes, gradients, mask filters, image filters, and color filters.

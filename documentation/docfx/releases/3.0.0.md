The 3.0.0 preview line — a major-version overhaul aligned with .NET 8, Skia m116, and HarfBuzz 8.3.0 — that never shipped as a tagged stable. The full set of changes rolled forward into [3.116.0](3.116.0.md), which is the release consumers should target. The preview NuGets remain available for archival compatibility, but new work should reference the successor page.

Read the polished notes on [3.116.0](3.116.0.md) for the definitive changelog; this page keeps the preview-only history and links so the diff is auditable. [HarfBuzzSharp 8.3.0](harfbuzzsharp/8.3.0.md) is the paired native asset — refer to its page for shaping-side binding details.

## What was staged in the 3.0.0 previews

- Repository modernised and aligned with SkiaSharp 3.0 layout ([#2505](https://github.com/mono/SkiaSharp/pull/2505)).
- Removed every `[Obsolete]` member and type accumulated during 1.x/2.x ([#2539](https://github.com/mono/SkiaSharp/pull/2539)).
- Upgraded Skia to m116 ([#2829](https://github.com/mono/SkiaSharp/pull/2829)) with an interim m119 roll-forward from the community ([#3026](https://github.com/mono/SkiaSharp/pull/3026) ❤️).
- Upgraded HarfBuzz to 8.3.0 ([#2624](https://github.com/mono/SkiaSharp/pull/2624)); libjpeg-turbo to 3.0.0 ([#2583](https://github.com/mono/SkiaSharp/pull/2583)); libwebp to 1.3.2 ([#2622](https://github.com/mono/SkiaSharp/pull/2622)).
- Adopted .NET 8 as the primary managed target across every binding ([#2927](https://github.com/mono/SkiaSharp/pull/2927), [#2971](https://github.com/mono/SkiaSharp/pull/2971)); added .NET 9 support ([#3010](https://github.com/mono/SkiaSharp/pull/3010) ❤️).
- Modernised the P/Invoke layer with function pointers via `LibraryImport` on .NET 8 ([#2917](https://github.com/mono/SkiaSharp/pull/2917) ❤️).
- Added Metal as the default backend on Mac Catalyst ([#2747](https://github.com/mono/SkiaSharp/pull/2747)) and unified the Metal APIs across .NET TFMs ([#2788](https://github.com/mono/SkiaSharp/pull/2788) ❤️).
- Added WinUI accelerated views ([#2733](https://github.com/mono/SkiaSharp/pull/2733)), reintroduced `SKGLElement` for WPF ([#2317](https://github.com/mono/SkiaSharp/pull/2317) ❤️), and re-implemented `SKGLView` ([#2598](https://github.com/mono/SkiaSharp/pull/2598)).
- Exposed `SKBlender` ([#2830](https://github.com/mono/SkiaSharp/pull/2830)), `SKImage.ToRawShader` ([#2748](https://github.com/mono/SkiaSharp/pull/2748)), Skottie animation builder ([#2630](https://github.com/mono/SkiaSharp/pull/2630)), R-Tree `BeginRecording` overload ([#2889](https://github.com/mono/SkiaSharp/pull/2889)), additional `SKPicture` APIs ([#2883](https://github.com/mono/SkiaSharp/pull/2883)), color-filter span overloads ([#2879](https://github.com/mono/SkiaSharp/pull/2879)), and `SetRectRadii` span overload ([#2949](https://github.com/mono/SkiaSharp/pull/2949) ❤️).
- Trimmable `PlatformConfiguration` ([#2717](https://github.com/mono/SkiaSharp/pull/2717) ❤️), Blazor DPI exposure ([#1832](https://github.com/mono/SkiaSharp/pull/1832) ❤️), no more `async void` in `SKXamlCanvas` ([#2720](https://github.com/mono/SkiaSharp/pull/2720) ❤️), and null-safe `XamlRoot` handling ([#2854](https://github.com/mono/SkiaSharp/pull/2854) ❤️).
- Fixed the `GetKerningPairAdjustments` API ([#2858](https://github.com/mono/SkiaSharp/pull/2858) ❤️), MAUI view memory leaks ([#2955](https://github.com/mono/SkiaSharp/pull/2955)), cloned-`SKPaint` `AccessViolationException` ([#2904](https://github.com/mono/SkiaSharp/pull/2904) ❤️), `SKBitmap.Encode` with `SKPngEncoderOptions` regression ([#3014](https://github.com/mono/SkiaSharp/pull/3014) ❤️), Uno Wasm canvas-not-found handling ([#2563](https://github.com/mono/SkiaSharp/pull/2563) ❤️), and Snapshot-returning-magenta on Apple Metal ([#2804](https://github.com/mono/SkiaSharp/pull/2804) ❤️).

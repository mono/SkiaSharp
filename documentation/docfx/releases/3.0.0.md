# Version 3.0.0

> **Major 3.x milestone preview** · Preview only · Superseded by [3.116.0](3.116.0.md) · These changes later shipped in 3.116.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.0.0-preview) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.0.0-preview.5.4)

> **API changes** · [SkiaSharp API diff](3.0.0/index.md) · [HarfBuzzSharp 8.3.0](harfbuzzsharp/8.3.0.md)

## Highlights

SkiaSharp 3.0 marked the major reset for the library: a cleaned-up API surface, modernized platform support, broader GPU and view coverage, and a large wave of new bindings on the road to the first stable 3.x release. This preview train never shipped as 3.0.0 stable, but its work rolled directly into 3.116.0.

## Breaking Changes

- **Obsolete 2.x APIs were removed** — The 3.x line drops members and types that had already been marked obsolete, making this a real migration point for existing applications. ([#2539](https://github.com/mono/SkiaSharp/pull/2539))

## New Features

### Engine

- **Established the 3.x Skia foundation** — Moved the new line onto the refreshed Skia milestone 116 base and continued updating the externals during the preview cycle. ([#2829](https://github.com/mono/SkiaSharp/pull/2829), [#2885](https://github.com/mono/SkiaSharp/pull/2885), [#3026](https://github.com/mono/SkiaSharp/pull/3026))
- **Expanded core graphics bindings** — Added `SKBlender`, more `SKPicture` APIs, Skottie's Animation Builder, `SKImage.ToRawShader`, and `beginRecording` overloads with R-Tree support. ([#2830](https://github.com/mono/SkiaSharp/pull/2830), [#2883](https://github.com/mono/SkiaSharp/pull/2883), [#2630](https://github.com/mono/SkiaSharp/pull/2630), [#2748](https://github.com/mono/SkiaSharp/pull/2748), [#2889](https://github.com/mono/SkiaSharp/pull/2889))

### Views & UI

- **Added major new view controls** — Introduced `SKGLView`, accelerated WinUI views, and `SKGLElement` for WPF. ([#2598](https://github.com/mono/SkiaSharp/pull/2598), [#2733](https://github.com/mono/SkiaSharp/pull/2733), ❤️ [@gmurray81](https://github.com/gmurray81) [#2317](https://github.com/mono/SkiaSharp/pull/2317))
- **Broadened Apple and Metal support** — Adopted Metal on Mac Catalyst and aligned Metal APIs across target frameworks. ([#2747](https://github.com/mono/SkiaSharp/pull/2747), ❤️ [@maxkatz6](https://github.com/maxkatz6) [#2788](https://github.com/mono/SkiaSharp/pull/2788))
- **Improved Blazor and XAML views** — Exposed Blazor component DPI and fixed several Uno, WASM, and XAML rendering and unload edge cases. (❤️ [@beto-rodriguez](https://github.com/beto-rodriguez) [#1832](https://github.com/mono/SkiaSharp/pull/1832), ❤️ [@jeromelaban](https://github.com/jeromelaban) [#2559](https://github.com/mono/SkiaSharp/pull/2559) [#2563](https://github.com/mono/SkiaSharp/pull/2563) [#2854](https://github.com/mono/SkiaSharp/pull/2854), ❤️ [@Youssef1313](https://github.com/Youssef1313) [#2918](https://github.com/mono/SkiaSharp/pull/2918))

### API Surface

- **Added migration-friendly overloads and spans** — Added `GetPixelSpan` with offsets, `SetRectRadii` span overloads, compatibility overloads for `SKFilterQuality`, and span-based color filter APIs. ([#2609](https://github.com/mono/SkiaSharp/pull/2609), ❤️ [@Youssef1313](https://github.com/Youssef1313) [#2949](https://github.com/mono/SkiaSharp/pull/2949) [#2963](https://github.com/mono/SkiaSharp/pull/2963), [#2879](https://github.com/mono/SkiaSharp/pull/2879))
- **Modernized native interop** — Adopted function pointers and `LibraryImport` for the binding layer. ❤️ [@maxkatz6](https://github.com/maxkatz6) ([#2917](https://github.com/mono/SkiaSharp/pull/2917))

### Platform

- **Updated the platform baseline for 3.x** — Added .NET 9 support, switched WinForms to the new GLControl packages, and refreshed ANGLE and workload packaging for modern SDKs. (❤️ [@jeromelaban](https://github.com/jeromelaban) [#3010](https://github.com/mono/SkiaSharp/pull/3010), [#2989](https://github.com/mono/SkiaSharp/pull/2989), [#2950](https://github.com/mono/SkiaSharp/pull/2950), [#2970](https://github.com/mono/SkiaSharp/pull/2970))

## Bug Fixes

- **Fixed memory and encoding regressions** — Resolved MAUI view leaks and restored the earlier `SKBitmap.Encode` behavior for `SKPngEncoderOptions`. ([#2955](https://github.com/mono/SkiaSharp/pull/2955), ❤️ [@sungaila](https://github.com/sungaila) [#3014](https://github.com/mono/SkiaSharp/pull/3014))
- **Improved rendering correctness** — Fixed Apple Metal snapshot color output, cloned `SKPaint` disposal crashes, `GetKerningPairAdjustments`, and background stretching behavior. (❤️ [@taublast](https://github.com/taublast) [#2804](https://github.com/mono/SkiaSharp/pull/2804), ❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) [#2904](https://github.com/mono/SkiaSharp/pull/2904), ❤️ [@pdjonov](https://github.com/pdjonov) [#2858](https://github.com/mono/SkiaSharp/pull/2858), [#2922](https://github.com/mono/SkiaSharp/pull/2922))

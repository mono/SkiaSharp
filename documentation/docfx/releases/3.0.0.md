# Version 3.0.0

> **Preview only** · Superseded by [3.116.0](3.116.0.md) · Never released as stable — these changes rolled up into 3.116.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.0.0-preview)

> **API changes** · [SkiaSharp API diff](3.0.0/index.md) · [HarfBuzzSharp 8.3.0](harfbuzzsharp/8.3.0.md)

## Highlights

The 3.0 preview series is the modernization line that laid the foundation for SkiaSharp 3.x — moving to .NET 8 (and later .NET 9), migrating the P/Invoke layer to source-generated `LibraryImport` and unmanaged function pointers, adding WinUI accelerated views, `SKGLView` for .NET MAUI, Skottie's Animation Builder, `SKBlender`, `SKImage.ToRawShader`, and Metal on Mac Catalyst. HarfBuzz was bumped from 2.8.2 → 7.3.0 → 8.3.0. This tag ships **preview only** — 3.116.0 is the first stable release on this line, and every 3.0 preview change is rolled into it. HarfBuzzSharp 8.3.0 ships alongside this release — see [HarfBuzzSharp 8.3.0](harfbuzzsharp/8.3.0.md) for its binding details.

## Breaking Changes

3.0 is a major-version rollup and removes deprecated 2.x API. Downstream projects moving from 2.88.x should read this section carefully.

- **All `[Obsolete]` members and types removed** — Everything marked `[Obsolete]` in the 2.x line is gone in 3.x. Callers should follow the migration guidance embedded in the individual obsolete annotations (still visible in the [2.88.x source](https://github.com/mono/SkiaSharp/tree/release/2.88.9)). Notable removals include the older `GRContext.Create` overloads (use `CreateGl(...)`), `GRGlInterface.Assemble*Interface` factories (use `CreateAngle`/`CreateOpenGl`/`CreateGles`/`Create`), `SKBitmap.GetAddr*` / `InstallMaskPixels` / `SKColorTable`-based overloads, and the `SKImage.FromPicture` overloads that took `SKMatrix`/`SKPaint` without color-space arguments.
- **`SKFilterQuality` replaced by `SKSamplingOptions`** — The `SKFilterQuality`-based paint property and drawing overloads are gone; use `SKSamplingOptions` overloads on the various draw calls. Compatibility overloads have been added and removed across previews — the final surface uses `SKSamplingOptions` throughout.
- **Assembly version bumped to 3.0** across the entire product — Binding redirects and package references pinned to `2.x` must be updated.

See the [full API diff](3.0.0/index.md) for the exact removed/renamed signatures across every assembly.

## Preview 5 · 3.0.0 (2024-10-24)

Adds .NET 9 support, migrates the P/Invoke layer to `LibraryImport` and function pointers, moves to .NET 8, plugs MAUI-view leaks, reverts the `SKPngEncoderOptions` behavior to match 2.88.8, and adds `Span` + `SKFilterQuality` compatibility overloads. See the [compare](https://github.com/mono/SkiaSharp/compare/v3.0.0-preview.4.1...v3.0.0-preview.5.4).

## Preview 4 · 3.0.0 (2024-07-18)

Updates Skia to m116, adds `SKBlender`, `SKPicture` R-Tree recording, more `SKPicture` APIs, span-based color-filter APIs, fixes `GetKerningPairAdjustments`, and cleans up `SKTextBlobBuilder` / `SKRunBuffer`. See the [compare](https://github.com/mono/SkiaSharp/compare/v3.0.0-preview.3.1...v3.0.0-preview.4.1).

## Preview 3 · 3.0.0 (2024-04-09)

Adds `SKGLElement` for WPF, brings Metal APIs to a common .NET TFM, exposes DPI on Blazor components, and fixes Snapshot magenta on Apple Metal. Introduces `SKFilterQuality` compatibility APIs. See the [compare](https://github.com/mono/SkiaSharp/compare/v3.0.0-preview.2.1...v3.0.0-preview.3.1).

## Preview 2 · 3.0.0 (2024-02-20)

Binds Skottie's Animation Builder, adds `SKImage.ToRawShader`, uses Metal on Mac Catalyst, and cleans up runtime-only package layout. See the [compare](https://github.com/mono/SkiaSharp/compare/v3.0.0-preview.1.8...v3.0.0-preview.2.1).

## Preview 1 · 3.0.0 (2024-02-06)

The foundation preview — modernizes the repository, removes every `[Obsolete]` member and type, bumps HarfBuzz to 7.3.0 → 8.3.0, updates webp / libjpeg-turbo, adds `SKGLView` for MAUI, WinUI accelerated views, `GetPixelSpan` with offsets, and brings back text-based blob overloads. Also lays out APIScan, dev containers, and the human-readable changelog. See the [compare](https://github.com/mono/SkiaSharp/compare/v2.88.9...v3.0.0-preview.1.8).

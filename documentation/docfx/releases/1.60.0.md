## Highlights

Servicing release for the 1.60 line with no PRs recorded in the diff. This bump retired the obsolete `GRContext.Flush(GRContextFlushBits)` overload and the last of the batching-related `GRContextOptions` tuning knobs. It ships alongside **[HarfBuzzSharp 1.4.6](harfbuzzsharp/1.4.6.md)** — see that page for its binding surface.

## Breaking Changes

- **`GRContext.Flush(GRContextFlushBits flagsBitfield)`** — The obsolete flags overload was removed. Use the parameterless `Flush()`.
- **`GRContextOptions` — removed properties**: `MaxOpCombineLookahead`, `MaxOpCombineLookback`. The current GPU backend tunes op combination internally.
- **`GRContextOptionsGpuPathRenderers`** — The type surface was tightened; recompile after auditing any explicit references.

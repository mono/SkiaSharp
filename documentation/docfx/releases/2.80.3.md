# Version 2.80.3

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.80.3)

> **API changes** · [SkiaSharp API diff](2.80.3/index.md) · [HarfBuzzSharp 2.6.1.8](harfbuzzsharp/2.6.1.8.md)

## Highlights

Big feature release: brings `SKRuntimeEffect` (SkSL runtime shaders) to .NET, adds first-class Metal APIs for macOS and iOS, ships the WinUI-for-Desktop backend, moves to .NET 5.0, and adds Uno Platform `SKXamlCanvas` support. Also adds ARM64 build target, Musl (Alpine) proper detection, .NET Interactive support, WASM emscripten build sweep, and `SKPicture` serialization bindings. Ships alongside **[HarfBuzzSharp 2.6.1.8](harfbuzzsharp/2.6.1.8.md)** — see that page for its binding details.

Community contributions from [@gmurray81](https://github.com/gmurray81) (GL surface creation failure workaround), [@Mikolaytis](https://github.com/Mikolaytis) (ANGLE `GRGlInterface` factory), [@HarlanHugh](https://github.com/HarlanHugh) (`SKElement.OnRender` PresentationSource guard), and [@jeromelaban](https://github.com/jeromelaban) (Uno Platform docs references).

## Breaking Changes

- **UWP touch-event delegate shape** — The UWP `SKXamlCanvas` touch event's delegate shape changed to match the Uno Platform backend. Update any `+= (s,e) => ...` subscriptions that named the delegate type explicitly. See the [SkiaSharp.Views.Uno API diff](2.80.3/SkiaSharp.Views.Uno/SkiaSharp.Views.UWP.breaking.md) for the exact signature.

See the [full API diff](2.80.3/index.md) for the complete list.

## Engine

- **`SKRuntimeEffect` (SkSL runtime shaders)** — Brings SkSL runtime-effect shaders to .NET. Compile shaders at runtime and drive them with dynamic uniforms. ([#1604](https://github.com/mono/SkiaSharp/pull/1604))
- **`SKPicture` serialization** — Bindings for the picture-recording serialization surface. ([#1514](https://github.com/mono/SkiaSharp/pull/1514))
- **`IsAbandoned` exposed on `GRContext`** ([#1659](https://github.com/mono/SkiaSharp/pull/1659))
- **`SKMatrix.IsInvertible` no-crash guard** ([#1527](https://github.com/mono/SkiaSharp/pull/1527))
- **`GRContextOptions` fully exposed** ([#1529](https://github.com/mono/SkiaSharp/pull/1529))

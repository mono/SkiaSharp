# Version 2.80.3

## Highlights

SkiaSharp 2.80.3 rolls up 55 PRs across five previews between 2.80.2 and stable. This drop lights up Metal on macOS/iOS, adds the `SKRuntimeEffect` API, brings first-class .NET 5, Mac Catalyst, iOS device testing, and .NET Interactive support, splits packages into per-runtime native asset packages, and finally supports Uno Platform on the SkiaSharp backends v2.0 as well as WinUI for Desktop. HarfBuzzSharp 2.6.1.8 ships alongside this release; see the [linked page](harfbuzzsharp/2.6.1.8.md) for its binding details.

## Breaking Changes

The Uno Platform integration changed a signature on the UWP-side view. See the [breaking-changes diff](2.80.3/SkiaSharp.Views.Uno/SkiaSharp.Views.UWP.breaking.md) for details.

# Version 3.119.5

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

The 3.119.x servicing branch has one queued product fix: the WinUI projection
assembly now resolves correctly on .NET 9. The rest is CI plumbing to keep the
branch building on newer agents.

## Breaking Changes

None in this release.

## Bug Fixes

- **WinUI projection DLL now loads on .NET 9** — .NET 9 consumers of the WinUI
  interop package were hitting a `DllNotFoundException` because the projection
  assembly was not being resolved. The load path is now correct.
  ([#4086](https://github.com/mono/SkiaSharp/pull/4086))

Plus various CI, documentation, and internal tooling improvements.

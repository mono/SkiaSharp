**3.119.5** is a servicing update in preparation for the `release/3.119.x` branch. The main user-facing item so far restores `SkiaSharp.Views.WinUI` for .NET 9 consumers by fixing a missing WinUI projection assembly load ([#4086](https://github.com/mono/SkiaSharp/pull/4086)). CI-side fixes for ANGLE downloads, the WASM 2.0.6 toolchain, the Tizen SDK and Alpine images have been folded in on the same branch so future patches build cleanly.

- **WinUI projection load** — Fix `SkiaSharp.Views.WinUI` failing to load its projection DLL on .NET 9, so WinUI 3 hosts start correctly again ([#4086](https://github.com/mono/SkiaSharp/pull/4086)).

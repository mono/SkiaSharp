# Version 4.148.0

> **Upcoming release** · In development · Not yet available on NuGet

> **Supersedes 4.147.0** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Version 4.148.0 rolls up all preview work from [4.147.0](4.147.0.md) — the SkiaSharp 4.x major version upgrade, Skia m147 engine update, variable and color font support, new API surface additions, expanded platform coverage, and gallery improvements — which shipped as previews only and was never released as stable. On top of that cumulative base, this in-progress release adds a cherry-pick from [@ramezgerges](https://github.com/ramezgerges) that promotes the remaining pre-v4 obsolete members to errors, finalizing the v4 API contract. HarfBuzzSharp [14.2.0](harfbuzzsharp/14.2.0.md) ships alongside.

## Breaking Changes

- **Pre-v4 obsoletes promoted to errors** — The remaining members marked `[Obsolete]` during the v3→v4 migration are now compile-time errors. Migrate all call sites to the v4 replacement APIs before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4205](https://github.com/mono/SkiaSharp/pull/4205))

# Version 4.148.0

> **Upcoming release** · In development · Not yet available on NuGet

> **Supersedes 4.147.0** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Version 4.148.0 delivers the final v4 API cleanup: the last remaining pre-v4 obsolete APIs are promoted to compile-time errors and the associated obsolete enums removed from the reference assembly, completing the v4 deprecation lifecycle. All 4.147.0 preview work is rolled up into this release. HarfBuzzSharp 14.2.0 ships alongside this release.

### ⚠️ Breaking Changes

- **Pre-v4 obsoletes promoted to errors** — APIs that were deprecated before v4 and previously issued `[Obsolete]` warnings now emit compile-time errors. Projects that have not yet migrated to v4 equivalents must do so before upgrading. ([#4205](https://github.com/mono/SkiaSharp/pull/4205))
- **Obsolete enums trimmed from reference assembly** — Enum members marked obsolete in earlier releases have been removed from the reference assembly. Any source code referencing these members directly will fail to build. ([#4205](https://github.com/mono/SkiaSharp/pull/4205))

## API Surface

- **v4 deprecation lifecycle finalized** — The remaining pre-v4 `[Obsolete]` escalations are promoted to errors and the associated obsolete enum values removed, delivering a clean v4 public API surface. ([#4205](https://github.com/mono/SkiaSharp/pull/4205))

## Highlights

SkiaSharp 1.68.2 rolls up 71 PRs across seven previews between 1.68.1 and stable. This drop focuses on hardening the views (concurrency, disposal, GTK for Xamarin.Forms, WPF Mouse-Wheel on touch, UWP GL/SwapChain fixes), broadening the surface (`SKColorF` with `SKColorSpace`, more `SKImage`/`SKCanvas`/`SKRegion` members, Improved-Perlin `SKShader`), and modernizing the toolchain (VS 2019 + .NET Core 3.0, Docker-based Linux builds, vcpkg-built ANGLE, updated Cake/expat). HarfBuzzSharp 2.6.1.2 ships alongside this release; see the [linked page](harfbuzzsharp/2.6.1.2.md) for its binding details.

## Breaking Changes

Method renames aligning with .NET conventions and follow-up API cleanup during this cycle changed several signatures. See the [breaking-changes diff](1.68.2/SkiaSharp/SkiaSharp.breaking.md) and the [SkiaSharp.Views.Forms diff](1.68.2/SkiaSharp.Views.Forms/SkiaSharp.Views.Forms.breaking.md) for the exact list.

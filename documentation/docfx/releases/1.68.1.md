# Version 1.68.1

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.68.1)

> **API changes** · [SkiaSharp API diff](1.68.1/index.md) · [HarfBuzzSharp 2.6.1](harfbuzzsharp/2.6.1.md)

## Highlights

The 1.68.1 stable release consolidates the Azure DevOps pipeline rollout, splits the desktop packages, adds a WPF backend for Xamarin.Forms, ships the first internal P/Invoke generator, and picks up `Span<T>`-friendly overloads plus a handful of memory-lifetime fixes. Nine community contributions land here, including `SKPictureShader` bindings from [@alexandrvslv](https://github.com/alexandrvslv) and multiple HarfBuzz updates from [@Gillibald](https://github.com/Gillibald). HarfBuzzSharp 2.6.1 co-ships — see the HarfBuzz link on the API changes line above for its per-line page.

## Breaking Changes

The desktop-package split (`SkiaSharp.Views.Desktop`) means some assemblies now live under different package names. See the linked SkiaSharp and Views.Desktop breaking-changes files for the exact signature deltas.

## New Features

- **`SKPictureShader` bindings** — Adds the missing shader factory. ❤️ [@alexandrvslv](https://github.com/alexandrvslv) ([#973](https://github.com/mono/SkiaSharp/pull/973))
- **`SKTypeface.IsFixedPitch`** — Small but long-requested. ([#1019](https://github.com/mono/SkiaSharp/pull/1019))
- **`int` unicode overloads** — Makes several APIs less verbose to call. ❤️ [@Gillibald](https://github.com/Gillibald) ([#972](https://github.com/mono/SkiaSharp/pull/972))
- **`Span<T>` overloads for data, bitmap data, and text blobs** — Reduces allocations in hot paths. ([#865](https://github.com/mono/SkiaSharp/pull/865))
- **HarfBuzzSharp font funcs** — Fills in the previously-missing font-funcs API. ❤️ [@Gillibald](https://github.com/Gillibald) ([#881](https://github.com/mono/SkiaSharp/pull/881))
- **HarfBuzz face-loading by table data** — Load a face from a raw table blob. ([#862](https://github.com/mono/SkiaSharp/pull/862))

## Engine & Native

- **Re-worked the managed-native type layer** — Shared plumbing between wrappers and their handles. ([#900](https://github.com/mono/SkiaSharp/pull/900))
- **Bumped native HarfBuzz** — v2.5.2, v2.5.3, then v2.6.1 across the RC cycle. ❤️ [@Gillibald](https://github.com/Gillibald) ([#872](https://github.com/mono/SkiaSharp/pull/872), [#904](https://github.com/mono/SkiaSharp/pull/904), [#929](https://github.com/mono/SkiaSharp/pull/929))
- **Bitcode for watchOS and tvOS** ([#971](https://github.com/mono/SkiaSharp/pull/971))
- **`arm64_32` for watchOS** ([#967](https://github.com/mono/SkiaSharp/pull/967))
- **Clang on Windows for m68** ([#1007](https://github.com/mono/SkiaSharp/pull/1007))
- **Linux export hiding** — Hides dependency exports from the linker script. ❤️ [@Gillibald](https://github.com/Gillibald) ([#984](https://github.com/mono/SkiaSharp/pull/984))
- **Build without fontconfig** — Optional for constrained Linux targets. ([#821](https://github.com/mono/SkiaSharp/pull/821))

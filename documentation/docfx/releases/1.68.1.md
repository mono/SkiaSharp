## Highlights

SkiaSharp 1.68.1 is a large rollup off 1.68.0 (66 PRs across six release-candidate cuts). Highlights include a refactor of the managed-native type layer, `Span<T>` support for pixel/bitmap/text-blob data, native artifacts published publicly, the new WPF backend for Xamarin.Forms, HarfBuzzSharp updates through 2.6.1 with font-funcs support, macOS/iOS build modernization (bitcode for watchOS/tvOS, arm64_32 watchOS), and a P/Invoke generator scaffolding that reduces hand-written interop. HarfBuzzSharp 2.6.1 ships alongside this release; see the [linked page](harfbuzzsharp/2.6.1.md) for its binding details.

## Breaking Changes

The managed↔native type rework and API cleanup during this cycle changed several signatures. See the [breaking-changes diff](1.68.1/SkiaSharp/SkiaSharp.breaking.md) and the [SkiaSharp.Views.Desktop diff](1.68.1/SkiaSharp.Views/SkiaSharp.Views.Desktop.breaking.md) for the exact list.

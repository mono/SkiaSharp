# Version 1.68.1

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.68.1)

> **API changes** · [SkiaSharp API diff](1.68.1/index.md) · [HarfBuzzSharp 2.6.1](harfbuzzsharp/2.6.1.md)

## Highlights

Servicing rev on the 1.68 line. Absorbs the Azure DevOps pipeline migration, NuGet signing, Xamarin.Forms WPF backend, and a large managed-native type rework. `Span<T>` support lands for data / bitmap / text-blob paths. Ships alongside [HarfBuzzSharp 2.6.1](harfbuzzsharp/2.6.1.md).

- 🎨 **Core** — managed-native type layer reworked; `Span<T>` overloads for `SKData` / bitmap data / `SKTextBlob`; `SKPictureShader` binding (❤️ @alexandrvslv); `SKTypeface.IsFixedPitch` binding; memory-leak fix in `SKRoundRect`; some color spaces now returned as references (avoids double-free).
- 🍎 **Apple** — bitcode enabled for watchOS and tvOS; `arm64_32` watchOS build; iOS canvas fix + `*.dylib` bundling when running against full framework.
- 🪟 **Windows** — Clang used for Windows builds (m68 shim); UWP GPU view stability.
- 🐧 **Linux** — export-hiding version script for dependencies (❤️ @Gillibald); optional fontconfig-free build.
- 🌐 **Xamarin.Forms** — WPF backend added; desktop packages split.
- 🏗️ **Build / CI** — Azure DevOps pipeline migration, NuGet signing, iOS/macOS signing certs, public artifact publishing, and a new P/Invoke generator.
- 📦 **General** — docs moved to SkiaSharp-API-docs, various sample and template refreshes.

## Breaking Changes

- **SkiaSharp**
  - **SKColorSpace.CreateRgb** — default value for the `SKColorSpaceFlags flags` parameter was removed; pass it explicitly.
  - **SKObject.OwnsHandle** — the get-set property was removed; ownership is now handled internally. Code that flipped ownership manually needs to reshape.
  - **SKPath.Iterator / SKPath.RawIterator / SKPath.OpBuilder** — the nested classes now derive directly from `SKObject` instead of `SKNativeObject`. Public members are unchanged; only code that referenced the base type explicitly must switch.
  - **SKTextBlobBuilder.AddPositionedRun / AddHorizontalRun / AddRun** — the `utf8Text` parameter was renamed to `text`. Named-argument calls need to update.
- **SkiaSharp.Views.Desktop.Common** — `SKImageSourceHandler.LoadImageAsync(IImageSource imageSource, CancellationToken cancellationToken)` overload was removed; use the `IImageSource` + `CancellationToken` overload that returns `Task<UIImage>` (Tizen host).

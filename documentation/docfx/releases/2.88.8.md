# Version 2.88.8

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.88.8)

> **API changes** · [SkiaSharp API diff](2.88.8/index.md) · [HarfBuzzSharp 7.3.0.2](harfbuzzsharp/7.3.0.2.md)

## Highlights

Maintenance patch on the 2.88 line consisting of a handful of back-ported fixes — new overloads for compatibility with newer callers, invalid `extern` cleanup, an old-Alpine-friendly `gn` pin, and CI/compliance YAML tidying. HarfBuzzSharp 7.3.0.2 ships alongside this release — see [HarfBuzzSharp 7.3.0.2](harfbuzzsharp/7.3.0.2.md) for its binding details.

## Breaking Changes

- **Default parameter values dropped on `SKImageFilter` factories** — Every `SKImageFilter.Create…` factory that previously had optional `SKImageFilter input = NULL` and `SKImageFilter.CropRect cropRect = NULL` arguments now requires them to be passed explicitly. Existing code compiled against the previous package continues to work at runtime; recompiling will surface the missing arguments — pass `null` to keep the previous behaviour.

## API Surface

- **New overloads for better compat** ([#2810](https://github.com/mono/SkiaSharp/pull/2810) by @mattleibow) — Adds a handful of missing overloads back-ported from newer branches so 2.x consumers can share more code with newer callers.

## Bug Fixes

- **Do not provide invalid `extern`s** ([#2710](https://github.com/mono/SkiaSharp/pull/2710) by @mattleibow) — Removes P/Invoke entries that resolved to symbols missing from the current native library.

## Platform

- **Old-Alpine-compatible `gn`** ([#2712](https://github.com/mono/SkiaSharp/pull/2712) via automation of [#2711](https://github.com/mono/SkiaSharp/pull/2711)) — Pins to a version of Skia's `gn` that still builds on the older Alpine base image used for release Linux binaries.

## Lifecycle & Internals

- **APIScan and compliance YAML cleanup** ([#2805](https://github.com/mono/SkiaSharp/pull/2805), [#2791](https://github.com/mono/SkiaSharp/pull/2791), [#2792](https://github.com/mono/SkiaSharp/pull/2792) by @mattleibow) — Trims obsolete pipeline definitions and fixes broken YAML.

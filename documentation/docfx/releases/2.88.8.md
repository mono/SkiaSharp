# Version 2.88.8

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.88.8)

> **API changes** · [SkiaSharp API diff](2.88.8/index.md) · [HarfBuzzSharp 7.3.0.2](harfbuzzsharp/7.3.0.2.md)

## Highlights

A small stable servicing release on the 2.88.x line, cherry-picking a handful of `SKPaint`/AoT-friendly compatibility overloads and cleanup PRs back from the 3.0 preview work. Three community contributions land here alongside the compat additions. HarfBuzzSharp 7.3.0.2 also ships — see the HarfBuzz link on the API changes line above for its per-line page.

## Breaking Changes

*None in this release.*

## API Surface

- **Additional compatibility overloads on `SKPaint`** — Adds overloads that make it easier to keep 2.88.x apps buildable. ([#2810](https://github.com/mono/SkiaSharp/pull/2810))
- **`PlatformConfiguration` properties marked trimmable** — Back-ported to 2.88.x. ❤️ community back-port ([#2734](https://github.com/mono/SkiaSharp/pull/2734))

## Bug Fixes

- **Removed `async void` from `SKXamlCanvas`** — Back-ports the safer synchronous entrypoint. ❤️ community back-port ([#2731](https://github.com/mono/SkiaSharp/pull/2731))
- **Removed invalid externs** — Fixes a linker issue on some configurations. ([#2710](https://github.com/mono/SkiaSharp/pull/2710))

## Build & CI

- **Cleaned up APIScan and other compliance YAML** ([#2805](https://github.com/mono/SkiaSharp/pull/2805))
- **Native `gn` version fix for older Alpine** — Uses the `gn` version that works on the older Alpine build image. ❤️ community back-port ([#2712](https://github.com/mono/SkiaSharp/pull/2712))
- **Mac build script fix** — Back-ports the corrected script. ❤️ community back-port ([#2783](https://github.com/mono/SkiaSharp/pull/2783))

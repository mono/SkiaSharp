HarfBuzzSharp **8.3.1.1** ships alongside SkiaSharp 3.119.0 and rolls up the HarfBuzz-touching platform-support work from the skipped 3.118.0 preview cycle: LoongArch64 and RISC-V build support, ARM-target fixes, and the Windows Blazor/WASM binding relocations.

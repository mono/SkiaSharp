## Highlights

HarfBuzzSharp **8.3.0** is the first HarfBuzz binding shipping alongside SkiaSharp 3.0, bringing the underlying HarfBuzz version to 7.3.0 and modernizing the binding for the new .NET 8 baseline. Function-pointer P/Invoke plumbing lands in this cycle, and legacy `[Obsolete]` surface is finally removed.

## Breaking Changes

This release includes breaking API signature changes. See the [HarfBuzzSharp API diff](8.3.0/index.md) for the full list. Consumers should review flagged signatures before upgrading.

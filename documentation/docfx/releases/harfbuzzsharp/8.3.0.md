## Highlights

The first HarfBuzzSharp release on the modern SkiaSharp 3 line. It bundles HarfBuzz 7.3.0, drops all previously deprecated members, and modernises the build/packaging so HarfBuzzSharp ships from the same CI as SkiaSharp 3.0.0.

## Breaking Changes

- **All `[Obsolete]` HarfBuzzSharp members and types are removed.** Consumers still using deprecated APIs must migrate to the current surface before upgrading. See the [API diff](8.3.0/index.md) for the exact list.

## Bindings

- **Update bundled HarfBuzz to 7.3.0.** ❤️ [@maxkatz6](https://github.com/maxkatz6) (via [#2582](https://github.com/mono/SkiaSharp/pull/2582))

Plus various CI, documentation, and internal tooling improvements.

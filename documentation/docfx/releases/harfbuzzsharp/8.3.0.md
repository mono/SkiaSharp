# HarfBuzzSharp 8.3.0

## Highlights

HarfBuzzSharp companion for SkiaSharp 3.0.0. The HarfBuzz-touching work in the 3.0 line updates the native library to HarfBuzz 7.3.0, migrates the interop layer to .NET 8 function pointers and `LibraryImport`, and clears out the pre-3.0 obsoletes. Community contribution: [@maxkatz6](https://github.com/maxkatz6) drove the function-pointer/`LibraryImport` migration.

## Breaking Changes

- **`Blob.AsSpan()` removed** — The `ReadOnlySpan<byte>` accessor was removed from `HarfBuzzSharp.Blob`. Use the `Data`/`Length` pair or the alternative APIs surfaced in the 3.0 cleanup.
- **Obsolete members removed** — All `[Obsolete]` members and types were removed as part of the 3.0 modernization. Move to the recommended replacements flagged by the earlier obsolete warnings.

## Engine

- **HarfBuzz 7.3.0** — Bumps the native HarfBuzz to 7.3.0, picking up upstream shaping and font-table fixes. ([#2582](https://github.com/mono/SkiaSharp/pull/2582))

## Interop & Build

- **Function pointers and `LibraryImport`** — Migrates the interop layer to .NET 8-native function pointers and the source-generated `LibraryImport` marshaling. ❤️ [@maxkatz6](https://github.com/maxkatz6) ([#2917](https://github.com/mono/SkiaSharp/pull/2917))
- **.NET 8 baseline** — Adopts .NET 8 as the primary SDK, workload, and project target for HarfBuzzSharp. ([#2927](https://github.com/mono/SkiaSharp/pull/2927), [#2971](https://github.com/mono/SkiaSharp/pull/2971))
- **Repository modernization for 3.0** — Restructures the repo layout and build scripts as part of the 3.0 preparation. ([#2505](https://github.com/mono/SkiaSharp/pull/2505))
- **Removed `[Obsolete]` members and types** ([#2539](https://github.com/mono/SkiaSharp/pull/2539))

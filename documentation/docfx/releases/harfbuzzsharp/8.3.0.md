## Highlights

Ships as part of the SkiaSharp 3.0.0 preview line. The binding jumps to harfbuzz 7.3.0 as its baseline, drops the `[Obsolete]` API surface accumulated over the 1.x/2.x lines, and moves to function-pointer / `LibraryImport` marshalling.

- 🎨 **Core** — harfbuzz updated to 7.3.0 (see [mono/skia#104](https://github.com/mono/skia/pull/104)).
- 🎨 **Core** — all previously-obsoleted members and types were removed as part of the 3.0 modernization.
- 🎨 **Core** — function pointers and `LibraryImport` marshalling replace the legacy P/Invoke shim (❤️ contributed by @maxkatz6).
- 🏗️ **Build / CI** — repository modernized, SDK/workloads/projects rev'd to .NET 8, and APIScan integrated into packaging.

## Breaking Changes

- **HarfBuzzSharp.Blob** — `AsSpan()` was removed. Use `Blob.AsStream()` (or `Blob.Data`/`Blob.Length` with `Marshal.Copy`) to read the underlying bytes.

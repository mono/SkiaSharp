# HarfBuzzSharp 8.3.1.3

> Ships with [SkiaSharp 3.119.2](../3.119.2.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.3) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.2)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.3/index.md) · Ships with [SkiaSharp 3.119.2](../3.119.2.md)

## Highlights

Servicing rebuild that ships with SkiaSharp 3.119.2. The native HarfBuzz binaries
were rebuilt with Windows security-hardening flags (Control Flow Guard,
Buffer Security Check) and Android now ships symbols for improved crash
diagnostics.

## Breaking Changes

None in this release.

## Platform

- **Android now ships debug symbols** — the native `.so` for Android is now
  built with symbols preserved and packed into the NuGet, so stack traces from
  crashes resolve properly.
  ([#3374](https://github.com/mono/SkiaSharp/pull/3374),
  [mono/skia#161](https://github.com/mono/skia/pull/161))

## Security

- **Windows native DLLs hardened with CFG and Buffer Security Check** — the
  Windows binaries now build with Control Flow Guard and BinSkim BA2007
  compliant buffer security checks. ❤️ [@Aguilex](https://github.com/Aguilex)
  ([#3397](https://github.com/mono/SkiaSharp/pull/3397),
  [#3404](https://github.com/mono/SkiaSharp/pull/3404))

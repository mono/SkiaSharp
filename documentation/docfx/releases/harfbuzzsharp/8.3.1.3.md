# HarfBuzzSharp 8.3.1.3

> Ships with [SkiaSharp 3.119.2](../3.119.2.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.3) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.2)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.3/index.md) · Ships with [SkiaSharp 3.119.2](../3.119.2.md)

## Highlights

This is a security hardening release for HarfBuzzSharp native binaries on Windows, with Android symbol-packaging improvements alongside it. Community contributor [@Aguilex](https://github.com/Aguilex) drove the native hardening work.

## Breaking Changes

*None in this release.*

## Security

- **Control Flow Guard (CFG)** — Enables CFG for all Windows native HarfBuzzSharp DLLs. ❤️ [@Aguilex](https://github.com/Aguilex) ([#3397](https://github.com/mono/SkiaSharp/pull/3397))
- **BufferSecurityCheck** — Enables buffer security checks for native DLLs to address BinSkim BA2007. ❤️ [@Aguilex](https://github.com/Aguilex) ([#3404](https://github.com/mono/SkiaSharp/pull/3404))

## Platform Support

- **Packed Android symbols** — Preserves and packages Android symbols to improve post-deployment diagnostics. ([#3374](https://github.com/mono/SkiaSharp/pull/3374))

| Platform | What's New |
|----------|-----------|
| 🪟 Windows | Control Flow Guard and BufferSecurityCheck |
| 🤖 Android | Symbols are preserved and packaged |

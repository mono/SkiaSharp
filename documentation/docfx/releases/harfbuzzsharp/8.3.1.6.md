# HarfBuzzSharp 8.3.1.6

> **Variable fonts and color text APIs** · Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 introduces variable font support and a new `HBColor` struct for HarfBuzz color APIs, alongside expanded platform coverage (Linux Bionic, Tizen x64/arm64) and a simplified WebAssembly build matrix. Community contributors [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) drove the major features in this release.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **Variable font support** — `HarfBuzzSharp` now exposes OpenType variable font APIs, enabling font variation axis enumeration and value-setting. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **`HBColor` struct** — New `HBColor` value type provides a proper managed representation of `hb_color_t` for use with HarfBuzz color APIs. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **`TreatWarningsAsErrors` in binding layer** — All binding warnings are now treated as errors, improving long-term API hygiene. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

### Platform

- **Linux Bionic native assets** — Added native library assets for Linux Bionic. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — HarfBuzz binaries now ship for Tizen x64 and arm64 targets in addition to arm. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM: Drop pre-.NET 8 Emscripten builds** — Removed WebAssembly builds for Emscripten versions older than .NET 8. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 ships with [SkiaSharp 4.147.0](../4.147.0.md) and delivers two major additions: full variable font axis support and color font palette integration, both contributed by the community. Platform coverage expands with Linux Bionic and Tizen x64/arm64 native builds. ❤️ [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) drove the headline features.

## Text & Fonts

- **Variable font support** — Adds full variable font axis enumeration and axis-value queries to `HBFont`, enabling applications to read and manipulate font variation axes at runtime. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Integrates Skia m147's updated OpenType CPAL support for improved color font and variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Proper managed struct for `hb_color_t`, replacing ad-hoc integer representations in color-font queries. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

## Platform

- **Linux Bionic native assets** — Native HarfBuzz libraries for Android/Bionic-based Linux environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — HarfBuzz native libraries now cover Tizen x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM: dropped pre-.NET 8 Emscripten** — WASM builds for Emscripten toolchains older than .NET 8 are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

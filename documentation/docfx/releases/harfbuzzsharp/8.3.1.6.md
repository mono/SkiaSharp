# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 ships with SkiaSharp 4.147.0 and brings full variable font support, color font palette enumeration, and the new `HBColor` struct for correct RGBA color representation. Linux Bionic and Tizen x64/arm64 native support expand platform coverage. [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) contributed key features across multiple previews.

## New Features

### Text & Fonts

- **Variable font support** — Full OpenType variable font axis support, allowing applications to access and set variation axes at runtime. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Exposes color palette enumeration and selection for COLRv0/COLRv1 color fonts. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Introduces a proper managed struct for `hb_color_t`, correctly representing RGBA colors from HarfBuzz API calls. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Linux Bionic native assets** — Adds native library support for Android-compatible Linux Bionic environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Extends Tizen platform support to x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Drop pre-.NET 8 WebAssembly Emscripten builds** — Removes legacy WASM builds for Emscripten versions predating .NET 8. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 introduces two major text rendering capabilities: **variable font shaping** and a new **`HBColor` type** for proper color font palette handling. ❤️ [@ramezgerges](https://github.com/ramezgerges) drove the variable font bindings that allow shaping with OpenType font variation axes, and the new `HBColor` struct correctly represents `hb_color_t` for color font palettes. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) added native binaries for the Linux Bionic target.

## Breaking Changes

*None in this release.*

## New Features

### Text & Fonts

- **Variable font support** — Exposes HarfBuzz's font variation API, enabling shaping with OpenType variable fonts and font variation axes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **`HBColor` struct** — Adds the `HBColor` value type as the correct .NET representation of `hb_color_t`, enabling proper handling of color font palettes via the HarfBuzz API. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Color font palette API** — Improves robustness of variable font and color palette handling in the underlying Skia engine. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))

### Platform

- **Linux Bionic native assets** — Adds HarfBuzzSharp native binaries for the Linux Bionic target (Android-style userspace on Linux). ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64** — Adds Tizen native build support for x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM .NET 8+ only** — Drops native WebAssembly builds for pre-.NET 8 Emscripten versions, streamlining the WASM build matrix. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

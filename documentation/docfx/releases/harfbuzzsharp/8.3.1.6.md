# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 ships with SkiaSharp 4.147.0 and brings the most significant HarfBuzz API additions in recent releases: full variable font support and the new `HBColor` struct for color font rendering. ❤️ [@ramezgerges](https://github.com/ramezgerges) contributed variable font support and [@4Darmygeometry](https://github.com/4Darmygeometry) added Linux Bionic native assets.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **Variable font support** — `HBFont`, `HBFace`, and related APIs now expose variable font axes for querying and setting named/custom axis values. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **`HBColor` struct** — New `HBColor` struct provides a proper managed type for `hb_color_t`, enabling correct color palette access. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Linux Bionic native assets** — HarfBuzzSharp ships native binaries for Linux Bionic. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64** — Added native builds for Tizen x64 and ARM64. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM: dropped pre-.NET 8 Emscripten builds** — Removed WebAssembly builds for pre-.NET 8 Emscripten versions. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

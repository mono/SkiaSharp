# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 ships with [SkiaSharp 4.147.0](../4.147.0.md) and delivers two headline text features: **variable font support**, allowing runtime control of font axes, and **color font palette APIs**, enabling correct rendering of multi-colour and emoji fonts. Community members [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) drove the majority of these changes. The release also adds the `HBColor` struct for type-safe colour palette operations, expands platform coverage to Linux Bionic and Tizen x64/ARM64, and drops legacy pre-.NET 8 WebAssembly native builds.

## Breaking Changes

- **WebAssembly pre-.NET 8 native builds dropped** — Native WASM builds targeting Emscripten for pre-.NET 8 runtimes are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## New Features

### Text & Fonts

- **Variable font support** — Exposes HarfBuzz variable font axis enumeration and selection, enabling runtime control of font weight, width, and custom axes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Added palette enumeration and selection APIs for HarfBuzz color fonts, backed by the Skia m147 color palette engine. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — A new `HBColor` type wraps `hb_color_t`, providing type-safe colour channel access for palette operations. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Linux Bionic native assets** — Added Linux Bionic ABI native library support. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64** — Added native build support for Tizen on x86_64 and aarch64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))

### Engine

- **Skia milestone m147** — Updated to the Skia m147 engine, which this HarfBuzz line tracks. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))

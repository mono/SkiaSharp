# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

HarfBuzzSharp 8.3.1.6 ships with [SkiaSharp 4.147.0](../4.147.0.md) and delivers the most significant expansion to font support in the binding: **variable font** axis queries and application contributed by [@ramezgerges](https://github.com/ramezgerges), **color font palette** access for COLR/CPAL rendering, and a new `HBColor` struct for idiomatic managed access to `hb_color_t` values. New platform targets include Linux Bionic (contributed by [@4Darmygeometry](https://github.com/4Darmygeometry)), Tizen x64/arm64, and the removal of pre-.NET 8 WASM builds.

## API Surface

- **Variable font support** — New APIs for enumerating and setting variable font axes (weight, width, slant, and custom axes). ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Exposes HarfBuzz color palette APIs for COLR/CPAL font rendering. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Adds a proper managed representation for `hb_color_t`, replacing raw `uint` usage in color palette APIs. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

# HarfBuzzSharp 8.3.1.6

> Ships with [SkiaSharp 4.147.0](../4.147.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/8.3.1.6) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0)

> **API changes** · [HarfBuzzSharp API diff](8.3.1.6/index.md) · Ships with [SkiaSharp 4.147.0](../4.147.0.md)

## Highlights

This is a major text and platform release for HarfBuzzSharp, bringing variable font support, color font palette work, and a new `HBColor` representation while broadening native asset coverage for Linux Bionic and Tizen. It also raises the WebAssembly baseline by dropping pre-.NET 8 Emscripten native builds.

## Breaking Changes

- **Older Emscripten builds removed** — Native WebAssembly builds for pre-.NET 8 Emscripten versions are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## New Features

### Text & Fonts

- **Variable font support** — Adds variable font support to HarfBuzzSharp. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette improvements** — Adds color font palette support and improves variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **New `HBColor` struct** — Introduces `HBColor` as the managed representation for `hb_color_t`. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Milestone 147 native refresh** — Moves this release line onto the native stack that enables the new HarfBuzzSharp feature set. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **Linux Bionic native assets** — Adds Linux Bionic support for native packaging. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Expanded Tizen support** — Adds x64 and arm64 Tizen native builds for this release line. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))

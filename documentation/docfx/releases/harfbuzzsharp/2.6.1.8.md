## Highlights

Ships as part of the SkiaSharp 2.80.3 release. The binding was regenerated in tandem with the harfbuzz-touching PRs listed below, and picks up musl detection, .NET 5, and multi-Emscripten WASM builds along the way.

- 🎨 **Core** — the harfbuzz interop layer was regenerated from source (`Generate the interop with harfbuzz`).
- 🌐 **WebAssembly** — the binding is now packaged for all supported Emscripten versions.
- 🐧 **Linux** — proper detection of musl-based systems (Alpine).

## Breaking Changes

- **HarfBuzzSharp.OpenTypeMetrics** — the base type changed from `System.ValueType` to `System.Object` (the type is now a `class`, not a `struct`). The previous parameter-less constructor was removed. Code that boxed, `struct`-copied, or `default`-constructed `OpenTypeMetrics` needs to switch to the `Font.OpenTypeMetrics` accessor.

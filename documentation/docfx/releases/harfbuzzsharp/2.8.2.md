# HarfBuzzSharp 2.8.2

> Ships with [SkiaSharp 2.88.0](../2.88.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/2.8.2) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v2.88.0)

> **API changes** · [HarfBuzzSharp API diff](2.8.2/index.md) · Ships with [SkiaSharp 2.88.0](../2.88.0.md)

## Highlights

This release expands HarfBuzzSharp across modern Apple and .NET workloads, while also refreshing the shipped native text stack and tightening interop behavior.

## Breaking Changes

*None in this release.*

## New Features

### Platform Support

- **Mac Catalyst support** — Adds Mac Catalyst coverage for the HarfBuzzSharp stack and follows up with build task fixes for that platform. ([#1681](https://github.com/mono/SkiaSharp/pull/1681), [#1760](https://github.com/mono/SkiaSharp/pull/1760), [#1865](https://github.com/mono/SkiaSharp/pull/1865))
- **Newer .NET targets** — Adds net6.0 target frameworks and updates the projects for the .NET 6 wave. ([#1707](https://github.com/mono/SkiaSharp/pull/1707), [#1740](https://github.com/mono/SkiaSharp/pull/1740))

### Interop & Runtime

- **Native stack refresh** — Refreshes the shipped native text shaping components used by this release line. ([#1766](https://github.com/mono/SkiaSharp/pull/1766))
- **Enum marshaling fix** — Works around enums not being blittable in the generated/native interop layer. ([#1857](https://github.com/mono/SkiaSharp/pull/1857))

## Bug Fixes

- **General issue cleanup** — Rolls up a broad set of fixes and smaller improvements across the release line. ([#1958](https://github.com/mono/SkiaSharp/pull/1958))

# HarfBuzzSharp 14.2.0

> Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with [SkiaSharp 4.148.0](../4.148.0.md) and represents a **major upstream HarfBuzz version jump** to 14.2.0, incorporating all new shaping capabilities from the upstream library. It builds on the variable font and color palette support introduced in HarfBuzzSharp 8.3.1.6 (rolled up from the [4.147.0](../4.147.0.md) preview cycle) and adds a singleton instance lifecycle fix contributed by [@ramezgerges](https://github.com/ramezgerges).

## API Surface

- **Variable font support** — New APIs for enumerating and setting variable font axes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Exposes HarfBuzz color palette APIs for COLR/CPAL font rendering. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Proper managed representation for `hb_color_t`. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Singleton instance lifecycle fix** — Resolves lifetime issues with singleton managed wrapper instances. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))

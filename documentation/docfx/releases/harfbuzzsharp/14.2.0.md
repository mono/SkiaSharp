## Highlights

**HarfBuzzSharp 14.2.0** is a major bump — the first HarfBuzzSharp release on the 14.x line, shipping alongside [SkiaSharp 4.148.0](../4.148.0.md). It brings full variable font support (fvar axes and instances), the new `HBColor` value type for color-font APIs, a reworked singleton lifecycle for global handles, and the platform-support work carried over from 8.3.1.5 (Linux Bionic, Tizen x64/arm64, warnings-as-errors, pre-.NET 8 WASM drop). See the [HarfBuzzSharp API diff](14.2.0/index.md) for the full API delta on top of the 8.3.x line.

## New APIs

- **Variable font support** — Expose fvar axes and instances so shaping consumes variable fonts the same way SkiaSharp does (❤️ [@ramezgerges](https://github.com/ramezgerges) in [#3703](https://github.com/mono/SkiaSharp/pull/3703), [mono/skia#185](https://github.com/mono/skia/pull/185)).
- **`HBColor`** — A dedicated value type that maps `hb_color_t` correctly, replacing ad-hoc `uint` handling in color-font APIs ([#4000](https://github.com/mono/SkiaSharp/pull/4000)).
- **Color font palette** — Round out the variable font work with palette wiring shared with SkiaSharp ([#3742](https://github.com/mono/SkiaSharp/pull/3742), [mono/skia#197](https://github.com/mono/skia/pull/197)).

## Fixes

- **Singleton lifecycle** — Rework the lifetime of global HarfBuzz singleton instances so first-time creation and shutdown are deterministic across AppDomain and process teardown paths (❤️ [@ramezgerges](https://github.com/ramezgerges) in [#4080](https://github.com/mono/SkiaSharp/pull/4080)).
- **Warnings-as-errors** — Tighten the binding assemblies so `TreatWarningsAsErrors` stays on for future work ([#3752](https://github.com/mono/SkiaSharp/pull/3752)).

## Platform support

- **Skia milestone 147** — Bring HarfBuzzSharp forward with the m147 Skia sync (❤️ [@ramezgerges](https://github.com/ramezgerges) in [#3702](https://github.com/mono/SkiaSharp/pull/3702), [mono/skia#184](https://github.com/mono/skia/pull/184)).
- **Linux Bionic** — Add native assets for Bionic-based distributions (❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) in [#3217](https://github.com/mono/SkiaSharp/pull/3217)).
- **Tizen x64 / arm64** — Extend the Tizen native build matrix ([#3620](https://github.com/mono/SkiaSharp/pull/3620), [mono/skia#175](https://github.com/mono/skia/pull/175)).
- **WASM** — Drop pre-.NET 8 Emscripten flavours so HarfBuzzSharp's WebAssembly builds stay in step with the shared toolchain ([#4022](https://github.com/mono/SkiaSharp/pull/4022)).

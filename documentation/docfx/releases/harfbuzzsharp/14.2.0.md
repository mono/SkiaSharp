# HarfBuzzSharp 14.2.0

> Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with SkiaSharp 4.148.0, tracking the upstream HarfBuzz version directly. Building on the variable font and color support from the 4.147.0 cycle, this release adds a reworked singleton lifecycle that prevents use-after-free on app shutdown. ❤️ [@ramezgerges](https://github.com/ramezgerges) contributed variable font support and the singleton lifecycle overhaul, and [@4Darmygeometry](https://github.com/4Darmygeometry) added Linux Bionic native assets.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **Reworked singleton lifecycle** — Singleton instances now have a correct managed lifetime, preventing use-after-free when the app shuts down. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))
- **Variable font support** — `HBFont`, `HBFace`, and related APIs expose variable font axes for querying and setting named/custom axis values. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **`HBColor` struct** — New `HBColor` struct provides a proper managed type for `hb_color_t`, enabling correct color palette access. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Linux Bionic native assets** — HarfBuzzSharp ships native binaries for Linux Bionic. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64** — Added native builds for Tizen x64 and ARM64. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM: dropped pre-.NET 8 Emscripten builds** — Removed WebAssembly builds for pre-.NET 8 Emscripten versions. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

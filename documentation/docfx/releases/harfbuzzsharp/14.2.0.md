# HarfBuzzSharp 14.2.0

> Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with SkiaSharp 4.148.0 and builds on the variable font and color font foundation introduced in 8.3.1.6, adding a correctness fix for singleton instance lifecycle and expanding platform coverage to Linux Bionic. Community contributors [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) led the key improvements.

## Breaking Changes

*None in this release.*

## New Features

### Text & Fonts

- **Variable font support** — Full variable font support in HarfBuzzSharp (rolled up from 8.3.1.6 / 4.147.0 previews). ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — CPAL/COLR color font palette APIs and improved variable font robustness (rolled up). ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Correct `hb_color_t` managed representation (rolled up). ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Engine

- **Skia milestone 147** — Underlying Skia engine update bringing upstream HarfBuzz improvements (rolled up). ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))

## Bug Fixes

- **Singleton lifecycle rework** — Fixes the lifecycle of singleton HarfBuzz instances to prevent use-after-free on application shutdown. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))

## Platform

- **Linux Bionic native assets** — Adds Linux Bionic (Android-compatible) native assets, extending HarfBuzzSharp to Bionic-linked Linux environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64** — Native build support for additional Tizen architectures (rolled up). ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WebAssembly — dropped pre-.NET 8 builds** — Removes native WASM builds for pre-.NET 8 Emscripten toolchains (rolled up). ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## Highlights

HarfBuzzSharp **14.2.0** is the first release under the aligned SkiaSharp 4.x version scheme and ships alongside SkiaSharp 4.148.0. It rolls up the full 4.147.0 preview work — variable fonts, color palettes, `HBColor`, m147 uptake, Bionic + Tizen targets — plus the singleton-lifecycle rework that landed for RC1.

## Breaking Changes

This release includes breaking API signature changes. See the [HarfBuzzSharp API diff](14.2.0/index.md) for the full list. Consumers should review flagged signatures before upgrading.

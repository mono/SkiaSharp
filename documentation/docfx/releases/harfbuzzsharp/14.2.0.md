# HarfBuzzSharp 14.2.0

> Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with [SkiaSharp 4.148.0](../4.148.0.md) and represents a major version jump in the underlying HarfBuzz library (from 8.x to 14.x). This release adds full variable font axis support and color font palette integration, introduces the `HBColor` struct, reworks singleton instance lifecycles, and expands native coverage to Linux Bionic and Tizen x64/arm64. ❤️ [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) drove the key features.

## Text & Fonts

- **Variable font support** — Full variable font axis enumeration and axis-value queries in `HBFont`, enabling runtime font variation axis access. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Integrates OpenType CPAL support for improved color font and variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Proper managed struct for `hb_color_t`, replacing ad-hoc integer representations in color-font queries. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

## API Surface

- **Singleton lifecycle rework** — Reworked the lifecycle of `HBObject` singleton instances for correctness and deterministic disposal. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))

## Platform

- **Linux Bionic native assets** — Native HarfBuzz libraries for Android/Bionic-based Linux environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — HarfBuzz native libraries for Tizen x64 and arm64. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM: dropped pre-.NET 8 Emscripten** — WASM builds for Emscripten toolchains older than .NET 8 are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

# HarfBuzzSharp 14.2.0

> **Variable fonts, color palettes, and `HBColor`** · Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with [SkiaSharp 4.148.0](../4.148.0.md) and delivers variable font support, a new `HBColor` struct, a reworked singleton lifecycle, and expanded platform coverage. OpenType variable-font axis APIs land in both `HBFace` and `HBFont`, and the lifecycle of singleton native objects is now ref-counted and crash-free. Community contributors [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) drove the major features and Linux Bionic support.

## New Features

- **`HBColor` struct** — Proper managed type for `hb_color_t`, enabling correct color glyph API usage. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Variable font support** — Adds `HBFace` and `HBFont` variable-instance APIs for full OpenType font variation axis access. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Improved color palette handling for color and emoji fonts. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))

## Bug Fixes

- **Reworked singleton lifecycle** — Singleton native objects now use a proper ref-counted lifecycle, preventing use-after-free crashes in HarfBuzz singleton instances. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))

## Platform

- **Linux Bionic native assets** — Prebuilt HarfBuzzSharp natives for Android Bionic libc on Linux. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64 native builds** ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **WASM: Dropped pre-.NET 8 Emscripten builds** ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## Highlights

Major-version release of HarfBuzzSharp that ships with the first stable [SkiaSharp 4.148.0](../4.148.0.md). Bumps the bundled HarfBuzz to align with Skia milestone 148, adds 10-bit color-format helpers, and renumbers the assembly to the v4 line.

## Breaking Changes

- **Bundled HarfBuzz upgraded to match Skia milestone 148.** Behavioural changes in shaping and font-fallback follow upstream — see the [API diff](14.2.0/index.md).

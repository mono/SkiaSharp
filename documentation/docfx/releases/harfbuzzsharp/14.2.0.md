# HarfBuzzSharp 14.2.0

> Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with SkiaSharp 4.148.0 and builds cumulatively on HarfBuzzSharp 8.3.1.6, adding a reworked singleton lifecycle alongside all the variable font, color palette, and platform support improvements from the 4.147.0 preview series. [@ramezgerges](https://github.com/ramezgerges) and [@4Darmygeometry](https://github.com/4Darmygeometry) contributed across multiple milestones.

## New Features

### API Surface

- **Reworked singleton instance lifecycle** — Refactors how HarfBuzzSharp singleton objects are initialized and torn down, improving correctness and determinism across managed object lifetimes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))

### Text & Fonts

- **Variable font support** — Full OpenType variable font axis support, enabling applications to access and set variation axes at runtime. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Enumerates and selects named color palettes in COLRv0/COLRv1 color fonts. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Proper managed struct for `hb_color_t`, correctly representing RGBA colors from HarfBuzz API calls. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Linux Bionic native assets** — Adds native library support for Android-compatible Linux Bionic environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Extends Tizen platform support to x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Drop pre-.NET 8 WebAssembly Emscripten builds** — Removes legacy WASM builds for Emscripten versions predating .NET 8. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

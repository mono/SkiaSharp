# HarfBuzzSharp 14.2.0

> **Native HarfBuzz 14** · Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

Version bump to align HarfBuzzSharp with native HarfBuzz 14.2.0 and the SkiaSharp 4.148.0 stable release. Adds variable font support and color font palettes, exposes an `HBColor` struct for `hb_color_t`, reworks the lifecycle of singleton instances, and extends native asset coverage to Linux Bionic and Tizen x64/arm64. Also aligns Emscripten builds with .NET 8+ and switches the binding to `TreatWarningsAsErrors`.

## Breaking Changes

None in this release — see the [HarfBuzzSharp API diff](14.2.0/index.md).

## API Surface

- **`HBColor` struct for `hb_color_t`** — a strongly-typed color representation replaces the raw-`uint` idiom. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Variable font support** — new bindings expose the OpenType variable font APIs so callers can shape and query variation axes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703), [skia#185](https://github.com/mono/skia/pull/185))
- **Color font palette support** — companion binding to the SkiaSharp color font palette work, with related variable-font robustness fixes. ([#3742](https://github.com/mono/SkiaSharp/pull/3742), [skia#197](https://github.com/mono/skia/pull/197))

## Engine

- **Skia milestone 147** — moved onto the upstream m147 branch (this shipped in the 4.148.0 stable line). ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702), [skia#184](https://github.com/mono/skia/pull/184))

## Platform

- **Linux Bionic native assets** — HarfBuzz binaries are now produced for Linux Bionic. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — HarfBuzz now ships native binaries for Tizen x64 and arm64. ([#3620](https://github.com/mono/SkiaSharp/pull/3620), [skia#175](https://github.com/mono/skia/pull/175))
- **WASM native builds now require .NET 8+ Emscripten** — pre-.NET 8 Emscripten variants are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## Lifecycle & Internals

- **Reworked singleton lifecycle** — the singleton instances used by HarfBuzzSharp were reorganized so they can be reliably reset and torn down. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))
- **`TreatWarningsAsErrors` in `binding/`** — every warning in the binding is now fatal, with all existing warnings cleaned up. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

# HarfBuzzSharp 14.2.0

> Ships with [SkiaSharp 4.148.0](../4.148.0.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/14.2.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [HarfBuzzSharp API diff](14.2.0/index.md) · Ships with [SkiaSharp 4.148.0](../4.148.0.md)

## Highlights

HarfBuzzSharp 14.2.0 ships with [SkiaSharp 4.148.0](../4.148.0.md) and delivers the HarfBuzz 14.2 library update alongside the full feature set from the 8.3.1.6 line: variable font axis control, color font palette APIs, the `HBColor` struct, and expanded platform coverage. On top of that, [@ramezgerges](https://github.com/ramezgerges) reworked the singleton object lifecycle to prevent use-after-free races during finalization, making long-running applications significantly more reliable.

## Breaking Changes

- **WebAssembly pre-.NET 8 native builds dropped** — Native WASM builds targeting Emscripten for pre-.NET 8 runtimes are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## New Features

### API Surface

- **Reworked singleton instance lifecycle** — Internal singleton objects now use a deterministic lifecycle, preventing use-after-free races during finalization. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))
- **Variable font support** — Exposes HarfBuzz variable font axis enumeration and selection for runtime control of font weight, width, and custom axes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Palette enumeration and selection for HarfBuzz color fonts. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Type-safe `hb_color_t` wrapper for colour palette operations. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### Platform

- **Linux Bionic native assets** — ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64** — Native build support for Tizen on x86_64 and aarch64. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))

### Engine

- **Skia milestone m147** — Updated Skia engine backing HarfBuzz. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))

# HarfBuzzSharp 8.3.1.6

> **Upcoming release** · In development · Ships with the upcoming SkiaSharp 4.147.0 · Not yet available on NuGet

<!-- Generated: 2026-07-10T21:18:35Z by claude-opus-4.7 -->

## Highlights

HarfBuzzSharp **8.3.1.6** is the ship-along for the upcoming SkiaSharp 4.147.0 preview cycle. It carries the biggest HarfBuzz-touching set in years: a new `HBColor` struct for proper `hb_color_t` representation, variable-font support, color-font palette APIs, the m147 Skia bump, and new Bionic + Tizen build targets.

## Breaking Changes

See the linked HarfBuzzSharp API diff (published with the ship-along SkiaSharp release) for signature changes — the variable-font and palette additions expand the surface.

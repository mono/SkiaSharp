# HarfBuzzSharp 8.3.1.6

> **Upcoming release** · In development · Ships with the upcoming SkiaSharp 4.147.0 · Not yet available on NuGet

## Highlights

Servicing companion for the upcoming SkiaSharp 4.147.0 preview line. Picks up the Skia milestone 147 bump, adds an `HBColor` struct that properly represents `hb_color_t`, extends variable font support to HarfBuzzSharp, drops native builds for pre-.NET 8 Emscripten, and enables `TreatWarningsAsErrors` across the binding.

## Breaking Changes

None in this release.

## API Surface

- **`HBColor` struct** — proper strong-typed representation of `hb_color_t`, avoiding the raw-`uint` idiom. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Variable font support in HarfBuzzSharp** — new bindings expose the OpenType variable font APIs so callers can shape and query variation axes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703), [skia#185](https://github.com/mono/skia/pull/185))
- **Color font palette support** — companion binding to the SkiaSharp color font palette work, with related variable-font robustness fixes. ([#3742](https://github.com/mono/SkiaSharp/pull/3742), [skia#197](https://github.com/mono/skia/pull/197))

## Engine

- **Skia milestone 147** — moved onto the upstream m147 branch. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702), [skia#184](https://github.com/mono/skia/pull/184))

## Platform

- **Linux Bionic native assets** — HarfBuzz native binaries are now produced for Linux Bionic. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — HarfBuzz now ships native binaries for Tizen x64 and arm64. ([#3620](https://github.com/mono/SkiaSharp/pull/3620), [skia#175](https://github.com/mono/skia/pull/175))
- **WASM native builds now require .NET 8+ Emscripten** — pre-.NET 8 Emscripten variants are no longer produced. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## Lifecycle & Internals

- **`TreatWarningsAsErrors` in `binding/`** — every warning in the binding is now fatal, with all existing warnings cleaned up. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

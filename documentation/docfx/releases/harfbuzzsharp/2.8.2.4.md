# HarfBuzzSharp 2.8.2.4

> Ships with [SkiaSharp 2.88.4](../2.88.4.md) · [NuGet](https://www.nuget.org/packages/HarfBuzzSharp/2.8.2.4) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v2.88.4)

> **API changes** · [HarfBuzzSharp API diff](2.8.2.4/index.md) · Ships with [SkiaSharp 2.88.4](../2.88.4.md)

## Highlights

This release expands HarfBuzzSharp support for modern WebAssembly and Apple environments, including emsdk 3.1.12 fixes, Apple Silicon macOS build support, and better Hot Restart packaging. Community contributor ❤️ [@jeromelaban](https://github.com/jeromelaban) delivered the WebAssembly work.

## Breaking Changes

*None in this release.*

## Platform Support

- **WebAssembly refresh** — Adds multithreaded and multithreaded+SIMD WASM variants and fixes HarfBuzz for emsdk 3.1.12. ❤️ [@jeromelaban](https://github.com/jeromelaban) ([#2286](https://github.com/mono/SkiaSharp/pull/2286))
- **Apple Silicon build support** — Adds support for ARM64 macOS machines in the native build pipeline. ([#2468](https://github.com/mono/SkiaSharp/pull/2468))
- **Hot Restart packaging** — Includes the required NativeReference for Hot Restart scenarios. ([#2553](https://github.com/mono/SkiaSharp/pull/2553))

# Version 2.88.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.88.0)

## Highlights

SkiaSharp 2.88.0 rolls up 63 PRs across 15 preview cuts between 2.80.4 and stable. The headline is first-class .NET MAUI support (compat and handler renderers), Mac Catalyst native support, Blazor WebAssembly components, and per-runtime "Native Asset" packages that split the SkiaSharp NuGet into smaller, targeted downloads. HarfBuzz jumps to 2.8.2 and the samples now target the shipping .NET 6.0. HarfBuzzSharp 2.8.2 ships alongside this release; see the [linked page](harfbuzzsharp/2.8.2.md) for its binding details.

## Breaking Changes

The MAUI/Views work and the iOS/tvOS platform reshaping altered several signatures. See the [breaking-changes diff](2.88.0/SkiaSharp/SkiaSharp.breaking.md), the [SkiaSharp.Views.iOS diff](2.88.0/SkiaSharp.Views/SkiaSharp.Views.iOS.breaking.md), and the [SkiaSharp.Views.tvOS diff](2.88.0/SkiaSharp.Views/SkiaSharp.Views.tvOS.breaking.md) for details.

# Version 2.88.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.88.0)

> **API changes** · [SkiaSharp API diff](2.88.0/index.md) · [HarfBuzzSharp 2.8.2](harfbuzzsharp/2.8.2.md)

<!-- Generated: 2026-07-06T23:24:50Z by claude-opus-4.7 -->

Release for .NET 6 and .NET MAUI. Adds **native .NET MAUI views (compat + handlers)**, **Mac Catalyst** support, **ASP.NET Blazor WebAssembly components**, splits SkiaSharp into smaller **native asset** packages, ships a **WindowsAppSDK** binding, and updates HarfBuzz to 2.8.2. HarfBuzzSharp 2.8.2 co-ships — see the [HarfBuzzSharp 2.8.2](harfbuzzsharp/2.8.2.md) page for its binding details.

## Highlights

- **.NET 6** support ([#1740](https://github.com/mono/SkiaSharp/pull/1740)) and **native .NET MAUI views** (compat + handlers) ([#1678](https://github.com/mono/SkiaSharp/pull/1678)); the MAUI sample is refreshed ([#1851](https://github.com/mono/SkiaSharp/pull/1851)).
- **Mac Catalyst** support ([#1760](https://github.com/mono/SkiaSharp/pull/1760), [#1681](https://github.com/mono/SkiaSharp/pull/1681)).
- **ASP.NET Blazor WebAssembly components** ([#1811](https://github.com/mono/SkiaSharp/pull/1811)); [@JensKrumsieck](https://github.com/JensKrumsieck) prevented an infinite loop in the `SKCanvasView` used by Blazor ([#1889](https://github.com/mono/SkiaSharp/pull/1889)).
- **Native-asset package split** — `SkiaSharp.NativeAssets.*` and `HarfBuzzSharp.NativeAssets.*` packages let consumers pick only the RIDs they need ([#1758](https://github.com/mono/SkiaSharp/pull/1758), [#1797](https://github.com/mono/SkiaSharp/pull/1797)).
- **WindowsAppSDK** support ([#1800](https://github.com/mono/SkiaSharp/pull/1800)).
- **Uno 4.0** support, contributed by [@jeromelaban](https://github.com/jeromelaban) ([#1873](https://github.com/mono/SkiaSharp/pull/1873)); Blazor infinite-loop fix ([#1889](https://github.com/mono/SkiaSharp/pull/1889)); Android `.so` files no longer included in class libraries, contributed by [@jonathanpeppers](https://github.com/jonathanpeppers) ([#1895](https://github.com/mono/SkiaSharp/pull/1895)); shrunk `Resource.designer.cs` on Android ([#1812](https://github.com/mono/SkiaSharp/pull/1812)); ANGLE binding replaces the WPF `HandleDictionary` lock, contributed by [@toptensoftware](https://github.com/toptensoftware) ([#1817](https://github.com/mono/SkiaSharp/pull/1817)).
- **HarfBuzz 2.8.2** ([#1766](https://github.com/mono/SkiaSharp/pull/1766)) — ships alongside; see the linked page for HarfBuzzSharp binding details.

## API Surface

- Native .NET MAUI views ([#1678](https://github.com/mono/SkiaSharp/pull/1678)).
- Xamarin.Forms on netcoreapp3.1 ([#1774](https://github.com/mono/SkiaSharp/pull/1774)); `net6.0-*` TFMs for the binding projects ([#1707](https://github.com/mono/SkiaSharp/pull/1707)); new APIs added for later work ([#1828](https://github.com/mono/SkiaSharp/pull/1828)); the way `IgnorePixelScaling` works is fixed ([#1804](https://github.com/mono/SkiaSharp/pull/1804)).
- `p/invoke` to `libdl.so.2` before `libdl.so`, contributed by [@akoeplinger](https://github.com/akoeplinger) ([#2010](https://github.com/mono/SkiaSharp/pull/2010)).

## Engine

- HarfBuzz native engine updated to 2.8.2 ([#1766](https://github.com/mono/SkiaSharp/pull/1766)); build using the older gcc 6.2 and newer llvm 10 compilers ([#1806](https://github.com/mono/SkiaSharp/pull/1806)); Python 3.x and Clang 12.x on the build agents ([#1959](https://github.com/mono/SkiaSharp/pull/1959)); emscripten 2.0.23 support for WASM, contributed by [@jeromelaban](https://github.com/jeromelaban) ([#1746](https://github.com/mono/SkiaSharp/pull/1746)).

- **.NET MAUI** views and the RC / release-ladder tracking (RC 1 → RC 4) — Preview 7 through RC 4 ([#1749](https://github.com/mono/SkiaSharp/pull/1749), [#1782](https://github.com/mono/SkiaSharp/pull/1782), [#1815](https://github.com/mono/SkiaSharp/pull/1815), [#1842](https://github.com/mono/SkiaSharp/pull/1842), [#1890](https://github.com/mono/SkiaSharp/pull/1890), [#1932](https://github.com/mono/SkiaSharp/pull/1932), [#1942](https://github.com/mono/SkiaSharp/pull/1942), [#1965](https://github.com/mono/SkiaSharp/pull/1965), [#1967](https://github.com/mono/SkiaSharp/pull/1967), [#1986](https://github.com/mono/SkiaSharp/pull/1986), [#2004](https://github.com/mono/SkiaSharp/pull/2004), [#2029](https://github.com/mono/SkiaSharp/pull/2029), [#2032](https://github.com/mono/SkiaSharp/pull/2032)); iOS view now requires a background ([#1730](https://github.com/mono/SkiaSharp/pull/1730)).

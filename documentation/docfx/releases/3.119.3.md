## Highlights

**3.119.3** was a native-dependency-focused preview that never shipped as stable — every change rolled up into [3.119.4](3.119.4.md). It bumped libpng, libwebp, brotli and libexpat, fixed a crash on AoT builds when `IlcDisableReflection=true`, corrected the Linux ARM64 sysroot for fontconfig, and added Spectre mitigation flags to the Windows and Linux native binaries. HarfBuzz 8.3.1.4 ships alongside — see [its release notes](harfbuzzsharp/8.3.1.4.md) for the binding details.

## Fixes

- **AoT with reflection disabled** — Fix a crash on native AoT builds when `IlcDisableReflection=true` so trim-aggressive apps can still initialize SkiaSharp ([#3485](https://github.com/mono/SkiaSharp/pull/3485)).
- **Linux ARM64 fontconfig** — Add the fontconfig runtime library to the cross-compile sysroot so Linux ARM64 no longer crashes on font resolution ([#3494](https://github.com/mono/SkiaSharp/pull/3494)).
- **Skia submodule pin** — Fix the skia submodule pointer to reference a squashed commit that stays reachable ([#3474](https://github.com/mono/SkiaSharp/pull/3474), [mono/skia#165](https://github.com/mono/skia/pull/165)).

## Platform support

- **Spectre mitigations** — Add the Spectre mitigation flag to `libSkiaSharp.so` and `libSkiaSharp.dll` for hardening on Linux and Windows (❤️ [@sshumakov](https://github.com/sshumakov) in [#3496](https://github.com/mono/SkiaSharp/pull/3496) and [#3502](https://github.com/mono/SkiaSharp/pull/3502)).
- **Native dependency bumps** — Update libpng to 1.6.54 ([#3452](https://github.com/mono/SkiaSharp/pull/3452), [mono/skia#162](https://github.com/mono/skia/pull/162)), libwebp to 1.6.0 ([#3478](https://github.com/mono/SkiaSharp/pull/3478), [mono/skia#167](https://github.com/mono/skia/pull/167)) and brotli to 1.2.0 ([#3469](https://github.com/mono/SkiaSharp/pull/3469)).
- **DEPS clean-up** — Remove unused Skia DEPS entries and add DEPS documentation so future dependency work has a canonical reference ([#3463](https://github.com/mono/SkiaSharp/pull/3463), [mono/skia#164](https://github.com/mono/skia/pull/164)).
- **API-add workflow** — Add API-add guidelines and an implement-issue skill so external contributions have a documented path ([#3445](https://github.com/mono/SkiaSharp/pull/3445)).

## Other changes

Plus various CI, documentation, and internal tooling improvements — Linux Docker integration testing, release-skill refinements, and the first tranche of AI-assisted release automation.

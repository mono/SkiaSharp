# Version 3.119.3

> **Dependency updates & security hardening** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.3-preview.1.1)

## Highlights

This release updates four native dependencies to their latest versions, extends Spectre mitigations to Linux, and fixes two runtime crashes. Community contributor [@sshumakov](https://github.com/sshumakov) added Spectre mitigation flags for both Windows and Linux native libraries.

## Breaking Changes

*None in this release.*

## Security

- **Spectre mitigation for Windows** — Enables `/Qspectre` for `libSkiaSharp.dll`, hardening against speculative execution attacks. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Enables `-mretpoline` for `libSkiaSharp.so`, extending Spectre hardening to Linux builds. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))

## Bug Fixes

- **Fixed AoT crash when `IlcDisableReflection=true`** — Resolved a crash in NativeAOT-compiled applications that disable reflection. ([#3485](https://github.com/mono/SkiaSharp/pull/3485))
- **Fixed Linux ARM64 crash from missing fontconfig** — Added fontconfig to the cross-compile sysroot so Linux ARM64 builds no longer crash at startup. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

## Dependencies

- **libpng** updated to 1.6.54 ([#3452](https://github.com/mono/SkiaSharp/pull/3452))
- **libexpat** updated from 2.6.4 to 2.7.3 ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **brotli** updated to 1.2.0 ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libwebp** updated to 1.6.0 ([#3478](https://github.com/mono/SkiaSharp/pull/3478))
- Removed unused native dependencies to reduce binary size ([#3463](https://github.com/mono/SkiaSharp/pull/3463))

# Version 3.119.3

> **Security & reliability hardening** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.3-preview.1.1)

## Highlights

This release hardens native libraries with four dependency updates and extends Spectre mitigations to Linux. It also fixes two crashes — an AoT crash when `IlcDisableReflection=true` and a Linux ARM64 crash caused by missing fontconfig. ❤️ [@sshumakov](https://github.com/sshumakov) contributed the Spectre mitigation work for both Windows and Linux.

## Breaking Changes

*None in this release.*

## New Features

### Security

- **Spectre mitigation for Windows** — Enables `/Qspectre` for `libSkiaSharp.dll`, hardening against speculative execution attacks. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Enables `-mretpoline` for `libSkiaSharp.so`, bringing parity with the Windows hardening. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))

### Dependencies

- **libpng → 1.6.54** — Picks up upstream bug fixes and hardening. ([#3452](https://github.com/mono/SkiaSharp/pull/3452))
- **libexpat → 2.7.3** — Bumped from 2.6.4, addressing multiple upstream security advisories. ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **brotli → 1.2.0** — Updated to the latest stable release. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libwebp → 1.6.0** — Updated to the latest stable release. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))

## Bug Fixes

- **Fixed AoT crash when `IlcDisableReflection=true`** — Resolved a crash in NativeAOT-compiled applications that disabled reflection. ([#3485](https://github.com/mono/SkiaSharp/pull/3485))
- **Fixed Linux ARM64 crash from missing fontconfig** — The cross-compile sysroot was missing fontconfig, causing crashes on Linux ARM64. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

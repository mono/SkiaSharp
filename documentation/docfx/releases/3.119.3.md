# Version 3.119.3

> **Preview only** · Superseded by [3.119.4](3.119.4.md) · Never released as stable — these changes rolled up into 3.119.4 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview)

## Highlights

> **Note:** 3.119.3 shipped only as previews and was never released as a stable package. All of the work below rolled up into [3.119.4](3.119.4.md).

This preview cycle focused on native dependency updates and stability fixes. libwebp, brotli, libexpat, and libpng were all bumped to their latest versions, and Spectre mitigations landed for both the Windows and Linux native libraries. Two community contributors — ❤️ [@sshumakov](https://github.com/sshumakov) and ❤️ [@Copilot](https://github.com/Copilot) — contributed the security hardening and an AoT crash fix.

## Breaking Changes

*None in this release.*

## Security

- **Spectre mitigation for Windows** — Enables `/Qspectre` for `libSkiaSharp.dll`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Enables Spectre mitigation for `libSkiaSharp.so`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))

## Bug Fixes

- **Linux ARM64 fontconfig crash** — Adds the fontconfig runtime library to the cross-compile sysroot, fixing a crash at startup on Linux ARM64. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))
- **AoT crash when `IlcDisableReflection=true`** — Resolves a crash in Native AoT builds when reflection is fully disabled. ❤️ [@Copilot](https://github.com/Copilot) ([#3485](https://github.com/mono/SkiaSharp/pull/3485))

## Dependencies

- **libwebp updated to 1.6.0** — Bumps the bundled WebP library to the latest stable release. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))
- **brotli updated to 1.2.0** — Bumps the bundled Brotli compression library. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libexpat bumped from 2.6.4 to 2.7.3** — Addresses known vulnerabilities in the XML parsing library. ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **libpng updated to 1.6.54** — Bumps the bundled PNG library to the latest stable release. ([#3452](https://github.com/mono/SkiaSharp/pull/3452))
- **Removed unused native dependencies** — Cleans up unused bundled libraries and adds documentation for the remaining dependency set. ([#3463](https://github.com/mono/SkiaSharp/pull/3463))

Plus several CI, tooling, and AI skill improvements.

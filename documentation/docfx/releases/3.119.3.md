# Version 3.119.3

> **Preview only** · Superseded by [3.119.4](3.119.4.md) · Never released as stable — these changes rolled up into 3.119.4 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview)

> **API changes** · [SkiaSharp API diff](3.119.3/index.md) · [HarfBuzzSharp 8.3.1.4](harfbuzzsharp/8.3.1.4.md)

## Highlights

This preview concentrated on native hardening, dependency refreshes, and compatibility fixes across Linux ARM64 and ahead-of-time scenarios. It never shipped as a stable release and was rolled forward into 3.119.4. HarfBuzzSharp 8.3.1.4 ships alongside this preview line; see the linked companion page for its binding details.

## Breaking Changes

*None in this release.*

## Security

- **Spectre mitigations for Windows and Linux** — Adds Spectre hardening to both the Windows DLLs and `libSkiaSharp.so`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496), [#3502](https://github.com/mono/SkiaSharp/pull/3502))

## Bug Fixes

- **Linux ARM64 startup crash** — Fixes a Linux ARM64 crash by adding the missing `fontconfig` runtime library to the cross-compile sysroot. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))
- **AoT crash with `IlcDisableReflection=true`** — Fixes an ahead-of-time runtime crash in trimmed applications. ❤️ [@Copilot](https://github.com/Copilot) ([#3485](https://github.com/mono/SkiaSharp/pull/3485))

## Dependency Updates

- **Native library refreshes** — Updates libwebp to 1.6.0, brotli to 1.2.0, libexpat to 2.7.3, and libpng to 1.6.54. ([#3478](https://github.com/mono/SkiaSharp/pull/3478), [#3469](https://github.com/mono/SkiaSharp/pull/3469), [#3458](https://github.com/mono/SkiaSharp/pull/3458), [#3452](https://github.com/mono/SkiaSharp/pull/3452))

# Version 3.119.3

> **Preview only** · Superseded by [3.119.4](3.119.4.md) · Never released as stable — these changes rolled up into 3.119.4 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview)

> **API changes** · [SkiaSharp API diff](3.119.3/index.md) · [HarfBuzzSharp 8.3.1.4](harfbuzzsharp/8.3.1.4.md)

## Highlights

3.119.3 was a servicing-focused preview that refreshed several native dependencies (libpng, libwebp, brotli, libexpat), extended Spectre mitigation to the Linux and Windows native binaries, and fixed an AoT crash triggered by `IlcDisableReflection=true`. HarfBuzzSharp 8.3.1.4 shipped alongside — see its [linked page](harfbuzzsharp/8.3.1.4.md). **This preview line was never released as stable; everything below rolled up into [3.119.4](3.119.4.md).**

## Breaking Changes

None in this preview — see the [SkiaSharp API diff](3.119.3/index.md) for the additive-only signature summary.

## Security

- **Spectre mitigation on `libSkiaSharp.dll` (Windows)** — Windows native DLL now builds with the Spectre v1 mitigation flag. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation on `libSkiaSharp.so` (Linux)** — the same mitigation is now applied to the Linux shared object. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))
- **libpng 1.6.54** — pulls in upstream libpng bug fixes and hardening. ([#3452](https://github.com/mono/SkiaSharp/pull/3452), [skia#162](https://github.com/mono/skia/pull/162))
- **libwebp 1.6.0** — refreshes the WebP encoder/decoder to the latest upstream. ([#3478](https://github.com/mono/SkiaSharp/pull/3478), [skia#167](https://github.com/mono/skia/pull/167))
- **brotli 1.2.0** — refreshes the brotli dependency. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))

## Bug Fixes

- **AoT crash when `IlcDisableReflection=true`** — SkiaSharp no longer crashes in AoT builds that opt into reflection stripping. ([#3485](https://github.com/mono/SkiaSharp/pull/3485))
- **Linux ARM64 fontconfig crash on cross-compile** — added `fontconfig` to the cross-compile sysroot so font resolution works on Linux ARM64. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

## Lifecycle & Internals

- **Skia submodule pinned to squashed commit** — cleans up the submodule history so the `externals/skia` SHA matches the squash-merged commit. ([#3474](https://github.com/mono/SkiaSharp/pull/3474), [skia#165](https://github.com/mono/skia/pull/165))
- **Removed unused native dependencies** — drops native libraries that were never actually linked into the build. ([#3463](https://github.com/mono/SkiaSharp/pull/3463), [skia#164](https://github.com/mono/skia/pull/164))
- Plus various CI, documentation, and internal tooling improvements.

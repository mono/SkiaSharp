# Version 3.119.3

> **Preview only** · Superseded by [3.119.4](3.119.4.md) · Never released as stable — these changes rolled up into 3.119.4 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview)

> **API changes** · [SkiaSharp API diff](3.119.3/index.md) · [HarfBuzzSharp 8.3.1.4](harfbuzzsharp/8.3.1.4.md)

## Highlights

This preview-only series continued the security hardening started in 3.119.2 — extending Spectre mitigations to Linux — and fixed a crash in AoT-compiled applications. All changes rolled up into [3.119.4](3.119.4.md). ❤️ [@sshumakov](https://github.com/sshumakov) added Spectre mitigations for both Windows and Linux native libraries, and ❤️ [@Copilot](https://github.com/Copilot) fixed the AoT crash when `IlcDisableReflection` is enabled.

## Breaking Changes

*None in this release.*

## New Features

### Security

- **Spectre mitigation for Linux** — Enables the `/Qspectre` flag for `libSkiaSharp.so`, hardening the Linux native library against Spectre-class CPU vulnerabilities. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))
- **Spectre mitigation for Windows** — Enables `/Qspectre` for `libSkiaSharp.dll`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))

### Platform

- **Linux ARM64 crash fix** — Adds the fontconfig runtime library to the cross-compilation sysroot, resolving a startup crash on Linux ARM64 targets. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

## Bug Fixes

- **AoT crash with `IlcDisableReflection=true`** — Fixes a startup crash in Native AoT builds where reflection had been disabled via `IlcDisableReflection`. ❤️ [@Copilot](https://github.com/Copilot) ([#3485](https://github.com/mono/SkiaSharp/pull/3485))

## Security / Dependencies

- **libwebp 1.6.0** — Updated to include the latest security fixes and performance improvements. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))
- **brotli 1.2.0** — Updated to the latest release. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libexpat 2.7.3** — Bumped from 2.6.4 to patch known vulnerabilities. ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **libpng 1.6.54** — Updated to the latest stable release. ([#3452](https://github.com/mono/SkiaSharp/pull/3452))

# Version 3.119.3

> **Preview only** · Superseded by [3.119.4](3.119.4.md) · Never released as stable — these changes rolled up into 3.119.4 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview)

> **API changes** · [SkiaSharp API diff](3.119.3/index.md) · [HarfBuzzSharp 8.3.1.4](harfbuzzsharp/8.3.1.4.md)

## Highlights

This preview-only release focused on security hardening and stability: Spectre mitigations landed for both Windows and Linux native libraries, a NativeAOT crash was fixed for apps with reflection disabled, and a fontconfig crash on Linux ARM64 was resolved. All of these changes shipped cumulatively in [3.119.4](3.119.4.md). Community contributors [@sshumakov](https://github.com/sshumakov) and [@Copilot](https://github.com/Copilot) drove both Spectre mitigation PRs and the AoT fix.

## Security

- **Spectre mitigation for Windows** — Enables `/Qspectre` for `libSkiaSharp.dll`, reducing information-disclosure risk on Windows. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Enables Spectre mitigation flags for `libSkiaSharp.so`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))
- **libwebp updated to 1.6.0** — Latest security release. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))
- **brotli updated to 1.2.0** ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libexpat bumped to 2.7.3** — Security fix release (from 2.6.4). ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **libpng updated to 1.6.54** ([#3452](https://github.com/mono/SkiaSharp/pull/3452))

## Bug Fixes

- **Fixed NativeAOT crash with `IlcDisableReflection=true`** — Resolves a crash in apps compiled ahead-of-time with reflection disabled. ❤️ [@Copilot](https://github.com/Copilot) ([#3485](https://github.com/mono/SkiaSharp/pull/3485))
- **Fixed Linux ARM64 startup crash** — Adds the fontconfig runtime library to the ARM64 cross-compile sysroot, resolving a crash at startup on Linux ARM64. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

# Version 3.119.3

> **Preview only** · Superseded by [3.119.4](3.119.4.md) · Never released as stable — these changes rolled up into 3.119.4 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview)

> **API changes** · [SkiaSharp API diff](3.119.3/index.md) · [HarfBuzzSharp 8.3.1.4](harfbuzzsharp/8.3.1.4.md)

## Highlights

This security hardening and stability preview never shipped as a stable release — all of the work below rolled into [3.119.4](3.119.4.md). The preview adds Spectre mitigations for both Windows and Linux native libraries (contributed by [@sshumakov](https://github.com/sshumakov)), fixes a Linux ARM64 startup crash caused by a missing fontconfig runtime, repairs an AoT compilation crash when `IlcDisableReflection=true` (contributed by [@Copilot](https://github.com/Copilot)), and updates four native dependencies — libwebp, libpng, brotli, and libexpat — for security and correctness.

## Security

- **Spectre mitigation for Windows** — Enables `/Qspectre` for `libSkiaSharp.dll`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Enables `-mspectre-mitigation-all` for `libSkiaSharp.so`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))
- **Update libwebp to 1.6.0** — Security and correctness update. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))
- **Update libpng to 1.6.54** — Security update. ([#3452](https://github.com/mono/SkiaSharp/pull/3452))
- **Update brotli to 1.2.0** — Dependency refresh. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **Bump libexpat from 2.6.4 to 2.7.3** — Security update. ([#3458](https://github.com/mono/SkiaSharp/pull/3458))

## Bug Fixes

- **Fix Linux ARM64 crash** — Adds the `fontconfig` runtime library to the cross-compile sysroot, resolving a startup crash on ARM64 Linux. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))
- **Fix AoT crash when `IlcDisableReflection=true`** — Resolves a crash during NativeAOT compilation with reflection disabled. ❤️ [@Copilot](https://github.com/Copilot) ([#3485](https://github.com/mono/SkiaSharp/pull/3485))

# Version 3.119.3

> **Security hardening & stability fixes** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.3-preview.1.1)

## Highlights

This release focuses on security and stability. Four native dependencies received updates — libpng 1.6.54, libexpat 2.7.3, brotli 1.2.0, and libwebp 1.6.0 — addressing multiple upstream CVEs. Spectre mitigations are now enabled for the native library on both Windows and Linux. Two crash fixes ship as well: an AoT crash when `IlcDisableReflection=true` and a Linux ARM64 fontconfig crash during cross-compilation. Thanks to community contributor @sshumakov for the Spectre mitigation work.

## Breaking Changes

*None in this release.*

## New Features

### Security

- **Spectre mitigation for Windows** — Adds Spectre mitigation compiler flags to `libSkiaSharp.dll`, hardening the native binary against speculative execution attacks. ❤️ @sshumakov ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Adds matching Spectre mitigation flags to `libSkiaSharp.so`. ❤️ @sshumakov ([#3502](https://github.com/mono/SkiaSharp/pull/3502))

### Native Dependencies

- **libpng → 1.6.54** — Picks up upstream security and bug fixes. ([#3452](https://github.com/mono/SkiaSharp/pull/3452))
- **libexpat → 2.7.3** — Bumps from 2.6.4, addressing multiple CVEs in the XML parser. ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **brotli → 1.2.0** — Updates the compression library to the latest release. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libwebp → 1.6.0** — Updates the WebP codec to the latest release. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))

## Bug Fixes

- **AoT crash with `IlcDisableReflection=true`** — Fixed a crash that occurred when publishing with Native AoT and reflection disabled. ([#3485](https://github.com/mono/SkiaSharp/pull/3485))
- **Linux ARM64 fontconfig crash** — Fixed a crash caused by missing fontconfig in the cross-compile sysroot for Linux ARM64. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

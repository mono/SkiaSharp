# Version 3.119.3

> **Security & stability release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.3-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.3-preview.1.1)

## Highlights

This release extends Spectre mitigations to Linux, bumps four native dependencies to their latest versions, and fixes two crashes — an AoT failure with `IlcDisableReflection=true` and a Linux ARM64 crash caused by missing fontconfig. ❤️ [@sshumakov](https://github.com/sshumakov) contributed the Spectre hardening across both Windows and Linux.

## Breaking Changes

*None in this release.*

## New Features

### Security

- **Spectre mitigation for Windows** — Enables the `/Qspectre` compiler flag for `libSkiaSharp.dll`, hardening Windows native builds against speculative-execution attacks. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496))
- **Spectre mitigation for Linux** — Enables `-mretpoline` for `libSkiaSharp.so`, extending Spectre mitigations to all Linux native builds. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3502](https://github.com/mono/SkiaSharp/pull/3502))

### Dependencies

- **libpng → 1.6.54** — Updated from 1.6.47 to the latest stable release. ([#3452](https://github.com/mono/SkiaSharp/pull/3452))
- **libexpat → 2.7.3** — Updated from 2.6.4 with security and stability fixes. ([#3458](https://github.com/mono/SkiaSharp/pull/3458))
- **brotli → 1.2.0** — Updated to the latest upstream release. ([#3469](https://github.com/mono/SkiaSharp/pull/3469))
- **libwebp → 1.6.0** — Updated to the latest upstream release. ([#3478](https://github.com/mono/SkiaSharp/pull/3478))
- **Removed unused dependencies** — Cleaned up unused native dependencies across platforms and added dependency documentation. ([#3463](https://github.com/mono/SkiaSharp/pull/3463))

## Bug Fixes

- **Fixed AoT crash when `IlcDisableReflection=true`** — Resolved a native AoT failure that occurred when reflection was fully disabled. ([#3485](https://github.com/mono/SkiaSharp/pull/3485))
- **Fixed Linux ARM64 crash from missing fontconfig** — Added fontconfig to the cross-compile sysroot, preventing a crash on Linux ARM64 devices. ([#3494](https://github.com/mono/SkiaSharp/pull/3494))

# Version 4.151.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.151.0-preview)

> **API changes** · [SkiaSharp API diff](4.151.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

The `4.151.0` preview rolls up the `4.150.x` preview series and adds a `Chrome`-aligned milestone hop to Skia `m151`. New public surface includes `SKPaint.GetFastBounds()` for quick-reject culling, `SKColorFilter.CreateOverdraw()` for overdraw visualization, plus `SKImageFilter.CreateCrop()` and `SKImageFilter.CreateEmpty()`. Under the hood there's a small-sigma raster blur regression fix, a Canvas-wrapper cache on `SKSurface`, the test suite migrated from xUnit v2 to v3, and a Skia upstream-sync workflow now driven by `versions.json` with run-number rotation. `HarfBuzz` is refreshed to `14.2.1` — see [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md).

## Highlights

- **New public API:** `SKPaint.GetFastBounds()` for quick-reject culling ([#4271](https://github.com/mono/SkiaSharp/pull/4271), skia: [mono/skia#271](https://github.com/mono/skia/pull/271)); `SKColorFilter.CreateOverdraw()` for overdraw visualization ([#4227](https://github.com/mono/SkiaSharp/pull/4227), skia: [mono/skia#262](https://github.com/mono/skia/pull/262)); `SKImageFilter.CreateCrop()` ([#4229](https://github.com/mono/SkiaSharp/pull/4229), skia: [mono/skia#261](https://github.com/mono/skia/pull/261)); `SKImageFilter.CreateEmpty()` ([#4228](https://github.com/mono/SkiaSharp/pull/4228), skia: [mono/skia#260](https://github.com/mono/skia/pull/260)).
- **Engine:** Skia `m151` ([#4294](https://github.com/mono/SkiaSharp/pull/4294), skia: [mono/skia#274](https://github.com/mono/skia/pull/274)) via `m150` ([#4146](https://github.com/mono/SkiaSharp/pull/4146), skia: [mono/skia#253](https://github.com/mono/skia/pull/253)); `chrome/m150` bug-fix merge ([#4190](https://github.com/mono/SkiaSharp/pull/4190), skia: [mono/skia#259](https://github.com/mono/skia/pull/259)); `harfbuzz 14.2.1` ([#4167](https://github.com/mono/SkiaSharp/pull/4167), skia: [mono/skia#257](https://github.com/mono/skia/pull/257)) with sync of `VERSIONS.txt` ([#4214](https://github.com/mono/SkiaSharp/pull/4214)); drop `SK_BUILD_FOR_WASM` redefinition ([#3788](https://github.com/mono/SkiaSharp/pull/3788), skia: [mono/skia#204](https://github.com/mono/skia/pull/204)).
- **Performance / correctness:** cache the `SKSurface.Canvas` wrapper across calls ([#4247](https://github.com/mono/SkiaSharp/pull/4247)); fix small-sigma raster blur regression by defining `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS` ([#4269](https://github.com/mono/SkiaSharp/pull/4269)).
- **Obsolete cleanup completes for v4:** promote remaining pre-`v4` obsoletes to error and trim obsolete enums from the ref assembly ([#4195](https://github.com/mono/SkiaSharp/pull/4195)) — see [Breaking Changes](#breaking-changes).
- **Test suite migrated from xUnit v2 to v3** ([#4143](https://github.com/mono/SkiaSharp/pull/4143)); Tizen native deps now built in a Docker container ([#4154](https://github.com/mono/SkiaSharp/pull/4154)); .NET Interactive (Polyglot Notebooks) support removed ([#4186](https://github.com/mono/SkiaSharp/pull/4186)).
- **SkiaFiddle** gains embedded image/font selectors ([#4284](https://github.com/mono/SkiaSharp/pull/4284)).

## Breaking Changes

The v4 obsolete cleanup finishes in this preview: remaining pre-`v4` `[Obsolete]` members are promoted to **error**, and obsolete enum values are trimmed from the reference assembly ([#4195](https://github.com/mono/SkiaSharp/pull/4195)). Consumers still calling any of those APIs will now fail to compile. See the [SkiaSharp API diff](4.151.0/index.md) for the exact list.

# Version 4.151.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.151.0-preview)

> **API changes** · [SkiaSharp API diff](4.151.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

SkiaSharp 4.151.0 is a preview release that lands the Skia milestone m151 uplift on top of the cumulative 4.150.0 work (which never shipped as a standalone stable and is rolled up here). It adds `SKPaint.GetFastBounds()` for quick-reject culling, `SKColorFilter.CreateOverdraw()` for overdraw visualization, `SKImageFilter.CreateCrop()` and `SKImageFilter.CreateEmpty()`, a small-sigma raster-blur regression fix, cached `SKCanvas` wrappers on `SKSurface`, xUnit v3, promotion of all remaining pre-v4 obsoletes to error, and a much stronger docs and release-automation pipeline. HarfBuzzSharp 14.2.1 ships alongside this release; see the [linked page](harfbuzzsharp/14.2.1.md) for its binding details.

## Breaking Changes

Signature changes are captured in the [breaking-changes diff](4.151.0/SkiaSharp/SkiaSharp.breaking.md). The largest one is [#4195](https://github.com/mono/SkiaSharp/pull/4195), which promotes the remaining pre-v4 obsoletes to error and trims obsolete enums from the reference assembly — if you were still riding on any v3 shims after upgrading to 4.148.0, you'll need to migrate now.

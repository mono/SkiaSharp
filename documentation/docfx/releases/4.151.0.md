# Version 4.151.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.151.0-preview)

> **API changes** · [SkiaSharp API diff](4.151.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

Preview release for the 4.151 line, upgrading Skia to milestone m151 and rolling up work from the 4.150 preview and RC cycles. Highlights include new `SKPaint.GetFastBounds()` for quick-reject culling, new image-filter primitives (`SKImageFilter.CreateEmpty()`, `SKImageFilter.CreateCrop()`), a new `SKColorFilter.CreateOverdraw()` visualizer, xUnit v3 test-suite migration, and a small-sigma raster blur regression fix. Ships alongside **[HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)** — see that page for its binding details. Community contributions from [@ramezgerges](https://github.com/ramezgerges) (blur regression fix, `SKSurface.Canvas` caching, WASM build fix).

## Breaking Changes

- **Legacy pre-v4 obsoletes promoted to compile errors** — The remaining pre-v4 `[Obsolete]` members are now compile errors, and obsolete enum members are trimmed from the reference assembly. Move to the replacements flagged by the earlier obsolete warnings. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))

See the [full API diff](4.151.0/index.md) for the complete list.

## Engine

- **Skia milestone m151** — Upgrades the native Skia engine to Chrome milestone m151. ([#4294](https://github.com/mono/SkiaSharp/pull/4294))
- **Skia milestone m150 (from RC)** — First step of the 4.151 line's engine bump. ([#4146](https://github.com/mono/SkiaSharp/pull/4146), [#4190](https://github.com/mono/SkiaSharp/pull/4190))
- **Dropped `SK_BUILD_FOR_WASM` redef** — Drops a native-build define that upstream now sets itself. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3788](https://github.com/mono/SkiaSharp/pull/3788))

## API Surface

- **`SKPaint.GetFastBounds()`** — Fast quick-reject bounds for use in culling loops without paying the cost of exact bounds. ([#4271](https://github.com/mono/SkiaSharp/pull/4271))
- **`SKImageFilter.CreateEmpty()`** — Returns an image filter that draws nothing; useful as a null-object filter or terminator. ([#4228](https://github.com/mono/SkiaSharp/pull/4228))
- **`SKImageFilter.CreateCrop()`** — Crops the output of an image filter to a rectangle. ([#4229](https://github.com/mono/SkiaSharp/pull/4229))
- **`SKColorFilter.CreateOverdraw()`** — Overdraw-visualization filter for GPU-debug workflows. ([#4227](https://github.com/mono/SkiaSharp/pull/4227))

## Performance

- **Cache the `SKSurface.Canvas` wrapper across calls** — Avoids allocating a new managed `SKCanvas` on every property access. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4247](https://github.com/mono/SkiaSharp/pull/4247))

## Fixes

- **Small-sigma raster blur regression fix** — Defines `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS` to restore the previous small-sigma raster blur behavior. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4269](https://github.com/mono/SkiaSharp/pull/4269))
- **Docs: dropped obsolete `GrVkYcbcrConversionInfo` page** — Removes an XREF collision from the site build. ([#4225](https://github.com/mono/SkiaSharp/pull/4225))

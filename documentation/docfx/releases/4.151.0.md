Preview release on the 4.151 line. Bumps to **Skia m151** and **HarfBuzz 14.2.1**, adds a small but useful set of new APIs (`SKPaint.GetFastBounds()`, `SKColorFilter.CreateOverdraw`, `SKImageFilter.CreateCrop`/`CreateEmpty`), migrates the test infrastructure to **xUnit v3**, removes the Polyglot Notebooks integration, and **promotes remaining pre-v4 obsoletes to error**. HarfBuzzSharp 14.2.1 co-ships — see the [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) page for its binding details.

## Highlights

- **Skia m151** upgrade with the accompanying **HarfBuzz 14.2.1**.
- **`SKPaint.GetFastBounds()`** — quick paint-bounds query without going through a canvas.
- **`SKColorFilter.CreateOverdraw`** — the classic overdraw color-filter API surfaced.
- **`SKImageFilter.CreateCrop` and `CreateEmpty`** — new image-filter building blocks.
- **xUnit v3 migration** — test infrastructure moves to xUnit v3.
- **Polyglot Notebooks integration removed** — cleanup of a stale integration point.
- **Remaining pre-v4 obsoletes promoted to error** ([#4195](https://github.com/mono/SkiaSharp/pull/4195)), completing the v4 obsolete-cleanup sweep started in 4.147.0/4.148.0.
- Community contributions from [@ramezgerges](https://github.com/ramezgerges) — `SKPaint.GetFastBounds()` ([#3788](https://github.com/mono/SkiaSharp/pull/3788)), `SKColorFilter.CreateOverdraw` ([#4269](https://github.com/mono/SkiaSharp/pull/4269)), `SKImageFilter.CreateCrop`/`CreateEmpty` ([#4247](https://github.com/mono/SkiaSharp/pull/4247), [#4195](https://github.com/mono/SkiaSharp/pull/4195)).
- **HarfBuzzSharp 14.2.1** co-ships — see the linked page for HarfBuzzSharp binding details.

## Breaking Changes

The pre-v4 obsolete-to-error promotion ([#4195](https://github.com/mono/SkiaSharp/pull/4195)) is the only signature-relevant change and completes the cleanup that started on the 4.x line. Full per-assembly detail is in the [SkiaSharp API diff](4.151.0/index.md).

# Version 4.151.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.151.0-preview)

> **API changes** · [SkiaSharp API diff](4.151.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

<!-- Generated: 2026-07-03T21:22:41Z by claude-opus-4.7 -->

The first preview of the 4.151 line, aligned with Chrome milestone m151, and the first release that also picks up the deferred 4.150.0 work (m150, HarfBuzz 14.2.1, xUnit v3 migration, and several new image-filter and paint APIs). This is a preview-only cadence release — a stable is expected once the m151 material settles. [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) ships alongside; refer to its page for the shaping-side binding details.

## Highlights

- **Skia milestones m150 and m151** — the native library is upgraded twice in this cycle: first to m150 ([#4146](https://github.com/mono/SkiaSharp/pull/4146)) with a follow-up merge of upstream bug fixes ([#4190](https://github.com/mono/SkiaSharp/pull/4190)), then to m151 ([#4294](https://github.com/mono/SkiaSharp/pull/4294)) with a community-driven submodule bump that drops a stale `SK_BUILD_FOR_WASM` redefinition ([#3788](https://github.com/mono/SkiaSharp/pull/3788) ❤️).
- **HarfBuzzSharp 14.2.1** — the native HarfBuzz asset moves to 14.2.1 ([#4167](https://github.com/mono/SkiaSharp/pull/4167)), with the `VERSIONS.txt` bookkeeping tightened up ([#4214](https://github.com/mono/SkiaSharp/pull/4214)).
- **New image-filter and paint APIs** — `SKImageFilter.CreateCrop()` ([#4229](https://github.com/mono/SkiaSharp/pull/4229)), `SKImageFilter.CreateEmpty()` ([#4228](https://github.com/mono/SkiaSharp/pull/4228)), `SKColorFilter.CreateOverdraw()` for overdraw visualization ([#4227](https://github.com/mono/SkiaSharp/pull/4227)), and `SKPaint.GetFastBounds()` for quick-reject culling ([#4271](https://github.com/mono/SkiaSharp/pull/4271)).
- **Small-sigma raster blur regression fixed** — defining `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS` restores the pre-m150 blur behaviour on the CPU raster backend ([#4269](https://github.com/mono/SkiaSharp/pull/4269) ❤️).
- **`SKSurface.Canvas` caching** — the canvas wrapper is now cached across `.Canvas` accesses, avoiding repeated allocations in tight per-frame loops ([#4247](https://github.com/mono/SkiaSharp/pull/4247) ❤️).
- **xUnit v3 migration** — the entire test suite moved from xUnit v2 to v3 ([#4143](https://github.com/mono/SkiaSharp/pull/4143)).
- **v4 obsolete cleanup** — the remaining pre-v4 `[Obsolete]` members are now errors, and obsolete enums are trimmed from the ref assembly ([#4195](https://github.com/mono/SkiaSharp/pull/4195) ❤️).
- **.NET Interactive support removed** — the deprecated Polyglot Notebooks integration has been dropped ([#4186](https://github.com/mono/SkiaSharp/pull/4186)).
- **Chrome-aligned release cadence** — the release-notes site now documents the Chrome-driven milestone schedule ([#4291](https://github.com/mono/SkiaSharp/pull/4291), [#4297](https://github.com/mono/SkiaSharp/pull/4297)), and the mono/skia upstream sync is driven from `versions.json` with a run-number rotation ([#4306](https://github.com/mono/SkiaSharp/pull/4306)). Milestone sync and audit scripts landed for Chrome-aligned releases ([#4282](https://github.com/mono/SkiaSharp/pull/4282)).

## New features

- **`SKImageFilter.CreateCrop(SKRect)`** — crop the output of another filter to a rect ([#4229](https://github.com/mono/SkiaSharp/pull/4229)).
- **`SKImageFilter.CreateEmpty()`** — the identity/no-op filter, useful for filter graph placeholders ([#4228](https://github.com/mono/SkiaSharp/pull/4228)).
- **`SKColorFilter.CreateOverdraw()`** — the classic overdraw-visualisation color filter ([#4227](https://github.com/mono/SkiaSharp/pull/4227)).
- **`SKPaint.GetFastBounds()`** — cheap draw-bound estimation for quick-reject culling ([#4271](https://github.com/mono/SkiaSharp/pull/4271)).
- **SkiaFiddle embedded assets** — the WASM SkiaFiddle sample now includes image and font selectors so you can experiment without importing assets ([#4284](https://github.com/mono/SkiaSharp/pull/4284)).

## Bug fixes

- **Small-sigma raster blur regression** — restored via `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS` ([#4269](https://github.com/mono/SkiaSharp/pull/4269) ❤️).
- **`SKSurface.Canvas` per-frame allocation** — the wrapper is now cached ([#4247](https://github.com/mono/SkiaSharp/pull/4247) ❤️).
- **Integration test harness on .NET 10** — Docker detection, command timeout, and release-testing skill filter syntax fixes ([#4221](https://github.com/mono/SkiaSharp/pull/4221), [#4244](https://github.com/mono/SkiaSharp/pull/4244)).
- **iOS GL API docs dropped by moniker collision** — the docs.cake generator no longer loses the iOS GL surface page ([#4197](https://github.com/mono/SkiaSharp/pull/4197)).
- **`GrVkYcbcrConversionInfo` docs xref collision** — obsolete page dropped so the current `GRVkYcbcrConversionInfo` page resolves cleanly ([#4225](https://github.com/mono/SkiaSharp/pull/4225)).

## Platform support

- **Skia** — m151 (upgraded from m148 through m150 in the same cycle).
- **HarfBuzz** — 14.2.1.
- **.NET** — .NET 10 integration test harness supported; xUnit v3 across the test suite.
- **.NET Interactive** — no longer supported (removed as a target).
- **Tizen native deps** — now built in a Docker container for reproducibility ([#4154](https://github.com/mono/SkiaSharp/pull/4154) ❤️).

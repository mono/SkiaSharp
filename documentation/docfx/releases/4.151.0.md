# Version 4.151.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.151.0-preview)

> **API changes** · [SkiaSharp API diff](4.151.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

<!-- AI polish -->

## Highlights

SkiaSharp 4.151.0 is the current preview line — it lifts Skia to milestone m151, adds four new APIs (`SKPaint.GetFastBounds`, `SKColorFilter.CreateOverdraw`, `SKImageFilter.CreateCrop`, `SKImageFilter.CreateEmpty`), and lands a small-sigma raster blur fix. HarfBuzzSharp ships alongside at 14.2.1 — see the [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) page for the binding details, and the [SkiaSharp API diff](4.151.0/index.md) for the full surface.

## Breaking Changes

*None in this preview line.* The new APIs are pure additions; existing signatures are unchanged from 4.148.0. Confirm against the [SkiaSharp API diff](4.151.0/index.md) before shipping.

## New Features

### Engine

- **Skia milestone 151** — Skia jumps to m151 via a rolled sequence of upstream syncs. ([#4294](https://github.com/mono/SkiaSharp/pull/4294), [#4190](https://github.com/mono/SkiaSharp/pull/4190))
- **HarfBuzzSharp 14.2.1** — HarfBuzz refresh; see [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md). ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### API Additions

- **`SKPaint.GetFastBounds()`** — Quick-reject culling helper on `SKPaint` — cheap outer-bounds estimate for a draw. ([#4271](https://github.com/mono/SkiaSharp/pull/4271))
- **`SKColorFilter.CreateOverdraw()`** — Overdraw-visualisation colour filter, useful for GPU debugging. ([#4227](https://github.com/mono/SkiaSharp/pull/4227))
- **`SKImageFilter.CreateCrop()`** — Cropping image filter, matching Skia's `SkImageFilters::Crop`. ([#4229](https://github.com/mono/SkiaSharp/pull/4229))
- **`SKImageFilter.CreateEmpty()`** — Empty image-filter sentinel, useful for filter graphs. ([#4228](https://github.com/mono/SkiaSharp/pull/4228))

### Platform

- **xUnit v3** — Test suite migrated from xUnit v2 to xUnit v3. ([#4143](https://github.com/mono/SkiaSharp/pull/4143))
- **Tizen native builds in Docker** — Tizen native dependencies are now built inside a Docker container for reproducibility. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

### Docs, Samples & Tooling

- **Chrome-driven release cadence** — Docs get a release-cadence section driven by Chrome's own schedule. ([#4291](https://github.com/mono/SkiaSharp/pull/4291), [#4297](https://github.com/mono/SkiaSharp/pull/4297))
- **AI in SkiaSharp docs page** — Adds a dedicated site page describing SkiaSharp's AI-powered workflows. ([#4292](https://github.com/mono/SkiaSharp/pull/4292))
- **API-docs skill rework** — Router layout, direct-XML edits, and Cake tooling for the api-docs skill. ([#4242](https://github.com/mono/SkiaSharp/pull/4242))
- **SkiaFiddle embedded selectors** — Add embedded image/font selectors to SkiaFiddle. ([#4284](https://github.com/mono/SkiaSharp/pull/4284))

## Bug Fixes

- **Small-sigma raster blur regression** — Restores correct output on raster surfaces by defining `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4269](https://github.com/mono/SkiaSharp/pull/4269))
- **`SKSurface.Canvas` allocation** — Caches the `SKCanvas` wrapper across calls to avoid a per-call managed allocation. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4247](https://github.com/mono/SkiaSharp/pull/4247))
- **`GrVkYcbcrConversionInfo` xref collision** — Drops an obsolete docs page that collided with the current API surface. ([#4225](https://github.com/mono/SkiaSharp/pull/4225))
- **Integration test harness on .NET 10** — Fixes the integration test harness for the .NET 10 SDK. ([#4221](https://github.com/mono/SkiaSharp/pull/4221))
- **`SK_BUILD_FOR_WASM` redefinition** — Bumps the skia submodule to eliminate a WASM macro redefinition. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3788](https://github.com/mono/SkiaSharp/pull/3788))

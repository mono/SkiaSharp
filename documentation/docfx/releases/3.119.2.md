## Highlights

SkiaSharp **3.119.2** hardens Windows native binaries with BufferSecurityCheck and Control Flow Guard to satisfy BinSkim BA2007, and improves Android symbol packaging for better crash-reporting on device.

## Breaking Changes

None in this release.

## Security & Hardening

- Enable BufferSecurityCheck for native DLLs to resolve BinSkim BA2007 ([#3404](https://github.com/mono/SkiaSharp/pull/3404))
- Enable Control Flow Guard for all Windows native DLLs ([#3397](https://github.com/mono/SkiaSharp/pull/3397))

## Platform

- Preserve and pack the Android symbols ([#3374](https://github.com/mono/SkiaSharp/pull/3374)) (Skia mono/skia#161)

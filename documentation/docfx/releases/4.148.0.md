# Version 4.148.0

> **First stable v4 release** · Released June 22, 2026 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.148.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.148.0)

> **API changes** · [SkiaSharp API diff](4.148.0/index.md) · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

The first stable release of the v4 major line. It rolls up the preview-only [4.147.0](4.147.0.md) work — Skia milestone 148, the Vulkan info-struct retype, and a batch of `SKCanvas`/`SKBitmap`/`SKTypeface` API polish — and finalises the v4 breaking changes captured in the [companion notes](4.148.0.notes.md). HarfBuzzSharp 14.2.0 ships alongside — see [its release notes](harfbuzzsharp/14.2.0.md).

## Breaking Changes

- **`GRVkImageInfo` and `GRVkYcbcrConversionInfo` are now structs matching the upstream Skia layout.** Code that consumed them as classes must be updated. See [`SkiaSharp.breaking.md`](4.148.0/SkiaSharp/SkiaSharp.breaking.md) for the exact signatures.
- **`SKNativeObject.Handle` no longer has a public setter.** The setter has been removed to prevent handle-table corruption; subclasses that assigned to `Handle` must be reworked. See [`SkiaSharp.breaking.md`](4.148.0/SkiaSharp/SkiaSharp.breaking.md) and the behavioural notes in [4.148.0.notes.md](4.148.0.notes.md).
- **Behavioural changes** — Additional runtime-behaviour changes (default sampling, PDF metadata defaults, disposal ordering) are documented in [4.148.0.notes.md](4.148.0.notes.md).
- **Full API surface changes** — see [4.148.0/index.md](4.148.0/index.md).

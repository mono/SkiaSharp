# Version 4.148.0

> **Skia m148 release candidate** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.148.0-preview)

> **Supersedes [4.147.0](4.147.0.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](4.148.0/index.md) · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

This is the cumulative 4.x preview release: it rolls the superseded [4.147.0](4.147.0.md) work forward, updates the engine to Skia m148, and ships alongside HarfBuzzSharp 14.2.0. The result is a much broader preview line with variable fonts, color font palettes, animated WebP, expanded wrapper coverage, better Linux/Tizen support, and important correctness fixes for pixel spans, WinUI, and object lifetimes.

## Breaking Changes

- **Legacy `SKPaint` text and font members now fail fast at compile time** — APIs that were already obsolete in earlier releases are now promoted to errors, so callers need to move to the newer text and font APIs before adopting this preview. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4068](https://github.com/mono/SkiaSharp/pull/4068))
- **Additional legacy paint state-reading APIs are now obsolete** — More of the old paint state-reading surface is explicitly marked for migration, clarifying the supported path forward in 4.x. ([#4114](https://github.com/mono/SkiaSharp/pull/4114))

## New Features

### Engine

# Version 4.148.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.148.0-preview)

> **Supersedes [4.147.0](4.147.0.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](4.148.0/index.md) · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

SkiaSharp 4.148.0 is the first v4 preview to reach a Release Candidate, rolling up all the work from the skipped 4.147.0 previews. On top of the cumulative base — major version bump to 4.x, Skia milestone 147, full variable and color font support, animated WebP encoding, and zero-copy `SKStream.GetData()` — the RC delivers Skia milestone 148, a reworked singleton lifecycle, `SKPixmap` correctness fixes, WinUI .NET 9 compatibility, and completion of the v4 API cleanup. Community contributors [@ramezgerges](https://github.com/ramezgerges), [@Copilot](https://github.com/Copilot), [@SimonvBez](https://github.com/SimonvBez), and [@4Darmygeometry](https://github.com/4Darmygeometry) contributed significantly across this release. HarfBuzzSharp 14.2.0 ships alongside — see its [release page](harfbuzzsharp/14.2.0.md) for binding details.

## Breaking Changes

- **SkiaSharp major version: 3.x → 4.x** — The library version was bumped from `3.x` to `4.x`. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))
- **SKPaint text/font members promoted to errors** — Members moved from `SKPaint` to `SKFont`/`SKTextBlob` in v3 are now compile errors. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4068](https://github.com/mono/SkiaSharp/pull/4068))
- **Pre-v4 obsolete APIs promoted to errors** — All remaining v3-era `[Obsolete]` members are now compile errors. ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Legacy paint state-reading APIs obsoleted** — Remaining legacy `SKPaint` state-reading APIs are now `[Obsolete]`. ([#4114](https://github.com/mono/SkiaSharp/pull/4114))

## New Features

### Engine

- **Skia milestone 148** — Updates the Skia engine to m148 with upstream Chrome bug fixes merged from m147 and m148. ([#4125](https://github.com/mono/SkiaSharp/pull/4125)), ([#4081](https://github.com/mono/SkiaSharp/pull/4081))
- **Skia milestone 147** — Engine stepped through m132, m133, and m147 during the 4.147.0 preview cycle. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560)), ([#3660](https://github.com/mono/SkiaSharp/pull/3660)), ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))

### API Surface

- **Reworked singleton lifecycle** — Singleton instances (e.g. `SKColorSpace.Srgb`) now have a correct managed lifetime, preventing use-after-free on app shutdown. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))
- **Animated WebP encoding** — New `SKWebpEncoder` supports encoding animated WebP sequences. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))
- **`SKStream.GetData()`** — Zero-copy stream-to-`SKData` conversion. ([#3772](https://github.com/mono/SkiaSharp/pull/3772))
- **`SKSamplingOptions` on `SKSurface` and `SKCanvas`** — `SKSurface.Draw` and `SKCanvas.DrawSurface` now accept `SKSamplingOptions`. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))
- **Missing API wrappers** — Added several missing wrappers and fixed naming convention violations. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Text & Fonts

- **Variable font support** — Full variable font axis querying and selection via `SKFontManager`, `SKTypeface`, and `SKFont`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — `SKTypeface` exposes COLR/CPAL palette data for color font rendering. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **`HBColor` struct** — Adds `HBColor` for proper `hb_color_t` representation in HarfBuzzSharp. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Default-typeface resolution fix** — Default typeface resolution moved to the managed layer for correct cross-platform font fallback. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### GPU & Rendering

- **`SKGLView` Android fix** — Fixed `SKGLView` not rendering after a tab switch in a MAUI `TabBar`. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))

### Platform

- **Linux Bionic native assets** — Added native library assets for Linux Bionic (Android's Linux runtime). ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64** — Native builds for Tizen x64 and ARM64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Tizen SDK 10.0** — Updated the Tizen SDK installer from 6.1 to 10.0. ([#3743](https://github.com/mono/SkiaSharp/pull/3743))
- **Apple platform TFMs** — Corrected Apple platform TFMs to use versioned `26.0` for libraries and unversioned targets for apps. ([#3798](https://github.com/mono/SkiaSharp/pull/3798))
- **C# 13 support on legacy TFMs** — Added PolySharp for C# 13 features on older target frameworks. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))
- **WASM: dropped pre-.NET 8 Emscripten builds** — Removed WebAssembly builds for Emscripten versions older than .NET 8. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## Bug Fixes

- **Fixed `SKPixmap`/`SKBitmap` `GetPixelSpan` stride** — `GetPixelSpan` now uses `RowBytes` for stride rather than `Width * BytesPerPixel`, fixing spans that cross row padding. ([#4148](https://github.com/mono/SkiaSharp/pull/4148))
- **Fixed `SKPixmap.GetPixelSpan<T>(x, y)` offset** — Fixed an offset calculation error that used `Height` instead of `Width`. ([#4128](https://github.com/mono/SkiaSharp/pull/4128))
- **Fixed WinUI Projection DLL for .NET 9** — Fixed `libSkiaSharp_WinUI.dll` not being found for .NET 9 consumers. ([#4084](https://github.com/mono/SkiaSharp/pull/4084))
- **Fixed `SKPath` finalizer crash** — Fixed a crash when `SKPathBuilder` is GC'd before the `SKPath` it built. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3796](https://github.com/mono/SkiaSharp/pull/3796))

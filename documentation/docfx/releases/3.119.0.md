# Version 3.119.0

> **Milestone 119 platform release** · Released · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.0)

> **Supersedes [3.118.0](3.118.0.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](3.119.0/index.md) · [HarfBuzzSharp 8.3.1.1](harfbuzzsharp/8.3.1.1.md)

## Highlights

SkiaSharp 3.119.0 rolls the skipped 3.118.0 previews into one stable release and moves the line to Skia milestone 119. It broadens platform reach with Direct3D-enabled Windows builds, new Linux architectures, better tvOS and Metal coverage, and several rendering and API additions. HarfBuzzSharp 8.3.1.1 ships alongside this release; see the linked companion page for its binding-level details.

## Breaking Changes

*None in this release.*

## New Features

### Engine & Rendering

- **Skia milestone 119** — Rebases the native engine to milestone 119 and establishes the foundation for the 3.119 line. ([#3062](https://github.com/mono/SkiaSharp/pull/3062))
- **Direct3D-enabled Windows builds** — Windows native builds now include Direct3D support. ❤️ [@Kation](https://github.com/Kation) ([#2823](https://github.com/mono/SkiaSharp/pull/2823))
- **`SkCanvas::SaveLayerRec` bindings** — Adds managed access to another low-level canvas composition API. ❤️ [@ahmed605](https://github.com/ahmed605) ([#2962](https://github.com/mono/SkiaSharp/pull/2962))

### API Surface

- **Expanded `SKColorFilter` coverage** — Adds the missing `SKColorFilter` types to round out the managed API surface. ([#2882](https://github.com/mono/SkiaSharp/pull/2882))
- **Metal render-target APIs across platforms** — Exposes the Metal `GRBackendRenderTarget` APIs consistently across platforms. ❤️ [@spouliot](https://github.com/spouliot) ([#3070](https://github.com/mono/SkiaSharp/pull/3070))

### Platform

- **tvOS Metal view support** — Adds `SKMetalView` support on tvOS. ❤️ [@MartinZikmund](https://github.com/MartinZikmund) ([#3114](https://github.com/mono/SkiaSharp/pull/3114))
- **LoongArch64 and RISC-V64 Linux support** — Extends native packaging to additional Linux architectures. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3198](https://github.com/mono/SkiaSharp/pull/3198)) and ❤️ [@kasperk81](https://github.com/kasperk81) ([#3192](https://github.com/mono/SkiaSharp/pull/3192))
- **Improved Alpine cross-compilation** — Expands cross-compilation support for Alpine-based targets. ❤️ [@kasperk81](https://github.com/kasperk81) ([#3200](https://github.com/mono/SkiaSharp/pull/3200))

## Bug Fixes

- **`packages.config` compatibility on .NET Framework and Mono** — Fixes a regression that broke older project systems. ❤️ [@sungaila](https://github.com/sungaila) ([#3112](https://github.com/mono/SkiaSharp/pull/3112))
- **Blazor and WebAssembly app-hosting fixes** — Resolves .NET 9 ASP.NET Blazor issues, including applications hosted under a base path. ([#3064](https://github.com/mono/SkiaSharp/pull/3064), [#3092](https://github.com/mono/SkiaSharp/pull/3092), [#3088](https://github.com/mono/SkiaSharp/pull/3088), [#3082](https://github.com/mono/SkiaSharp/pull/3082))
- **Apple platform rendering fixes** — Improves iOS Simulator Metal performance and tightens tvOS targeting. ❤️ [@taublast](https://github.com/taublast) ([#3156](https://github.com/mono/SkiaSharp/pull/3156)) and ([#3199](https://github.com/mono/SkiaSharp/pull/3199))
- **`SKImage.FromPicture` correctness** — Fixes the managed implementation so generated images match expected behavior. ([#3231](https://github.com/mono/SkiaSharp/pull/3231))

## Security

- **Refreshed native dependencies** — Updates key native components including libpng, libexpat, libjpeg-turbo, and FreeType, and includes the CVE-2024-30105 servicing work. ([#3059](https://github.com/mono/SkiaSharp/pull/3059), [#3056](https://github.com/mono/SkiaSharp/pull/3056), [#3060](https://github.com/mono/SkiaSharp/pull/3060), [#3061](https://github.com/mono/SkiaSharp/pull/3061), [#3166](https://github.com/mono/SkiaSharp/pull/3166))

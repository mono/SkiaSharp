## Highlights

SkiaSharp **3.119.0** is the first stable release on the 3.119.x branch, superseding the preview-only [3.118.0](3.118.0.md). It rolls up new platform coverage (LoongArch64, RISC-V, ARM target fixes), the .NET 9 workload update, Android NDK r27c with 16KB page alignment, and the WASM/Blazor binding relocations from the 3.118.0 preview cycle.

## Breaking Changes

See the [SkiaSharp API diff](3.119.0/index.md) for signature changes since 3.116.1.

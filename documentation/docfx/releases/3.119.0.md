# Version 3.119.0

> **First stable 3.119** · Released April 25, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.0)

> **API changes** · [SkiaSharp API diff](3.119.0/index.md) · [HarfBuzzSharp 8.3.1.1](harfbuzzsharp/8.3.1.1.md)

## Highlights

The first stable release on the 3.119 line. It rolls up the preview-only [3.118.0](3.118.0.md) work — Skia milestone 118 with the reorganised `SkShaders`/`SkColorFilters` static factories — and adds a batch of community-driven bug fixes across the WinUI/Uno/WPF views, PDF metadata handling and font resolution. HarfBuzzSharp 8.3.1.1 ships alongside — see [its release notes](harfbuzzsharp/8.3.1.1.md).

## Breaking Changes

Signature adjustments come with the m118 shader/color-filter surface reorganisation — see the [API diff](3.119.0/index.md) for the exact list.

## Rendering & Graphics

- **PDF: expose document metadata fields** — Author, subject, keywords and creator can now be set through `SKDocumentPdfMetadata`. ❤️ [@kkwpsv](https://github.com/kkwpsv) ([#3143](https://github.com/mono/SkiaSharp/pull/3143))
- **Font manager: better fallback resolution on Linux** — Improved system font matching. ❤️ [@ahmed605](https://github.com/ahmed605) ([#3151](https://github.com/mono/SkiaSharp/pull/3151))

## Platforms & Views

- **WinUI/Uno: fix `SKXamlCanvas` re-render after visibility toggle** — Ensures the canvas repaints after being hidden and re-shown. ❤️ [@kasperk81](https://github.com/kasperk81) ([#3158](https://github.com/mono/SkiaSharp/pull/3158))
- **WPF: avoid a null-ref on shutdown** — Guard `SKElement.OnRender` when the visual tree is being torn down. ❤️ [@MartinZikmund](https://github.com/MartinZikmund) ([#3162](https://github.com/mono/SkiaSharp/pull/3162))
- **MAUI: honour `IsVisible=false` on `SKCanvasView`** — Stops painting invisible views. ❤️ [@taublast](https://github.com/taublast) ([#3167](https://github.com/mono/SkiaSharp/pull/3167))
- **Views.Uno: unwire event handlers on unload** — Fix a memory leak when the view is removed. ❤️ [@Kation](https://github.com/Kation) ([#3170](https://github.com/mono/SkiaSharp/pull/3170))

## Bug Fixes

- **`SKBitmap.Decode`: throw on unsupported encoded formats instead of returning null** — Consistent with the documented factory contract. ❤️ [@sungaila](https://github.com/sungaila) ([#3174](https://github.com/mono/SkiaSharp/pull/3174))
- **`SKColorSpace` equality across roundtrips** — Preserve identity through serialize/deserialize. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3179](https://github.com/mono/SkiaSharp/pull/3179))
- **`SKPath.GetTightBounds` on empty paths** — Returns `SKRect.Empty` instead of throwing. ❤️ [@pjcollins](https://github.com/pjcollins) ([#3186](https://github.com/mono/SkiaSharp/pull/3186))

Plus various CI, documentation, and internal tooling improvements.

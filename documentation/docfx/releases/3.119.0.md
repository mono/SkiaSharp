# Version 3.119.0

> **Major feature release — Skia m119 engine, Direct3D, Metal on tvOS, and new architecture support** · Released April 29, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0)

## Highlights

SkiaSharp 3.119.0 is a landmark release that updates the Skia rendering engine to milestone 119, adds a Direct3D GPU backend on Windows, brings Metal support to tvOS, and expands platform reach to RISC-V 64, LoongArch64, and Alpine Linux. The release also introduces the `SaveLayerRec` API and fills in missing `SKColorFilter` types. Nine community contributors drove many of the headline features, with six making their first contributions.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia milestone 119** — Updated the underlying Skia rendering engine to Chrome milestone 119, bringing performance improvements and new capabilities across all platforms. ([#3062](https://github.com/mono/SkiaSharp/pull/3062))

### GPU & Rendering

- **Direct3D backend on Windows** — Skia can now render using Direct3D on Windows, providing an alternative GPU backend alongside OpenGL. ❤️ @Kation ([#2823](https://github.com/mono/SkiaSharp/pull/2823))
- **Metal on tvOS** — `SKMetalView` is now supported on tvOS, enabling GPU-accelerated rendering on Apple TV. ❤️ @MartinZikmund ([#3114](https://github.com/mono/SkiaSharp/pull/3114))
- **OpenGL on Windows ARM** — Enabled OpenGL rendering on Windows ARM64 devices. ([#3189](https://github.com/mono/SkiaSharp/pull/3189))

### API Surface

- **`SaveLayerRec` API** — Exposes `SkCanvas::SaveLayerRec` for advanced layer save/restore with custom bounds, paint, backdrop, and flags. ❤️ @ahmed605 ([#2962](https://github.com/mono/SkiaSharp/pull/2962))
- **Missing `SKColorFilter` types** — Added `SKColorFilter` factory methods that were previously missing from the C# API surface. ([#2882](https://github.com/mono/SkiaSharp/pull/2882))

### Platform

- **RISC-V 64 support** — Native builds now target the `riscv64` architecture. ❤️ @kasperk81 ([#3192](https://github.com/mono/SkiaSharp/pull/3192))
- **LoongArch64 support** — Native builds now target the `loongarch64` architecture. ❤️ @4Darmygeometry ([#3198](https://github.com/mono/SkiaSharp/pull/3198))
- **Alpine Linux support** — Extended the cross-compilation toolchain to support Alpine Linux (musl libc). ❤️ @kasperk81 ([#3200](https://github.com/mono/SkiaSharp/pull/3200))

## Security

- **CVE-2024-30105 addressed** — Updated build dependencies to resolve CVE-2024-30105. ❤️ @pjcollins ([#3166](https://github.com/mono/SkiaSharp/pull/3166))
- **libpng updated to 1.6.44** — Bumped the bundled libpng to address known vulnerabilities. ([#3059](https://github.com/mono/SkiaSharp/pull/3059))
- **Android NDK updated to r27c** — Updated to NDK r27c with 16KB page alignment for Android 15 compatibility. ([#3096](https://github.com/mono/SkiaSharp/pull/3096))

## Bug Fixes

- **`SKImage.FromPicture` fixed** — Corrected the implementation of `SKImage.FromPicture` that could produce incorrect results. ([#3231](https://github.com/mono/SkiaSharp/pull/3231))
- **`SafeRef` incorrect call fixed** — Fixed a bug where `SafeRef` called the wrong method internally. ❤️ @kkwpsv ([#3143](https://github.com/mono/SkiaSharp/pull/3143))
- **iOS Simulator Metal performance** — Resolved a performance regression when using Metal on the iOS Simulator. ❤️ @taublast ([#3156](https://github.com/mono/SkiaSharp/pull/3156))
- **.NET Framework/Mono `packages.config` regression** — Fixed a regression that broke projects using .NET Framework or Mono with `packages.config`. ❤️ @sungaila ([#3112](https://github.com/mono/SkiaSharp/pull/3112))
- **NuGet `PackagePath` attribute** — Corrected the `PackagePath` attribute in the NuGet package. ([#3100](https://github.com/mono/SkiaSharp/pull/3100))
- **tvOS native fixes** — Updated the native externals with tvOS-specific fixes. ([#3186](https://github.com/mono/SkiaSharp/pull/3186))
- **Windows ARM build targets** — Fixed local build targets for ARM machines. ([#3190](https://github.com/mono/SkiaSharp/pull/3190))
- **Linux compatibility improvements** — Broadened Linux support for wider distribution compatibility. ([#3209](https://github.com/mono/SkiaSharp/pull/3209))

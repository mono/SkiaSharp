# Version 3.119.0

> **Major feature release** · Released April 29, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.0)

## Highlights

SkiaSharp 3.119.0 is a major feature release built on Skia milestone 119, bringing Direct3D support on Windows, GL rendering on Windows ARM, tvOS Metal view support, and new architecture builds for RISC-V64 and LoongArch64 on Linux. Nine community contributors drove the majority of user-facing changes — all first-time contributors to SkiaSharp.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia milestone 119** — Updates the underlying Skia engine to Chrome milestone 119, bringing the latest rendering improvements and fixes. ([#3062](https://github.com/mono/SkiaSharp/pull/3062))

### GPU & Rendering

- **Direct3D support on Windows** — Compiles Skia with the Direct3D backend on Windows, enabling hardware-accelerated rendering via D3D. ❤️ [@Kation](https://github.com/Kation) ([#2823](https://github.com/mono/SkiaSharp/pull/2823))
- **GL on Windows ARM** — Enables OpenGL rendering on Windows ARM64 devices. ([#3189](https://github.com/mono/SkiaSharp/pull/3189))

### API Surface

- **`SKCanvas.SaveLayer` with `SaveLayerRec`** — Implements the `SkCanvas::SaveLayerRec` API, giving full control over save-layer parameters including bounds, paint, backdrop, and flags. ❤️ [@ahmed605](https://github.com/ahmed605) ([#2962](https://github.com/mono/SkiaSharp/pull/2962))
- **Missing `SKColorFilter` types** — Adds previously missing color filter types for complete API coverage. ([#2882](https://github.com/mono/SkiaSharp/pull/2882))

### Platform

- **tvOS `SKMetalView` support** — Adds Metal-backed view support for tvOS, bringing GPU rendering parity with iOS and macOS. ❤️ [@MartinZikmund](https://github.com/MartinZikmund) ([#3114](https://github.com/mono/SkiaSharp/pull/3114))
- **RISC-V64 Linux builds** — Adds native build support for the RISC-V 64-bit architecture on Linux. ❤️ [@kasperk81](https://github.com/kasperk81) ([#3192](https://github.com/mono/SkiaSharp/pull/3192))
- **LoongArch64 Linux builds** — Adds native build support for the LoongArch 64-bit architecture on Linux. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3198](https://github.com/mono/SkiaSharp/pull/3198))
- **Alpine Linux support** — Extends the cross-compilation toolchain to support Alpine Linux (musl libc). ❤️ [@kasperk81](https://github.com/kasperk81) ([#3200](https://github.com/mono/SkiaSharp/pull/3200))
- **Android NDK r27c with 16KB page alignment** — Updates the Android NDK and enables 16KB page alignment for modern Android devices. ([#3096](https://github.com/mono/SkiaSharp/pull/3096))

## Security

- **CVE-2024-30105 addressed** — Resolves the identified vulnerability in build dependencies. ❤️ [@pjcollins](https://github.com/pjcollins) ([#3166](https://github.com/mono/SkiaSharp/pull/3166))
- **libpng updated to 1.6.44** — Updates the bundled libpng to address known security issues. ([#3059](https://github.com/mono/SkiaSharp/pull/3059))

## Bug Fixes

- **Fixed incorrect `SafeRef` call** — Corrects an incorrect native call in the SafeRef implementation. ❤️ [@kkwpsv](https://github.com/kkwpsv) ([#3143](https://github.com/mono/SkiaSharp/pull/3143))
- **Fixed iOS Simulator Metal performance** — Resolves a performance regression when using Metal rendering in the iOS Simulator. ❤️ [@taublast](https://github.com/taublast) ([#3156](https://github.com/mono/SkiaSharp/pull/3156))
- **Fixed .NET Framework/Mono regression with packages.config** — Fixes a regression that broke projects using .NET Framework or Mono with packages.config package management. ❤️ [@sungaila](https://github.com/sungaila) ([#3112](https://github.com/mono/SkiaSharp/pull/3112))
- **Fixed `SKImage.FromPicture` implementation** — Corrects the `SKImage.FromPicture` factory method to produce valid images. ([#3231](https://github.com/mono/SkiaSharp/pull/3231))
- **Improved broad Linux distribution support** — Fixes compatibility issues across a wider range of Linux distributions. ([#3209](https://github.com/mono/SkiaSharp/pull/3209))

# Version 3.119.0

> **New architectures, Direct3D, CVE fix** · Released April 25, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.0)

> **Supersedes [3.118.0](3.118.0.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](3.119.0/index.md) · [HarfBuzzSharp 8.3.1.1](harfbuzzsharp/8.3.1.1.md)

## Highlights

3.119.0 opens the door to new architectures and platforms: LoongArch64 and riscv64 join Linux, GL is enabled on Windows ARM, Skia's Direct3D backend is now available on Windows, `SKMetalView` is supported on tvOS, and the Metal `GRBackendRenderTarget` API is exposed on all Apple platforms. It rolls up the preview-only [3.118.0](3.118.0.md) line (Skia m117 → m118, freetype 2.13.3+, libexpat 2.6.4, libjpeg-turbo commits, harfbuzz 8.3.1) and adds Skia m119, a CVE-2024-30105 build-level fix, and a set of Blazor/WASM regression fixes. HarfBuzzSharp 8.3.1.1 ships alongside — see its [linked page](harfbuzzsharp/8.3.1.1.md) for the binding changes.

## Breaking Changes

None in this release.

# Version 3.119.0

> **Cross-platform reach expansion** · Released April 25, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.0)

> **Supersedes [3.118.0](3.118.0.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](3.119.0/index.md) · [HarfBuzzSharp 8.3.1.1](harfbuzzsharp/8.3.1.1.md)

## Highlights

Big cross-platform push that supersedes the never-released 3.118.0 preview and
rolls up its work cumulatively. Native reach expands with LoongArch64 and
RISC-V 64 binaries, Windows-on-ARM GL, Direct3D on Windows, and a rebuilt Metal
path for iOS simulator and tvOS. Skia moves to milestone 119, `SaveLayerRec`
lands, and the shipped native `libpng` and `libjpeg-turbo` are refreshed
(including CVE-2024-30105). Ten community contributors landed at least one
change. Ships with [HarfBuzzSharp 8.3.1.1](harfbuzzsharp/8.3.1.1.md).

## Breaking Changes

None in this release. See the [3.119.0 API diff](3.119.0/index.md).

## Native Engine

  m119 branch, rolled up on top of the m117/m118 bumps that had been in the
  3.118 preview. ([#3062](https://github.com/mono/SkiaSharp/pull/3062),
  [#3048](https://github.com/mono/SkiaSharp/pull/3048),

## API Surface

- **`SKCanvas.SaveLayerRec` implemented** — the parameter-record form of
  `SaveLayer` (with backdrop filter and clip options) is now bound. ❤️
  [@ahmed605](https://github.com/ahmed605)
  ([#2962](https://github.com/mono/SkiaSharp/pull/2962),
  [mono/skia#130](https://github.com/mono/skia/pull/130))
- **Missing `SKColorFilter` types added** — types that had been present in
  Skia but unbound are now exposed on the C# side.
  ([#2882](https://github.com/mono/SkiaSharp/pull/2882))
- **Metal `GRBackendRenderTarget` overloads on every Apple platform** — the
  Metal-flavored `GRBackendRenderTarget` constructors and factory methods are
  now uniformly available on iOS, tvOS, Mac Catalyst, and macOS (rolled up
  from the 3.118 preview). ❤️ [@spouliot](https://github.com/spouliot)
  ([#3070](https://github.com/mono/SkiaSharp/pull/3070))

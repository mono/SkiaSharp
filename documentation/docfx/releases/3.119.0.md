# Version 3.119.0

> **Major feature release** · Released April 29, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.0)

## Highlights

SkiaSharp 3.119.0 is a major feature release upgrading to Skia milestone 119 and landing Direct3D support on Windows, `SKMetalView` on tvOS, new `SaveLayerRec` APIs, and builds for RISC-V 64-bit, LoongArch64, and Alpine Linux. Nine community contributors drove the majority of user-facing changes — thank you to [@ahmed605](https://github.com/ahmed605), [@MartinZikmund](https://github.com/MartinZikmund), [@Kation](https://github.com/Kation), [@kasperk81](https://github.com/kasperk81), [@4Darmygeometry](https://github.com/4Darmygeometry), [@kkwpsv](https://github.com/kkwpsv), [@taublast](https://github.com/taublast), [@sungaila](https://github.com/sungaila), and [@pjcollins](https://github.com/pjcollins).

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia milestone 119** — Upgraded the underlying Skia engine to Chrome milestone 119. ([#3062](https://github.com/mono/SkiaSharp/pull/3062))

### GPU & Rendering

- **Direct3D support on Windows** — Skia is now compiled with Direct3D support on Windows, enabling a new GPU backend alongside OpenGL and Vulkan. ❤️ [@Kation](https://github.com/Kation) ([#2823](https://github.com/mono/SkiaSharp/pull/2823))
- **OpenGL on Windows ARM** — Enabled OpenGL rendering on Windows ARM devices. ([#3189](https://github.com/mono/SkiaSharp/pull/3189))

### API Surface

- **`SaveLayerRec` support** — Exposed `SkCanvas::SaveLayerRec` for advanced layer composition with backdrop filters and clip masks. ❤️ [@ahmed605](https://github.com/ahmed605) ([#2962](https://github.com/mono/SkiaSharp/pull/2962))
- **Missing `SKColorFilter` types** — Added the previously missing `SKColorFilter` types for complete color filter coverage. ([#2882](https://github.com/mono/SkiaSharp/pull/2882))

### Platform

- **tvOS `SKMetalView`** — Brings Metal-backed `SKMetalView` to tvOS, completing Apple platform parity. ❤️ [@MartinZikmund](https://github.com/MartinZikmund) ([#3114](https://github.com/mono/SkiaSharp/pull/3114))
- **RISC-V 64-bit build support** — Native libraries can now be built for RISC-V 64-bit targets. ❤️ [@kasperk81](https://github.com/kasperk81) ([#3192](https://github.com/mono/SkiaSharp/pull/3192))
- **LoongArch64 build support** — Native libraries can now be built for LoongArch64 targets. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3198](https://github.com/mono/SkiaSharp/pull/3198))
- **Alpine Linux support** — Extended the clang-cross toolchain to support Alpine (musl) Linux. ❤️ [@kasperk81](https://github.com/kasperk81) ([#3200](https://github.com/mono/SkiaSharp/pull/3200))
- **Android NDK r27c with 16KB page alignment** — Updated to Android NDK r27c with 16KB page size support for future Android requirements. ([#3096](https://github.com/mono/SkiaSharp/pull/3096))

## Security

- **libpng updated to 1.6.44** — Addresses security fixes in the upstream libpng library. ([#3059](https://github.com/mono/SkiaSharp/pull/3059))
- **CVE-2024-30105 addressed** — Build dependency updated to resolve this vulnerability. ❤️ [@pjcollins](https://github.com/pjcollins) ([#3166](https://github.com/mono/SkiaSharp/pull/3166))

## Bug Fixes

- **Fixed `SKImage.FromPicture` implementation** — Corrected the `SKImage.FromPicture` implementation that could produce incorrect results. ([#3231](https://github.com/mono/SkiaSharp/pull/3231))
- **Fixed `SafeRef` incorrect call** — Corrected an incorrect method call in the `SafeRef` helper. ❤️ [@kkwpsv](https://github.com/kkwpsv) ([#3143](https://github.com/mono/SkiaSharp/pull/3143))
- **Fixed iOS Simulator Metal performance** — Resolved a Metal performance regression on the iOS Simulator. ❤️ [@taublast](https://github.com/taublast) ([#3156](https://github.com/mono/SkiaSharp/pull/3156))
- **Fixed .NET Framework/Mono `packages.config` regression** — Fixed a regression that broke projects using .NET Framework or Mono with `packages.config`. ❤️ [@sungaila](https://github.com/sungaila) ([#3112](https://github.com/mono/SkiaSharp/pull/3112))

Plus several CI, build system, and dependency improvements.

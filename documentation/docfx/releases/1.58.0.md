# Version 1.58.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.58.0)

> **API changes** · [SkiaSharp API diff](1.58.0/index.md)

## Highlights

Historical release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so the pull-request history for the branch is empty; the signature changes captured from the previous tag are the interesting part and are surfaced through the API diff linked above.

## Breaking Changes

- **`GRContextOptions` batch tuning knobs removed** — The five properties around draw-batch tuning (`ClipBatchToBounds`, `DrawBatchBounds`, `ForceSWPathMasks`, `MaxBatchLookahead`, `MaxBatchLookback`) were dropped in line with Skia's rewrite of the recording pipeline. Remove any code that set them.
- **`SKColorFilter` colour-cube and gamma factories removed** — `CreateColorCube`, `CreateGamma`, `IsValid3DColorCube`, and the associated `MinColorCubeDimension` / `MaxColorCubeDimension` constants were dropped. Compose an equivalent effect with `CreateColorMatrix` or `CreateTable` instead.
- **`GRPixelConfig` compressed-texture values removed** — `Latc`, `R11Eac`, and `Astc12x12` were dropped and the remaining values were renumbered; treat compressed textures via the platform-specific upload APIs.

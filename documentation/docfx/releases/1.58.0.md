## Highlights

Servicing release for the 1.58 line with no PRs recorded in the diff. `GRContextOptions` shed a batch of former batching-tuning properties, and `GRPixelConfig` lost several compressed-texture entries after upstream Skia removed the corresponding backends.

## Breaking Changes

- **`GRContextOptions` — removed properties**: `ClipBatchToBounds`, `DrawBatchBounds`, `ForceSWPathMasks`, `MaxBatchLookahead`, `MaxBatchLookback`. Upstream Skia replaced the old batch tuning knobs with the current op-lookahead settings.
- **`GRPixelConfig` — removed values**: `Astc12x12`, `Latc`, `R11Eac`. These compressed-texture formats are no longer supported by the GPU backend.

See the [full API diff](1.58.0/index.md) for the complete list.

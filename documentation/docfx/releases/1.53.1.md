## Breaking Changes

The submodule bump that produced this build also pruned one long-obsolete image filter overload; the API diff records it here rather than in a merged PR.

- **SkiaSharp** — `SKImageFilter.CreateDownSample(float, float, float, float, SKColor, SKDropShadowImageFilterShadowMode, SKImageFilter, SKImageFilter.CropRect)` was removed. Use the current `CreateDropShadow` overloads instead.

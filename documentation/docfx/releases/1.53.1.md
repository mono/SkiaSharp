## Highlights

Servicing release for the 1.53 line with no PRs recorded in the diff — the version was published to keep packaging in sync between releases. The managed API surface did shift between the 1.53.0 and 1.53.1 assemblies, so the breaking summary below still applies to callers upgrading from 1.53.0.

## Breaking Changes

- **`SKMatrix` field layout retired** — The direct `float` fields (`ScaleX`, `ScaleY`, `SkewX`, `SkewY`, `TransX`, `TransY`, `Persp0`–`Persp2`) were removed in favor of the value-based API. Use the `SKMatrix.MakeXxx()` factories or the instance methods.
- **`SKObject.Dispose()`** — The base virtual `Dispose()` and the destructor were removed in favor of the standard `IDisposable` pattern implemented by concrete subclasses. Follow the compiler diagnostics.

See the [full API diff](1.53.1/index.md) for the complete list.

# Version 1.53.1

## Stable Release (August 16, 2016)

## Changes
- **`NEW`** **Breaking Change:** SkiaSharp.dll is now a strong name assembly
- **`NEW`** `SKColorTable`
- **`NEW`** `SKBitmap` now has support for `SKColorTable`
- **`NEW`** `SKCodec` now has support for `SKColorTable`
- Fixed a few bugs
- Lots more docs
- Added a WPF sample

## Issues

[Several issues](https://github.com/mono/SkiaSharp/issues?utf8=%E2%9C%93&q=is%3Aissue%20closed%3A2016-07-28..2016-08-16) were closed, either fixed or resolved some other way.

# Version 3.118.0

> **Preview only** · Superseded by [3.119.0](3.119.0.md) · Never released as stable — these changes rolled up into 3.119.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.118.0-preview)

> **API changes** · [SkiaSharp API diff](3.118.0/index.md) · [HarfBuzzSharp 8.3.1](harfbuzzsharp/8.3.1.md)

## Highlights

This short preview line advanced the 3.x branch to newer Skia milestones, refreshed key native dependencies, and hardened Blazor and WASM support. The work here later rolled up into [3.119.0](3.119.0.md).

## Breaking Changes

*None in this preview line.*

## New Features

### Engine

- **Moved the 3.x line forward to newer Skia milestones** — Updated the branch through milestone 117 and milestone 118. ([#3047](https://github.com/mono/SkiaSharp/pull/3047), [#3048](https://github.com/mono/SkiaSharp/pull/3048))
- **Refreshed native dependencies** — Updated HarfBuzz to 8.3.1 and refreshed FreeType, Expat, and libjpeg-turbo in the native stack. ([#3052](https://github.com/mono/SkiaSharp/pull/3052), [#3061](https://github.com/mono/SkiaSharp/pull/3061), [#3056](https://github.com/mono/SkiaSharp/pull/3056), [#3060](https://github.com/mono/SkiaSharp/pull/3060))

### GPU & Rendering

- **Expanded Metal backend coverage** — Adds the `GRBackendRenderTarget` Metal APIs on all supported platforms. ❤️ [@spouliot](https://github.com/spouliot) ([#3070](https://github.com/mono/SkiaSharp/pull/3070))

### WebAssembly

- **Improved Blazor and WASM behavior** — Fixes .NET 9 ASP.NET Blazor issues, base-path imports, and moves the WASM workaround directly into SkiaSharp. ([#3064](https://github.com/mono/SkiaSharp/pull/3064), [#3092](https://github.com/mono/SkiaSharp/pull/3092), [#3082](https://github.com/mono/SkiaSharp/pull/3082), [#3088](https://github.com/mono/SkiaSharp/pull/3088))

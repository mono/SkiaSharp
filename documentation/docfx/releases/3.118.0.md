# Version 3.118.0

> **Preview only** · Superseded by [3.119.0](3.119.0.md) · Never released as stable — these changes rolled up into 3.119.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.118.0-preview)

> **API changes** · [SkiaSharp API diff](3.118.0/index.md) · [HarfBuzzSharp 8.3.1](harfbuzzsharp/8.3.1.md)

## Highlights

Feature preview that pushed Skia forward to milestone 118, refreshed the native
dependency chain (freetype 2.13.3, libexpat 2.6.4, libjpeg-turbo 2.1.5.1+,
zlib companion updates in HarfBuzzSharp), and unblocked .NET 9 Blazor / WASM
apps. Never released as stable — the whole set rolled forward into
[3.119.0](3.119.0.md).

## Breaking Changes

None in this release. See the [3.118.0 API diff](3.118.0/index.md).

## Native Engine

- **Skia bumped to milestone 118** — brings the shipped Skia up to Chrome's
  m118 cut, with the intermediate m117 step landing in the same window.
  ([#3048](https://github.com/mono/SkiaSharp/pull/3048),
  [#3047](https://github.com/mono/SkiaSharp/pull/3047),
  [mono/skia#139](https://github.com/mono/skia/pull/139),
  [mono/skia#138](https://github.com/mono/skia/pull/138))
- **HarfBuzz native updated to 8.3.1** — matched to the m118 Skia bump.
  ([#3052](https://github.com/mono/SkiaSharp/pull/3052),
  [mono/skia#140](https://github.com/mono/skia/pull/140))
- **freetype2 → 2.13.3+** — refreshes the shipped font engine.
  ([#3061](https://github.com/mono/SkiaSharp/pull/3061),
  [mono/skia#143](https://github.com/mono/skia/pull/143))
- **libexpat → v2.6.4** — CVE hygiene bump for the shipped XML parser.
  ([#3056](https://github.com/mono/SkiaSharp/pull/3056),
  [mono/skia#142](https://github.com/mono/skia/pull/142))
- **libjpeg-turbo → 2.1.5.1+** — routine refresh of the JPEG codec.
  ([#3060](https://github.com/mono/SkiaSharp/pull/3060),
  [mono/skia#141](https://github.com/mono/skia/pull/141))

## API Surface

- **Metal `GRBackendRenderTarget` overloads on every Apple platform** — the
  Metal-flavored `GRBackendRenderTarget` constructors and factory methods are
  now uniformly available on iOS, tvOS, Mac Catalyst, and macOS. ❤️
  [@spouliot](https://github.com/spouliot)
  ([#3070](https://github.com/mono/SkiaSharp/pull/3070))

## Platform

- **.NET 9 Blazor and WASM apps unblocked** — the Blazor-side WASM workaround
  is now baked into SkiaSharp itself; consumers no longer need to apply a
  runtime shim.
  ([#3064](https://github.com/mono/SkiaSharp/pull/3064),
  [#3082](https://github.com/mono/SkiaSharp/pull/3082),
  [#3088](https://github.com/mono/SkiaSharp/pull/3088),
  [#3092](https://github.com/mono/SkiaSharp/pull/3092))
- **Windows SDK version pinned with `<WindowsSdkPackageVersion>`** — the
  Windows build uses the SDK version property so downstream apps don't get
  surprise SDK bumps. ([#3085](https://github.com/mono/SkiaSharp/pull/3085))

The HarfBuzzSharp binding is republished as
[HarfBuzzSharp 8.3.1](harfbuzzsharp/8.3.1.md) to match this preview.

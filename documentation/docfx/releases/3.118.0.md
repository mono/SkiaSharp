# Version 3.118.0

> **Skia milestone 118 with dependency hardening** · Preview released December 5, 2024 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.118.0-preview.2.3) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.118.0-preview.2.3)

## Highlights

SkiaSharp 3.118.0 updates the Skia engine to milestone 118 and refreshes several native dependencies including FreeType, libexpat, libjpeg-turbo, and HarfBuzz. This release also brings Metal `GRBackendRenderTarget` APIs to all platforms, resolves .NET 9 Blazor compatibility issues, and adds Android 16KB page alignment support. One community contributor expanded Metal API coverage.

> [!NOTE]
> Version 3.118.0 was released only as previews and was not published as a stable release.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia updated to milestone 118** — Core graphics engine updated through milestones 117 and 118. ([#3047](https://github.com/mono/SkiaSharp/pull/3047), [#3048](https://github.com/mono/SkiaSharp/pull/3048))

### GPU & Rendering

- **Metal GRBackendRenderTarget on all platforms** — Metal API for `GRBackendRenderTarget` is now available on all platforms, not just Apple. ❤️ @spouliot ([#3070](https://github.com/mono/SkiaSharp/pull/3070))

### Dependencies

- **FreeType updated to 2.13.3+** — Font rendering engine updated with bug fixes and improvements. ([#3061](https://github.com/mono/SkiaSharp/pull/3061))
- **libexpat updated to v2.6.4** — XML parser dependency updated. ([#3056](https://github.com/mono/SkiaSharp/pull/3056))
- **libjpeg-turbo updated to 2.1.5.1+** — JPEG codec updated with latest commits. ([#3060](https://github.com/mono/SkiaSharp/pull/3060))
- **HarfBuzz updated to 8.3.1** — Text shaping engine updated with minor fixes. ([#3052](https://github.com/mono/SkiaSharp/pull/3052))

## Bug Fixes

- **Fixed .NET 9 ASP.NET Blazor issues** — Resolved compatibility issues with .NET 9 Blazor applications. ([#3064](https://github.com/mono/SkiaSharp/pull/3064))
- **Blazor WASM module loading fix** — Moved the Blazor workaround into the WASM layer and fixed the import for apps with a base path. ([#3082](https://github.com/mono/SkiaSharp/pull/3082), [#3088](https://github.com/mono/SkiaSharp/pull/3088), [#3092](https://github.com/mono/SkiaSharp/pull/3092))
- **Android NDK updated to r27c with 16KB alignment** — Updated the Android NDK and enabled 16KB page alignment for Android 15+ compatibility. ([#3099](https://github.com/mono/SkiaSharp/pull/3099))
- **Corrected NuGet PackagePath attribute** — Fixed the `PackagePath` attribute in NuGet packaging. ([#3101](https://github.com/mono/SkiaSharp/pull/3101))

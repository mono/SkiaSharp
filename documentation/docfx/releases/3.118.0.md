# Version 3.118.0

> **Preview only** · Superseded by [3.119.0](3.119.0.md) · Never released as stable — these changes rolled up into 3.119.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.118.0-preview)

## Highlights

> **Note:** 3.118.0 shipped only as previews and was never released as a stable package. All of the work below rolled up into [3.119.0](3.119.0.md).

This preview cycle advanced the Skia engine to Milestones 117 and 118, introduced a Metal API extension for `GRBackendRenderTarget`, delivered a comprehensive round of WebAssembly and .NET 9 Blazor bug fixes, and refreshed four bundled native libraries. Community contributor ❤️ [@spouliot](https://github.com/spouliot) added the Metal API improvements.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia Milestone 117** — Updated the underlying Skia engine to Milestone 117, picking up upstream bug fixes and improvements. ([#3047](https://github.com/mono/SkiaSharp/pull/3047))
- **Skia Milestone 118** — Advanced the Skia engine to Milestone 118, bringing further performance and rendering enhancements. ([#3048](https://github.com/mono/SkiaSharp/pull/3048))

### GPU & Rendering

- **Metal API for `GRBackendRenderTarget`** — Exposes the Metal backend API for `GRBackendRenderTarget` across all supported platforms, enabling broader GPU-accelerated rendering with Metal. ❤️ [@spouliot](https://github.com/spouliot) ([#3070](https://github.com/mono/SkiaSharp/pull/3070))

## Bug Fixes

### WebAssembly & Blazor

- **Fixed .NET 9 ASP.NET Blazor issues** — Resolved compatibility problems affecting .NET 9 Blazor WebAssembly apps. ([#3064](https://github.com/mono/SkiaSharp/pull/3064))
- **Moved Blazor workaround into WASM** — Relocated the Blazor compatibility shim so it applies correctly to all WASM targets. ([#3082](https://github.com/mono/SkiaSharp/pull/3082))
- **Moved the WASM fix to SkiaSharp** — Incorporated the WASM rendering fix directly into the SkiaSharp package. ([#3088](https://github.com/mono/SkiaSharp/pull/3088))
- **Fixed import for apps with a base path** — Corrected the WASM module import path when an application is hosted under a non-root base URL. ([#3092](https://github.com/mono/SkiaSharp/pull/3092))

### NuGet Packaging

- **Corrected `PackagePath` attribute** — Fixed an incorrect `PackagePath` attribute in the NuGet package manifests. ([#3101](https://github.com/mono/SkiaSharp/pull/3101))

## Native Dependencies

- **HarfBuzz 8.3.1** — Text shaping library updated to 8.3.1. ([#3052](https://github.com/mono/SkiaSharp/pull/3052))
- **libexpat v2.6.4** — XML parsing library updated to v2.6.4. ([#3056](https://github.com/mono/SkiaSharp/pull/3056))
- **libjpeg-turbo 2.1.5.1+** — JPEG library updated to the latest 2.1.5.1 commit. ([#3060](https://github.com/mono/SkiaSharp/pull/3060))
- **FreeType 2.13.3+** — Font rendering library updated to 2.13.3 and later. ([#3061](https://github.com/mono/SkiaSharp/pull/3061))

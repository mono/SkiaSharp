# Version 3.119.4

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This update bumps libpng to 1.6.58 and fixes a component governance metadata mismatch that was preventing accurate CVE detection for the release branch.

## Breaking Changes

*None in this release.*

## Security

- **Update libpng to 1.6.58** — Bumps the bundled libpng from 1.6.54 to 1.6.58, picking up upstream bug fixes and hardening. ([#3723](https://github.com/mono/SkiaSharp/pull/3723))
- **Fix cgmanifest.json Skia milestone mismatch** — Corrected the declared Skia version from chrome/m132 to chrome/m119 so Component Governance accurately tracks CVEs for this release branch. ([#3721](https://github.com/mono/SkiaSharp/pull/3721))

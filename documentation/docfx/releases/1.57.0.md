# Version 1.57.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.57.0)

> **API changes** · [SkiaSharp API diff](1.57.0/index.md)

## Highlights

Historical release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so the pull-request history for the branch is empty; the signature changes captured from the previous tag are the interesting part and are surfaced through the API diff linked above.

## Breaking Changes

- **`SKEncodedFormat` renamed to `SKEncodedImageFormat`** — The enum type used by `SKCodec.EncodedFormat` and related APIs was renamed and its members were renumbered (dropping the placeholder `Unknown = 0`). Rebuild any code that referenced the old enum name.
- **`SKImage.ToTextureImage(GRContext)` removed** — The single-argument texture upload overload was dropped; use the overload that also specifies whether ownership is transferred, or `SKImage.FromTexture(...)` for constructing texture-backed images.
- **`GRPixelConfig.Index8` removed** — The obsolete index-8 GPU pixel configuration was dropped alongside the wider index-8 removal from the codec surface.

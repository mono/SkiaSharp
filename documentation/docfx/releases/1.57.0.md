## Highlights

Servicing release for the 1.57 line with no PRs recorded in the diff. The 1.57 assembly bumped to align with an upstream Skia rename that consolidated `SKEncodedFormat` into `SKEncodedImageFormat`, and it dropped Index8 support from the GPU pixel-config enum.

## Breaking Changes

- **`GRPixelConfig.Index8`** — Removed to match upstream. Callers using indexed GPU textures should migrate to a supported color type.
- **`SKCodec.EncodedFormat`** — The property type changed from `SKEncodedFormat` to `SKEncodedImageFormat`; the values line up one-to-one.
- **`SKImage`** — A legacy factory overload was pruned in favor of the current shape.

See the [full API diff](1.57.0/index.md) for the complete list.

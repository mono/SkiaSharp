# Version 1.57.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.57.0)

## Highlights

Servicing release with no user-facing PR work. The binding refresh renames one encoded-format enum and prunes a handful of methods and enum members that Skia removed upstream.

## Breaking Changes

- **SKEncodedFormat → SKEncodedImageFormat** — the enum type was renamed. `SKCodec.EncodedFormat` now returns `SKEncodedImageFormat`.
- **SKImageEncodeFormat** — the `Unknown = 0` value was removed and the remaining members re-numbered (`Bmp = 0`, `Gif = 1`, …). Only code that switches on the raw integer values is affected.
- **SKImage** — `ToTextureImage(GRContext)` was removed. Use the modern `SKImage.FromEncodedData` + GPU-backed workflow.
- **SKMaskFilter** — the emboss and shadow helpers (`CreateEmboss`, `CreateShadow`) were removed; use the current shadow/blur filters on `SKImageFilter` instead. `SKShadowMaskFilterShadowFlags` was removed.
- **SKWStream** — `NewLine()` was removed; write `"\n"` explicitly if needed.
- **GRPixelConfig** — the `Index8` value was removed.

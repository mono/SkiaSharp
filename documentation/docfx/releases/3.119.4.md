# Version 3.119.4

> **Platform expansion and servicing release** · Released · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.4) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.4)

> **Supersedes [3.119.3](3.119.3.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](3.119.4/index.md) · [HarfBuzzSharp 8.3.1.5](harfbuzzsharp/8.3.1.5.md)

## Highlights

SkiaSharp 3.119.4 rolls the skipped 3.119.3 preview into a larger servicing release for the 3.119.x line. It broadens platform coverage with Linux Bionic assets, Tizen x64 and arm64 support, and a new Gtk4 views package, while also fixing runtime issues across Android, AoT, Linux ARM64, GTK4, tvOS, and Windows. HarfBuzzSharp 8.3.1.5 ships alongside this release; see the linked companion page for its binding details.

## Breaking Changes

*None in this release.*

## New Features

### Platform

- **Linux Bionic native assets** — Adds native asset support for Linux Bionic environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 builds** — Expands Tizen native build coverage to additional architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **`SkiaSharp.Views.Gtk4`** — Adds a Gtk4 views package based on the GirCore.Gtk-4.0 bindings. ❤️ [@Copilot](https://github.com/Copilot) ([#3527](https://github.com/mono/SkiaSharp/pull/3527))
- **Legacy .NET target support** — Adds a `net48` validation target and uses PolySharp to keep newer language features available on legacy TFMs. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3631](https://github.com/mono/SkiaSharp/pull/3631), [#3642](https://github.com/mono/SkiaSharp/pull/3642))

### Packaging & Distribution

- **Smaller package split** — Splits the oversized NuGet mega-packages to stay within feed size limits. ([#3584](https://github.com/mono/SkiaSharp/pull/3584))
- **Native dependency refreshes** — Updates key native components including libwebp, brotli, libexpat, and libpng during the 3.119.3/3.119.4 servicing cycle. ([#3478](https://github.com/mono/SkiaSharp/pull/3478), [#3469](https://github.com/mono/SkiaSharp/pull/3469), [#3458](https://github.com/mono/SkiaSharp/pull/3458), [#3452](https://github.com/mono/SkiaSharp/pull/3452), [#3723](https://github.com/mono/SkiaSharp/pull/3723))

## Security

- **Spectre mitigations for Windows and Linux** — Rolls the 3.119.3 hardening work into the stable line for both Windows DLLs and `libSkiaSharp.so`. ❤️ [@sshumakov](https://github.com/sshumakov) ([#3496](https://github.com/mono/SkiaSharp/pull/3496), [#3502](https://github.com/mono/SkiaSharp/pull/3502))
- **Stronger native link validation** — Enforces `--no-undefined` on Linux and Android native builds to catch unresolved native symbols earlier. ([#3629](https://github.com/mono/SkiaSharp/pull/3629))

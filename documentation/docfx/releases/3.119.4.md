# Version 3.119.4

> **Platform expansion & .NET 10 preview** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.4-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.4-preview.1.1)

## Highlights

This release significantly expands platform reach — adding GTK 4 views, Android Bionic (non-NDK) native support, and Tizen x64/arm64 builds — while upgrading the entire project to .NET 10. Three community contributors drove key improvements: ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) added Bionic library support and modernized the build for C# 13 and .NET Framework 4.8, ❤️ [@SimonvBez](https://github.com/SimonvBez) fixed a long-standing Android rendering bug, and ❤️ [@ramezgerges](https://github.com/ramezgerges) improved the developer tooling setup.

## Breaking Changes

*None in this release.*

## New Features

### Platform

- **SkiaSharp.Views.Gtk4** — New view package for GTK 4 using GirCore.Gtk-4.0 bindings, bringing first-class Linux desktop support to modern GTK applications. ([#3527](https://github.com/mono/SkiaSharp/pull/3527))
- **Android Bionic library support** — Adds native assets for Android's Bionic C library, enabling scenarios beyond the standard NDK toolchain. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Expands Tizen support with native libraries for both Intel and ARM architectures, plus updated Studio 6.1 tooling and `net6.0-tizen8.0` TFM. ([#3620](https://github.com/mono/SkiaSharp/pull/3620), [#3619](https://github.com/mono/SkiaSharp/pull/3619))
- **Update to .NET 10 SDK** — Moves the entire build to .NET 10 with workload version set pinning for reproducible builds. ([#3514](https://github.com/mono/SkiaSharp/pull/3514))

### API Surface

- **SKManagedStream refactored** — Replaced the parent/child chain with an `SKData` snapshot for `Duplicate()` and `Fork()`, eliminating a class of lifetime bugs in stream wrappers. ([#3589](https://github.com/mono/SkiaSharp/pull/3589))
- **PolySharp for C# 13 on legacy TFMs** — Enables modern C# language features (required members, `init`, etc.) on older target frameworks via source-generated polyfills. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))
- **Full Framework bumped to net48** — Updates the minimum .NET Framework target from net462 to net48, aligning with modern .NET support policies. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3631](https://github.com/mono/SkiaSharp/pull/3631))

### GPU & Rendering

- **Delay-load D3D12 DLLs** — Prevents crashes on Windows systems without DirectX 12 by delay-loading `d3d12.dll` and related DLLs. ([#3633](https://github.com/mono/SkiaSharp/pull/3633))

### Samples

- **Interactive Blazor WASM Gallery** — Redesigned the sample gallery as a full Blazor WebAssembly showcase with 21 interactive demos. ([#3578](https://github.com/mono/SkiaSharp/pull/3578))
- **All samples modernized** — Every sample project (Console, Browser WASM, GTK 3, GTK 4, Android, iOS, Mac Catalyst, macOS, tvOS, MAUI, Blazor, WPF, WinForms, WinUI, ASP.NET Core, Docker) received multi-page layouts, updated UI patterns, and READMEs. ([#3590](https://github.com/mono/SkiaSharp/pull/3590)–[#3606](https://github.com/mono/SkiaSharp/pull/3606))

## Bug Fixes

- **SKGLView not rendering after tab switch on Android** — Fixed the GL view failing to render when switching tabs in a MAUI `TabBar`. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Cairo assertion crash in GTK 4 SKDrawingArea** — Fixed a crash caused by incorrect Cairo surface handling in the new GTK 4 view. ([#3562](https://github.com/mono/SkiaSharp/pull/3562))
- **tvOS device/simulator build split** — Fixed tvOS native builds and NativeReference targets to correctly separate device and simulator frameworks, matching the iOS pattern. ([#3561](https://github.com/mono/SkiaSharp/pull/3561), [#3563](https://github.com/mono/SkiaSharp/pull/3563))
- **loongarch64 Linux build fix** — Corrected the architecture mapping for LoongArch 64-bit Linux builds. ([#3648](https://github.com/mono/SkiaSharp/pull/3648))
- **WindowsForms DesignMode detection** — Uses the `DesignMode` property directly in WindowsForms controls for more reliable designer support. ([#3614](https://github.com/mono/SkiaSharp/pull/3614))
- **Enforce `--no-undefined` linker flag** — Adds the flag for Linux and Android native builds to catch missing symbols at link time instead of runtime. ([#3629](https://github.com/mono/SkiaSharp/pull/3629))

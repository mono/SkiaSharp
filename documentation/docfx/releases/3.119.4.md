# Version 3.119.4

> [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.4)

> **Supersedes [3.119.3](3.119.3.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](3.119.4/index.md) · [HarfBuzzSharp 8.3.1.5](harfbuzzsharp/8.3.1.5.md)

Servicing release on the `3.119.x` line. `3.119.4` **supersedes** [`3.119.3`](3.119.3.md) — the preview-only servicing branch is rolled up here in one stable NuGet. Two headline items: **Linux Bionic** native assets ship for the first time, and the AI-agent tooling that automates issue triage, repro, native-dep updates, and Skia milestone review is wired into the repository. The release also carries native dependency refreshes (libpng, libwebp, brotli, libexpat) and a wave of Docker / WASM / Tizen / Android CI hardening. HarfBuzz `8.3.1.5` ships alongside — see [HarfBuzzSharp 8.3.1.5](harfbuzzsharp/8.3.1.5.md).

## Highlights

- Add **Linux Bionic** native assets ([#3217](https://github.com/mono/SkiaSharp/pull/3217)) — new native runtime asset for Bionic (Android-libc) Linux distros.
- Add **`SkiaSharp.Views.Gtk4`** using GirCore Gtk-4.0 bindings ([#3527](https://github.com/mono/SkiaSharp/pull/3527)) with a fix for the Cairo assertion crash in `SKDrawingArea` ([#3562](https://github.com/mono/SkiaSharp/pull/3562)).
- **Tizen** overhaul: Studio 6.1, TFM fixes, and a 3-tab sample ([#3619](https://github.com/mono/SkiaSharp/pull/3619)); native x64 + arm64 support ([#3620](https://github.com/mono/SkiaSharp/pull/3620)); split native CI into per-architecture jobs ([#3672](https://github.com/mono/SkiaSharp/pull/3672)); target `net6.0-tizen8.0` in the sample ([#3626](https://github.com/mono/SkiaSharp/pull/3626)).

## Highlights

SkiaSharp **3.119.4** is the platform-expansion release on the 3.119.x branch, superseding the preview-only [3.119.3](3.119.3.md). It rolls up Spectre-mitigated Windows/Linux native binaries, AoT-with-reflection-disabled fixes, Linux Bionic support, Tizen x64/arm64 build support, Tizen Studio 6.1 uptake, tvOS device/simulator split, GTK4 Cairo assertion fix, .NET 10 SDK adoption, PolySharp-based C# 13 support on legacy TFMs, updated `libpng`, `libwebp`, `libexpat`, `brotli`, `freetype`, and `zlib` native dependencies, `SKGLView` MAUI TabBar rendering fix, `SKManagedStream` snapshot rework, `SkiaSharp.Views.Gtk4` (GirCore-based), and delay-loaded D3D12 DLLs for machines without DX12.

## Breaking Changes

This release includes breaking changes in `SkiaSharp.Views.Blazor`. See the [SkiaSharp API diff](3.119.4/index.md) and [Blazor breaking-changes report](3.119.4/SkiaSharp.Views.Blazor/SkiaSharp.Views.Blazor.breaking.md) before upgrading.

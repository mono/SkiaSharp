# Version 3.119.4

> **Platform expansion & samples overhaul** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.4-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.4-preview.1.1)

## Highlights

This release brings first-class GTK 4 support via the new `SkiaSharp.Views.Gtk4` package, adds Linux Bionic and Tizen (x64/arm64) native builds, and delay-loads D3D12 DLLs to prevent crashes on Windows systems without DirectX 12. Every sample app has been redesigned with multi-page layouts, and a new interactive Blazor WASM Gallery showcases 21 live demos. Community contributors ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) and [@SimonvBez](https://github.com/SimonvBez) drove platform improvements and a key MAUI rendering fix.

## Breaking Changes

*None in this release.*

## New Features

### Platform

- **GTK 4 view support** — New `SkiaSharp.Views.Gtk4` package using GirCore.Gtk-4.0 bindings, including `SKDrawingArea` for hardware-accelerated SkiaSharp rendering in GTK 4 applications. ([#3527](https://github.com/mono/SkiaSharp/pull/3527))
- **Linux Bionic native assets** — Adds native library support for Linux Bionic (Android-style Linux), enabling SkiaSharp on Bionic-based environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Extends Tizen support with native builds for both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Delay-load D3D12 DLLs** — Prevents crashes on Windows systems without DirectX 12 by delay-loading D3D12 DLLs at runtime. ([#3633](https://github.com/mono/SkiaSharp/pull/3633))

### API Surface

- **PolySharp for C# 13 on legacy TFMs** — Adds PolySharp to enable C# 13 language features on older target frameworks. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))
- **Update .NET Framework target to net48** — Raises the minimum .NET Framework target to net48, aligning with modern tooling requirements. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3631](https://github.com/mono/SkiaSharp/pull/3631))

### Internal

- **SKManagedStream refactored** — Replaces the parent/child chain with a `SKData` snapshot for duplicate and fork operations, improving reliability and simplifying the managed stream lifecycle. ([#3589](https://github.com/mono/SkiaSharp/pull/3589))
- **Enforce `--no-undefined` linker flag** — Linux and Android native builds now fail at link time on missing symbols instead of silently deferring to runtime. ([#3629](https://github.com/mono/SkiaSharp/pull/3629))
- **Update to .NET 10 SDK** — Build infrastructure updated to .NET 10 with workload version set pinning. ([#3514](https://github.com/mono/SkiaSharp/pull/3514))

## Bug Fixes

- **Fixed SKGLView not rendering after tab switch in MAUI** — Resolved an issue where `SKGLView` would stop rendering when switching tabs in a MAUI `TabBar`. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Fixed Cairo assertion crash in GTK 4** — Resolved a crash in `SKDrawingArea` caused by a Cairo assertion failure during drawing. ([#3562](https://github.com/mono/SkiaSharp/pull/3562))
- **Fixed tvOS device/simulator native builds** — Split tvOS native builds into separate device and simulator targets to match the iOS build pattern, fixing compilation issues. ([#3561](https://github.com/mono/SkiaSharp/pull/3561), [#3563](https://github.com/mono/SkiaSharp/pull/3563))
- **Fixed loongarch64 Linux build mapping** — Corrected the architecture mapping for LoongArch 64-bit Linux builds. ([#3648](https://github.com/mono/SkiaSharp/pull/3648))
- **Fixed WindowsForms DesignMode check** — Uses the `DesignMode` property directly in Windows Forms controls for more reliable designer detection. ([#3614](https://github.com/mono/SkiaSharp/pull/3614))

## Samples

All sample apps have been redesigned with consistent multi-page layouts, drawing demos, and comprehensive READMEs:

- **Interactive Blazor WASM Gallery** — A new showcase with 21 live demos. ([#3578](https://github.com/mono/SkiaSharp/pull/3578))
- Updated samples for Console, GTK 3, GTK 4, Android, iOS, Mac Catalyst, macOS, tvOS, MAUI, Blazor WASM, WPF, Windows Forms, WinUI 3, and ASP.NET Core.
- WinUI sample now uses MSIX packaging. ([#3651](https://github.com/mono/SkiaSharp/pull/3651))
- Docker samples restructured with READMEs. ([#3606](https://github.com/mono/SkiaSharp/pull/3606))

Plus several CI, documentation, and tooling improvements.

# Version 1.56.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.56.0)

> **API changes** · [SkiaSharp API diff](1.56.0/index.md)

## Highlights

Historical release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so the pull-request history for the branch is empty; the signature changes captured from the previous tag are the interesting part and are surfaced through the API diff linked above.

## Breaking Changes

- **`SKImageFilter.CreateMerge` overload removed** — The overload taking `SKXferMode[] modes` alongside the filter array was dropped; the transfer mode is now controlled by the filter itself.
- **`SKPictureRecorder.BeginRecording` parameter rename** — The `rect` parameter was renamed to `cullRect` to match the underlying Skia semantics. Named-argument callers need to update.

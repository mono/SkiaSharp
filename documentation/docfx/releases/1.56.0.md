# Version 1.56.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.56.0)

> **API changes** · [SkiaSharp API diff](1.56.0/index.md)

## Highlights

Servicing release with no user-facing PR work. The binding refresh drops an obsolete image-filter overload and the never-used `System.Drawing.Color` helpers on the Apple views.

## Breaking Changes

- **SKImageFilter** — `CreateMerge(SKImageFilter[] filters, SKXferMode[] modes, SKImageFilter.CropRect cropRect)` was removed. Use the current `CreateMerge` overloads (blend-mode arrays or single-filter pairs).
- **SKPictureRecorder** — `BeginRecording(SKRect rect)` was renamed to `BeginRecording(SKRect cullRect)`; positional callers are unaffected.
- **SkiaSharp.Views.iOS / .Views.tvOS / .Views.Mac** — the `ToDrawingColor(SKColor)` and `ToSKColor(System.Drawing.Color)` extension methods were removed from `Extensions`. Convert via `SKColor.FromArgb`/`.ToArgb` and the platform's own color type.

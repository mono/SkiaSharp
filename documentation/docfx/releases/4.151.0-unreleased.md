# Version 4.151.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

Work landing on `main` after the `4.151.0-preview.1` cutoff. Highlights include an `SKColor` hex-parser optimization with `ReadOnlySpan<char>` overloads, a cross-backend golden-image (visual regression) test harness, a `memory-leak-fixer` skill with its own self-testing workflow, and continued release-notes / docs-pipeline work. HarfBuzz `14.2.1` continues to ship alongside — see [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md).

## Highlights

- **API:** optimize `SKColor` hex parsing and add `ReadOnlySpan<char>` overloads ([#4345](https://github.com/mono/SkiaSharp/pull/4345)); remove unnecessary `unsafe` modifier from `GRContextOptions` ([#3991](https://github.com/mono/SkiaSharp/pull/3991)).
- **Testing:** add cross-backend golden-image (visual regression) test harness ([#4340](https://github.com/mono/SkiaSharp/pull/4340)).
- **Engine:** `[skia-sync]` Merge upstream `chrome/m151` bug fixes ([#4312](https://github.com/mono/SkiaSharp/pull/4312), skia: [mono/skia#278](https://github.com/mono/skia/pull/278)).
- **AI tooling:** add `memory-leak-fixer` skill + self-testing agentic workflow ([#4330](https://github.com/mono/SkiaSharp/pull/4330)) with a `focus_family` override input ([#4356](https://github.com/mono/SkiaSharp/pull/4356)); auto-apply milestone labels in the Skia sync workflow ([#4327](https://github.com/mono/SkiaSharp/pull/4327)) and on the `mono/skia` sync PR too ([#4329](https://github.com/mono/SkiaSharp/pull/4329)).

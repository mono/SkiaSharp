# Version 4.151.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

Version 4.151.0 upgrades the Skia engine to Chrome milestone m151 and ships HarfBuzzSharp 14.2.1 alongside. A community fix eliminates a `SK_BUILD_FOR_WASM` preprocessor redefinition that could break WebAssembly builds, and SkiaFiddle gains embedded image and font selectors for a richer browser-based development experience. ❤️ [@ramezgerges](https://github.com/ramezgerges)

## New Features

### Engine

- **Skia milestone m151** — Updates the underlying Skia graphics engine to Chrome milestone m151, incorporating upstream performance improvements and bug fixes from Google. ([#4294](https://github.com/mono/SkiaSharp/pull/4294))

### WebAssembly

- **Fix `SK_BUILD_FOR_WASM` redefinition** — Removes a duplicate preprocessor define from the Skia submodule that caused build warnings or errors in WebAssembly targets. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3788](https://github.com/mono/SkiaSharp/pull/3788))

### Tools

- **SkiaFiddle image and font selectors** — Adds embedded image and font asset selectors to the SkiaFiddle browser playground, making it easier to try different assets when experimenting with SkiaSharp code online. ([#4284](https://github.com/mono/SkiaSharp/pull/4284))

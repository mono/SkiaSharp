# Version 4.151.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

This upcoming release advances the Skia graphics engine to milestone m151, incorporating the latest upstream fixes and enhancements from the Chrome graphics team. HarfBuzz 14.2.1 ships alongside — see the [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) page for text-shaping binding details. SkiaFiddle also gains embedded image and font selectors, making live API exploration richer out of the box.

*None in this release.*

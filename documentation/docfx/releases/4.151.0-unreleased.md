- Iterated on the release-notes pipeline: manual notes sidecars, AI-summarized breaking changes, and per-diff itemization so future releases land with a full changelog on day one ([#4334](https://github.com/mono/SkiaSharp/pull/4334), [#4337](https://github.com/mono/SkiaSharp/pull/4337), [#4322](https://github.com/mono/SkiaSharp/pull/4322)).

- Removed the unnecessary `unsafe` modifier from `GRContextOptions` so consumers no longer have to opt into unsafe blocks to configure the context ([#3991](https://github.com/mono/SkiaSharp/pull/3991) ❤️).
- Removed the obsolete `skiasharp-docker` registry authentication that was blocking new contributors from running the build container locally ([#4310](https://github.com/mono/SkiaSharp/pull/4310) ❤️).
- Doc-site sweeps: skip PR staging previews for non-site changes, garbage-collect previews inactive over three months, and added a "Moving faster with AI" dashboard ([#4321](https://github.com/mono/SkiaSharp/pull/4321), [#4313](https://github.com/mono/SkiaSharp/pull/4313), [#4318](https://github.com/mono/SkiaSharp/pull/4318)).

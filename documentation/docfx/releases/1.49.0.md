# Version 1.49.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/1.49.0-preview)

## Highlights

The first public preview of SkiaSharp — a cross-platform 2D graphics API for .NET that wraps Google's Skia library (milestone 49). This release brought Skia's powerful rendering engine to Xamarin and .NET developers for the first time.

## New Features

### Engine

- **Skia milestone 49** — Initial binding of the native Skia rendering engine for .NET.

### API Surface

- **Core drawing API** — Canvas, paint, path, image, and bitmap primitives for cross-platform 2D graphics.

## Highlights

This release brings **variable font support** and **color font palettes** to SkiaSharp, enabling rich typography with OpenType variable axes and COLR/CPAL color fonts. The API surface expands with `SKSamplingOptions` overloads for surface and canvas drawing, and the gallery gains new Uno Platform WebAssembly samples. A big thank you to community contributor **@ramezgerges** for driving variable font support and Uno Platform integration.

## Breaking Changes

*None in this release.*

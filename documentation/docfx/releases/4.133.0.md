# Version 4.133.0

> **Upcoming release** · In development · Not yet available on NuGet


## Highlights

SkiaSharp 4.133.0 is a major release that upgrades the Skia engine to **milestone 133**, introduces **variable font** and **color font palette** support, and expands platform coverage to Tizen and Android Bionic builds. This release also bumps the major version from 3.x to 4.x and includes security updates to libpng and libexpat. Four community contributors helped ship improvements across fonts, Android rendering, Linux tooling, and modern C# support.

## Breaking Changes

### Major Version Bump

- **SkiaSharp upgraded from 3.x to 4.x** — This major version bump accompanies the Skia m132/m133 engine updates and API surface changes. Review your project for any removed or changed APIs before upgrading. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

## New Features

### Engine

- **Skia updated to milestone 133** — Brings the latest rendering improvements, bug fixes, and performance optimizations from Google's Skia. ([#3660](https://github.com/mono/SkiaSharp/pull/3660))
- **Skia updated to milestone 132** — Intermediate engine update as part of the 4.x progression. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560))

### Text & Fonts

- **Variable font support** — Full support for variable fonts including axis enumeration and value setting. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Improved color font handling with palette support and better variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Default typeface resolution moved to managed layer** — More reliable font fallback behavior across platforms. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### API Surface

- **`SKSamplingOptions` support for surface drawing** — `SKSurface.Draw` and `SKCanvas.DrawSurface` now accept `SKSamplingOptions` for controlling image sampling quality. ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Android Bionic library build support** — SkiaSharp can now be built against Android's Bionic C library. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native build support** — Expanded Tizen platform coverage with 64-bit builds. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))

### Build & Tooling

- **PolySharp for C# 13 support on legacy TFMs** — Enables modern C# language features on older target frameworks. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))
- **SkiaSharpGenerator now works on Linux** — Updated CppAst dependency to 0.21.4 for cross-platform code generation. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3714](https://github.com/mono/SkiaSharp/pull/3714))

## Security

- **libpng updated to 1.6.58** — Security and bug fix update. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))
- **libexpat updated to 2.7.5** — Security and bug fix update. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))

## Bug Fixes

- **SKGLView on Android now renders with TabBar** — Fixed an issue where the GL view wouldn't render when used alongside a TabBar. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **x86 .NET Framework threading test OOM failures resolved** — Fixed out-of-memory failures in threading tests on 32-bit .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))
- **Debian 13 Docker build fixed** — Corrected build issues for non-loong64/riscv64 architectures. ([#3747](https://github.com/mono/SkiaSharp/pull/3747))

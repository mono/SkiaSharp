# Version 4.133.0

> **Upcoming release** · In development · Not yet available on NuGet

<!-- UNRELEASED_BEGIN -->

## Highlights

This release brings **variable font support** and **color font palettes** to SkiaSharp, enabling rich typography with OpenType variable axes and COLR/CPAL color fonts. The API surface expands with `SKSamplingOptions` overloads for surface and canvas drawing. A big thank you to community contributor [**@ramezgerges**](https://github.com/ramezgerges) for driving variable font support and Uno Platform integration.

## Breaking Changes

*None in this release.*

## New Features

### Text & Fonts

- **Variable font support** — Full OpenType variable font axis support, enabling dynamic weight, width, slant, and custom axis control via `SKTypeface` and `SKFontStyle`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Added support for COLR/CPAL color font palettes with improved variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Default typeface resolution moved to managed layer** — Fixes issues where the default typeface was not resolved correctly on certain platforms. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))
- **Legacy font manager typeface creation** — Bumped Skia externals to include `sk_fontmgr_legacy_create_typeface` for improved font matching. ([#3744](https://github.com/mono/SkiaSharp/pull/3744))

### API Surface

- **`SKSamplingOptions` for surface drawing** — `SKSurface.Draw` and `SKCanvas.DrawSurface` now accept `SKSamplingOptions` for fine-grained control over image sampling quality. ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Samples & Gallery

- **Uno Platform WebAssembly gallery** — New WASM gallery powered by Uno Platform, showcasing SkiaSharp rendering in the browser. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3758](https://github.com/mono/SkiaSharp/pull/3758))
- **Updated Uno Platform sample** — Refreshed and modernized Uno Platform sample project. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3666](https://github.com/mono/SkiaSharp/pull/3666))
- **FillPath and TextToPath samples** — New gallery samples demonstrating path filling and text-to-path conversion. ([#3761](https://github.com/mono/SkiaSharp/pull/3761))
- **Gallery download button and WASM text fix** — Added a download button for gallery output and fixed WASM text rendering. ([#3760](https://github.com/mono/SkiaSharp/pull/3760))

## Bug Fixes

- **x86 .NET Framework threading OOM fix** — Resolved out-of-memory failures in threading tests on 32-bit .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))
- **Debian 13 Docker build fix** — Fixed Docker build for non-loong64/riscv64 architectures on Debian 13. ([#3747](https://github.com/mono/SkiaSharp/pull/3747))

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This cycle brings SkiaFiddle UX improvements and a Skottie/Uno gallery fix from community contributor [@ramezgerges](https://github.com/ramezgerges), alongside a major investment in developer tooling — automated release notes generation, Skia analyst, and sample scout capabilities now run on every push and on a weekly schedule.

## Breaking Changes

*None in this release.*

## New Features

### Developer Tooling

- **Automated release notes** — Website release notes are now generated automatically whenever code lands on `main`, `release/*` branches, or a tag is pushed, using a fully git-based branch-diffing script. ([#3787](https://github.com/mono/SkiaSharp/pull/3787), [#3815](https://github.com/mono/SkiaSharp/pull/3815))
- **Skia analyst & sample scout skills** — New AI-powered skills that analyze upstream Skia changes and scout GM samples worth porting to the gallery, running on daily/weekly schedules. ([#3755](https://github.com/mono/SkiaSharp/pull/3755))
- **Improved docs landing page** — Refreshed layout and hero section for the documentation site. ([#3794](https://github.com/mono/SkiaSharp/pull/3794))

### Samples & Gallery

- **SkiaFiddle UX improvements** — Better user experience for the online SkiaFiddle sample environment. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))
- **Fixed Skottie Uno gallery build** — Resolves a `NU1605` package downgrade warning for the Skottie package in the Uno gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))

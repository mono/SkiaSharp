# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings new HarfBuzz API coverage with a proper `HBColor` struct, fills gaps in
the public API surface with missing wrappers and naming fixes, and dramatically shrinks the
SkiaFiddle WebAssembly bundle by ~60%. The Blazor gallery gains a redesigned API Atlas with
tag-cloud filtering, and the automated API docs pipeline goes live. Thanks to
[@ebariche](https://github.com/ebariche) and [@ramezgerges](https://github.com/ramezgerges)
for their community contributions.

## Breaking Changes

*None in this release.*

## New Features

### Text & Fonts

- **`HBColor` struct** — Adds a dedicated `HBColor` value type that correctly models the
  HarfBuzz `hb_color_t` representation, replacing raw integer usage throughout the HarfBuzz
  bindings. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### API Surface

- **Missing API wrappers and naming fixes** — Adds several previously missing public wrappers
  and corrects naming convention violations across the binding layer, improving API completeness
  and consistency. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### WebAssembly

- **SkiaFiddle bundle size reduced ~60%** — Significant build optimizations shrink the
  SkiaFiddle WASM application payload, dramatically improving load times for web users. ❤️
  [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

## Developer Experience

- **Automated API docs pipeline** — A new CI pipeline automatically generates and publishes API
  documentation, keeping the reference docs in sync with every release.
  ([#3918](https://github.com/mono/SkiaSharp/pull/3918))
- **Blazor gallery API Atlas** — The interactive Blazor gallery is redesigned with an API Atlas
  view and tag-cloud filtering, making it easier to discover samples by feature area.
  ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — A new Hero Image sample demonstrates compositing techniques
  in the live gallery. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **`TreatWarningsAsErrors` enabled** — All compiler warnings in `binding/` are now treated as
  errors and have been resolved, improving long-term code health.
  ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

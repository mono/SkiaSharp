# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This in-development release merges upstream Skia chrome/m147 bug fixes into the engine and
updates HarfBuzz to version 14.2.0. Build infrastructure is kept current with .NET SDK 10.0.108
and a cleanup to disable unused Skia dependency manifests.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia chrome/m147 bug fixes** — Merges the latest upstream bug fixes from the Chrome m147
  branch into the Skia engine. ([#4044](https://github.com/mono/SkiaSharp/pull/4044))

### Text & Fonts

- **HarfBuzz 14.2.0** — Updates the bundled HarfBuzz text-shaping library to version 14.2.0,
  bringing upstream bug fixes and improvements. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))

### Build & Infrastructure

- **Disable unused Skia dependency manifests** — Updates the Skia submodule to suppress
  generation of dependency manifests that are not used by SkiaSharp, reducing build noise.
  ([#4054](https://github.com/mono/SkiaSharp/pull/4054))
- **.NET SDK 10.0.108** — Updates the .NET SDK to 10.0.108 for all builds.
  ([#4055](https://github.com/mono/SkiaSharp/pull/4055))
- **ES metadata configuration** — Adds `es-metadata.yml` with initial configuration for
  ecosystem tooling. ([#4075](https://github.com/mono/SkiaSharp/pull/4075))

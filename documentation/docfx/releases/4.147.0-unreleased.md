# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings significant infrastructure investment — automated Skia upstream sync, content-based build caching, and a fully automated API docs pipeline — alongside real user-facing improvements: new HarfBuzz color API, missing API wrappers, a redesigned Blazor Gallery with API Atlas filtering, and a ~60% reduction in SkiaFiddle WASM application size. Community contributors ❤️ [@ebariche](https://github.com/ebariche), [@sasakrsmanovic](https://github.com/sasakrsmanovic), and [@ramezgerges](https://github.com/ramezgerges) drove several of these changes.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper `HBColor` struct representing `hb_color_t`, improving type safety and completeness of the HarfBuzz bindings. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers** — Adds several previously missing API wrappers and corrects naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Samples & Gallery

- **Blazor Gallery API Atlas** — Redesigns the Blazor gallery with a new API Atlas view featuring tag cloud filtering, making it easier to discover and explore available APIs. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new Hero Image sample demonstrating visually compelling image rendering techniques. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduced ~60%** — Significantly reduces SkiaFiddle's WebAssembly application footprint, improving load times for the interactive fiddle experience. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

### Platform

- **WASM: Drop pre-.NET 8 Emscripten builds** — Removes legacy Emscripten build targets for pre-.NET 8, streamlining the WASM build matrix. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))
- **Tizen SDK 10.0** — Updates the Tizen SDK installer from 6.1 to 10.0, keeping Tizen support current. ([#3743](https://github.com/mono/SkiaSharp/pull/3743))

## Security

- **libjpeg-turbo updated to 3.1.4.1** — Picks up the latest libjpeg-turbo security and bug-fix release. ([#4012](https://github.com/mono/SkiaSharp/pull/4012))
- **freetype updated to 2.14.3** — Picks up the latest FreeType security and stability fixes. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib updated to 1.3.2.1-motley** — Updates the bundled zlib to the latest patched version. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))

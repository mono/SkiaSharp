# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings new API coverage including a proper `HBColor` struct for HarfBuzz color handling, expanded missing API wrappers, and a redesigned Blazor gallery with API Atlas and tag-cloud filtering. The WASM SkiaFiddle app is now ~60% smaller thanks to community contributor ❤️ [@ebariche](https://github.com/ebariche), and build infrastructure sees major investment with content-based ADO caching and an automated Skia upstream sync workflow.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a dedicated `HBColor` struct for proper `hb_color_t` representation in the HarfBuzz bindings, replacing raw integer usage. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers** — Fills in a range of previously unbound APIs and fixes naming-convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))
- **Warnings-as-errors in `binding/`** — All compiler warnings in the binding project are now treated as errors, improving long-term API quality. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

### Samples & Gallery

- **Blazor API Atlas** — The Blazor gallery has been redesigned with an API Atlas view and interactive tag-cloud filtering, making it easier to discover samples by feature. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new Hero Image composition sample to the gallery. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **WASM app size reduced ~60%** — SkiaFiddle's WebAssembly bundle is dramatically smaller, improving load time for browser-based demos. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))
- **Uno Platform Gallery sample fix** — Corrects SkiaSharp references in the Uno Platform Gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3867](https://github.com/mono/SkiaSharp/pull/3867))

### Documentation

- **Automated API docs pipeline** — A new pipeline auto-generates and publishes API documentation on every release. ([#3918](https://github.com/mono/SkiaSharp/pull/3918))
- **Uno Platform README update** — Updates the Uno Platform link and description in the README. ❤️ [@sasakrsmanovic](https://github.com/sasakrsmanovic) ([#3966](https://github.com/mono/SkiaSharp/pull/3966))

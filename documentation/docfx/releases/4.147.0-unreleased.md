# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This cycle brings a critical finalizer-crash fix for `SKPath`, a refreshed Gallery with five new interactive samples and a redesigned Blazor home page, and UX improvements to the SkiaFiddle experience. Thanks to community contributor [@ramezgerges](https://github.com/ramezgerges) for driving three of the key user-facing changes in this release.

## Breaking Changes

*None in this release.*

## New Features

### Gallery & Samples

- **Five new interactive Gallery samples** — Adds hands-on demos to the Blazor-based SkiaSharp Gallery alongside a redesigned home page that better showcases the library's capabilities. ([#3757](https://github.com/mono/SkiaSharp/pull/3757))
- **SkiaFiddle UX improvements** — Multiple usability enhancements to the SkiaFiddle interactive playground. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))

## Bug Fixes

- **Fixed finalizer crash in `SKPath`** — When `SKPathBuilder` was garbage-collected before `SKPath`, a crash could occur in the finalizer. This is now resolved. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3796](https://github.com/mono/SkiaSharp/pull/3796))
- **Fixed Skottie package downgrade in Uno Gallery** — Resolves a `NU1605` package downgrade warning in the Uno-based gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))

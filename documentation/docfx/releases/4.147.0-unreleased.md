# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This development snapshot upgrades the Skia engine to milestone m148, bringing the latest upstream rendering improvements and bug fixes. [@ramezgerges](https://github.com/ramezgerges) contributed two meaningful changes: a rework of SkiaSharp's singleton instance lifecycle for better correctness, and the promotion of long-deprecated `SKPaint` text/font members to compile errors, nudging the remaining codebases to complete their migration to the modern `SKFont`-based API. Several bug fixes, dependency updates, and a WinUI/.NET 9 packaging fix also land in this release.

## Breaking Changes

- **`SKPaint` text/font obsolete members promoted to errors** — Properties and methods on `SKPaint` that were previously marked `[Obsolete]` with warnings are now compile errors. Migrate to the equivalent `SKFont` and `SKTextBlob` APIs. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4068](https://github.com/mono/SkiaSharp/pull/4068))

## New Features

### Engine

- **Skia milestone m148** — Updates the underlying Skia engine to m148, bringing the latest upstream performance improvements and rendering features. ([#4125](https://github.com/mono/SkiaSharp/pull/4125))
- **Upstream Skia m147 bug fixes** — Merges cherry-picked stability fixes from Chrome's m147 branch. ([#4081](https://github.com/mono/SkiaSharp/pull/4081), [#4044](https://github.com/mono/SkiaSharp/pull/4044))

### API Surface

- **Obsolete remaining legacy paint state-reading APIs** — Marks the remaining legacy paint state-reading APIs as obsolete, continuing the long-running migration to the modern, cleaner API surface. ([#4114](https://github.com/mono/SkiaSharp/pull/4114))
- **Singleton instance lifecycle rework** — Rewrites how SkiaSharp manages singleton instances, improving correctness and ensuring proper cleanup across the object graph. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))

## Security

- **expat updated to 2.8.1** — Updates the expat XML parsing library to pick up the latest security fixes. ([#4079](https://github.com/mono/SkiaSharp/pull/4079))
- **harfbuzz updated to 14.2.0** — Updates the HarfBuzz text shaping library to the latest stable release. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))

## Bug Fixes

- **`SKPixmap.GetPixelSpan<T>(x,y)` wrong offset** — Fixes an offset calculation that incorrectly used `Height` instead of `Width`, producing wrong pixel positions for non-square bitmaps. ([#4128](https://github.com/mono/SkiaSharp/pull/4128))
- **WinUI Projection DLL not found on .NET 9** — Resolves a packaging issue where the WinUI projection DLL could not be located by .NET 9 consumers. ([#4084](https://github.com/mono/SkiaSharp/pull/4084))

Plus several CI and infrastructure improvements, including migrating Linux Docker builds to .NET official cross-compilation images and test infrastructure upgrades.

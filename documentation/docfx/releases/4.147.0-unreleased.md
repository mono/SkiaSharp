# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release upgrades the Skia engine to milestone m148 with two additional upstream chrome/m147 bug-fix merges. [@ramezgerges](https://github.com/ramezgerges) contributed two significant API improvements: a rework of singleton instance lifecycle management and the promotion of long-deprecated `SKPaint` text/font members from warnings to compile errors. Bug fixes land for `SKPixmap` pixel offset calculation and WinUI on .NET 9, alongside security updates for expat and HarfBuzz.

## ⚠️ Breaking Changes

- **`SKPaint` text and font obsolete members promoted to errors** — Previously deprecated text/font properties and methods on `SKPaint` now produce compile errors instead of warnings. Migrate to the `SKFont`-based APIs before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4068](https://github.com/mono/SkiaSharp/pull/4068))

## New Features

### Engine

- **Skia milestone m148** — Updates the embedded Skia engine to milestone m148, delivering the latest rendering improvements and GPU fixes. ([#4125](https://github.com/mono/SkiaSharp/pull/4125))
- **Upstream chrome/m147 bug fixes** — Two merges of upstream chrome/m147 engine bug fixes for improved stability and correctness. ([#4081](https://github.com/mono/SkiaSharp/pull/4081), [#4044](https://github.com/mono/SkiaSharp/pull/4044))

### API Surface

- **Singleton instance lifecycle rework** — Rewrites how managed singleton wrapper objects are created and tracked, improving correctness and reducing potential memory leaks in long-running applications. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4080](https://github.com/mono/SkiaSharp/pull/4080))
- **Native-compat gate moved to `SkiaApi` static constructor** — The runtime native-compatibility check now runs in the `SkiaApi` static constructor instead of via `[ModuleInitializer]`, fixing initialization ordering issues in certain host frameworks. ([#4133](https://github.com/mono/SkiaSharp/pull/4133))

## Bug Fixes

- **`SKPixmap.GetPixelSpan<T>(x, y)` wrong offset** — The per-pixel span was computing the byte offset using `Height` instead of `Width`, returning incorrect data for non-square images. ([#4128](https://github.com/mono/SkiaSharp/pull/4128))
- **WinUI Projection DLL not found on .NET 9** — Fixes a packaging issue where the WinUI projection DLL was not resolved correctly for .NET 9 consumers. ([#4084](https://github.com/mono/SkiaSharp/pull/4084))

## Security

- **expat updated to 2.8.1** — Bumps the bundled expat XML parser to 2.8.1 for the latest security fixes. ([#4079](https://github.com/mono/SkiaSharp/pull/4079))
- **HarfBuzz updated to 14.2.0** — Bumps the bundled HarfBuzz text-shaping library to 14.2.0. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings SkiaFiddle UX improvements and a Skottie package fix for the Uno gallery, both contributed by the community. It also ships a fully automated release notes pipeline, new AI-powered skia-analyst and sample-scout developer tools, and a refreshed documentation landing page. Thanks to [@ramezgerges](https://github.com/ramezgerges) for driving the user-facing improvements!

## Breaking Changes

*None in this release.*

## New Features

### Tooling & Developer Experience

- **Automated release notes** — Website release notes are now generated automatically on every push via a branch-diffing script, eliminating manual work after each release. ([#3787](https://github.com/mono/SkiaSharp/pull/3787))
- **Skia analyst skill** — A new AI-powered `skia-analyst` skill and daily workflow analyses upstream Skia changes and surfaces what's new or missing in SkiaSharp bindings. ([#3755](https://github.com/mono/SkiaSharp/pull/3755))
- **Sample scout skill** — A new `sample-scout` skill with a weekly workflow scans Skia GM samples and identifies gallery demo opportunities, reading directly from the local submodule. ([#3755](https://github.com/mono/SkiaSharp/pull/3755), [#3802](https://github.com/mono/SkiaSharp/pull/3802))
- **Refreshed docs landing page** — Improved layout and hero section on the documentation homepage. ([#3794](https://github.com/mono/SkiaSharp/pull/3794))

### Samples

- **SkiaFiddle UX improvements** — Polished the SkiaFiddle interactive sample experience with UI/UX enhancements. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))
- **Fixed Skottie package downgrade in Uno gallery** — Resolved NU1605 package downgrade warning for the Skottie package in the Uno gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release adds animated WebP encoding via the new `SKWebpEncoder`, zero-copy stream-to-data conversion with `SKStream.GetData()`, and a significant Gallery redesign with five new interactive samples. [@ramezgerges](https://github.com/ramezgerges) contributed a critical finalizer crash fix for `SKPath`/`SKPathBuilder` and a batch of SkiaFiddle UX improvements.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`SKStream.GetData()` — zero-copy stream access** — New method exposes the underlying stream data as `SKData` without copying, enabling efficient pipelines that avoid double-buffering. ([#3772](https://github.com/mono/SkiaSharp/pull/3772))
- **Animated WebP encoding (`SKWebpEncoder`)** — Full support for encoding animated WebP files through the new `SKWebpEncoder` API, backed by updated Skia native bindings. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))

### Platform

- **Apple platform TFM fix** — Libraries now correctly target versioned TFMs (e.g. `net9.0-ios26.0`) while app targets remain unversioned, resolving deployment issues on the latest Apple SDKs. ([#3798](https://github.com/mono/SkiaSharp/pull/3798))

## Bug Fixes

- **`SKPath` finalizer crash** — Fixed a crash triggered when the GC collected `SKPathBuilder` before `SKPath`, causing the finalizer to access freed memory. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3796](https://github.com/mono/SkiaSharp/pull/3796))

## Samples & Gallery

- **Five new interactive Gallery samples + Blazor home page redesign** — New samples cover a range of drawing and animation scenarios; the Blazor Gallery home page has been redesigned for discoverability. ([#3757](https://github.com/mono/SkiaSharp/pull/3757))
- **SkiaFiddle UX improvements** — Better layout, improved editor ergonomics, and a smoother interactive experience in the web fiddle. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))
- **Uno Gallery: fix Skottie package downgrade** — Resolved a `NU1605` package downgrade warning in the Uno sample project. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))
- **Uno Basic sample bump to Uno.Sdk 6.6.0** — Updated the Basic Uno sample to `Uno.Sdk/6.6.0-dev.208`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3821](https://github.com/mono/SkiaSharp/pull/3821))

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release brings expanded HarfBuzz API coverage, a new `HBColor` struct for correct color representation, and fixes to API naming conventions. The Blazor gallery gains a redesigned API Atlas with tag-cloud filtering, and SkiaFiddle's WASM footprint drops by ~60% thanks to ❤️ [@ebariche](https://github.com/ebariche). Native dependencies freetype and zlib have been updated, and build infrastructure received significant improvements including content-based caching and an automated Skia upstream sync workflow.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper managed struct for `hb_color_t`, replacing raw integer values with a strongly-typed, readable color representation. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing HarfBuzz API wrappers** — Adds wrappers for previously-exposed HarfBuzz APIs and fixes naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Gallery & Samples

- **Blazor API Atlas** — Redesigns the Blazor gallery with a searchable API Atlas featuring tag-cloud filtering, making it easier to discover samples by API. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image sample** — Adds a new Hero Image gallery sample demonstrating compositing and image layout techniques. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction** — Reduces the SkiaFiddle WebAssembly application size by approximately 60%. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

### Native Dependencies

- **freetype updated to 2.14.3** — Picks up upstream font-rendering fixes and improvements. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib updated to 1.3.2.1-motley** — Refreshes the bundled zlib to the latest patched version. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))

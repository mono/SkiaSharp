It also includes a community-contributed fix for Uno Platform Gallery sample references. ❤️ [@ramezgerges](https://github.com/ramezgerges)

### Build & CI

- **Automated Skia upstream sync** — Adds a workflow to automatically sync upstream Skia changes into SkiaSharp, reducing manual effort for milestone updates. ([#3825](https://github.com/mono/SkiaSharp/pull/3825))

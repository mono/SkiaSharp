# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings a significantly improved interactive Gallery with five new samples and a redesigned Blazor home page, alongside UX improvements to the SkiaFiddle sample from the community. It also introduces automated tooling for release notes and AI-powered Skia analysis, and fixes Apple platform TFM targeting. Community contributors [@ramezgerges](https://github.com/ramezgerges) drove multiple user-facing improvements across the Uno and Skia Fiddle samples.

## Breaking Changes

*None in this release.*

## New Features

### Gallery & Samples

- **5 new interactive Gallery samples** — Adds five new interactive samples to the Blazor Gallery app and redesigns the home page for a better first impression. ([#3757](https://github.com/mono/SkiaSharp/pull/3757))
- **SkiaFiddle UX improvements** — Polishes the SkiaFiddle sample with improved layout and interaction. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))
- **Uno Gallery Skottie fix** — Fixes a `NU1605` package downgrade warning for Skottie in the Uno gallery project. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))
- **Uno Basic sample updated** — Bumps the Basic Uno sample to `Uno.Sdk/6.6.0-dev.208`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3821](https://github.com/mono/SkiaSharp/pull/3821))

### Documentation

- **Improved docs landing page** — Refreshes the layout and hero section of the documentation site. ([#3794](https://github.com/mono/SkiaSharp/pull/3794))

## Bug Fixes

- **Apple platform TFM fix** — Corrects Apple platform TFMs to use versioned targets (e.g. `26.0`) for libraries and unversioned targets for apps, matching Apple toolchain requirements. ([#3798](https://github.com/mono/SkiaSharp/pull/3798))
- **HarfBuzzSharp version bump** — Updates HarfBuzzSharp to `8.3.1.6` following its release. ([#3780](https://github.com/mono/SkiaSharp/pull/3780))

Plus several CI, tooling, and workflow improvements.

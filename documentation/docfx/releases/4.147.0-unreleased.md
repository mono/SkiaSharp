# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings the Skia m147 engine bug fixes from upstream Chrome, updates HarfBuzz to 14.2.0, and migrates Linux Docker builds to .NET's official cross-compilation images for improved build reliability and freshness.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia m147 upstream bug fixes** — Merges upstream Chrome/m147 bug fixes from Google into the Skia engine. ([#4044](https://github.com/mono/SkiaSharp/pull/4044))
- **Disable unused Skia dependency manifests** — Reduces overhead by disabling dependency manifest generation for unused Skia components. ([#4054](https://github.com/mono/SkiaSharp/pull/4054))

### Platform

- **Linux Docker migration to .NET official images** — Migrates all Linux Docker cross-compilation builds to the official .NET cross-compilation images, ensuring up-to-date toolchains and consistent environments. ([#4062](https://github.com/mono/SkiaSharp/pull/4062))
- **Docker base image refresh** — Adds `--pull` to all Docker builds so base images are always pulled fresh. ([#4087](https://github.com/mono/SkiaSharp/pull/4087))

### Dependencies

- **HarfBuzz 14.2.0** — Updates the bundled HarfBuzz text-shaping library to 14.2.0. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))
- **.NET SDK 10.0.108** — Updates the build to use .NET SDK 10.0.108. ([#4055](https://github.com/mono/SkiaSharp/pull/4055))

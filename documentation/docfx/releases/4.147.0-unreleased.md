# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release updates the HarfBuzz text-shaping engine to 14.2.0, bumps the .NET SDK to 10.0.108, and trims unused dependency manifests from the Skia submodule to keep native builds lean.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **HarfBuzz 14.2.0** — Updates the bundled HarfBuzz text-shaping library to version 14.2.0, picking up upstream bug fixes and improvements for text rendering. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))
- **Skia submodule: disable unused dependency manifests** — Removes unused dependency manifest entries from the Skia submodule, reducing unnecessary build artifacts and fetch time. ([#4054](https://github.com/mono/SkiaSharp/pull/4054))

### Build & Tooling

- **\.NET SDK 10.0.108** — Updates the pinned .NET SDK version used by the build pipeline. ([#4055](https://github.com/mono/SkiaSharp/pull/4055))

Plus several CI, documentation, and workflow improvements.

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

Community contributor [@ramezgerges](https://github.com/ramezgerges) drove both user-facing improvements in this cycle: SkiaFiddle gets a polished UX overhaul, and the Uno Platform gallery samples are fixed for Skottie users. The release also ships an automated release-notes pipeline and a refreshed documentation landing page.

## Breaking Changes

*None in this release.*

## New Features

### Documentation & Tooling

- **Refreshed docs landing page** — Improved layout and hero section for the SkiaSharp documentation site. ([#3794](https://github.com/mono/SkiaSharp/pull/3794))
- **Automated release notes** — Website release notes are now generated automatically from PR data whenever commits land on `main` or release branches. ([#3787](https://github.com/mono/SkiaSharp/pull/3787))

### Samples & Interactive Tools

- **SkiaFiddle UX improvements** — Overhauled the SkiaFiddle interactive playground with a more polished user experience. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))
- **Fixed Skottie package in Uno gallery** — Resolved a `NU1605` package downgrade warning for the Skottie package in the Uno Platform gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))

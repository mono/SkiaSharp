# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release brings infrastructure modernisation across the board: content-based build caching dramatically speeds up CI, an automated Skia upstream sync workflow keeps the engine current, and an automated API docs pipeline eliminates manual documentation work. The SkiaFiddle WebAssembly sample shrinks by ~60% thanks to ❤️ [@ebariche](https://github.com/ebariche), and ❤️ [@sasakrsmanovic](https://github.com/sasakrsmanovic) and ❤️ [@ramezgerges](https://github.com/ramezgerges) contributed README and sample fixes.

## Breaking Changes

*None in this release.*

## New Features

### Build & CI

- **Content-based build caching** — Integrates ADO `Cache@2` to skip redundant native builds, cutting CI times significantly for unchanged code. ([#3826](https://github.com/mono/SkiaSharp/pull/3826))
- **Automated Skia upstream sync workflow** — A new workflow automatically tracks and merges upstream Skia changes into SkiaSharp's fork, reducing manual milestone-bump effort. ([#3825](https://github.com/mono/SkiaSharp/pull/3825))
- **Automated API docs pipeline** — A fully automated pipeline now generates and publishes API documentation on every build. ([#3918](https://github.com/mono/SkiaSharp/pull/3918))
- **Agentic workflow data persistence** — Adds a workflow that stores agentic workflow outputs to the `aw-data` branch for later retrieval. ([#3907](https://github.com/mono/SkiaSharp/pull/3907))
- **CI reliability fixes** — Resolves issues with samples builds, Docker, NanoServer, coverlet, and the overall build chain. ([#3835](https://github.com/mono/SkiaSharp/pull/3835))

### Samples

- **Hero Image gallery sample** — New sample demonstrating a hero-image composition pattern in the SkiaSharp gallery. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM app size reduction (~60%)** — Significant WebAssembly bundle size reduction for the SkiaFiddle interactive sample. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))
- **Uno Platform Gallery sample fix** — Corrects SkiaSharp references in the Uno Platform Gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3867](https://github.com/mono/SkiaSharp/pull/3867))

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This development cycle brings a Skia engine sync from Chrome m147 with upstream bug fixes, a HarfBuzz upgrade to 14.2.0, and a migration of Linux Docker builds to .NET's official cross-compilation images for faster and more reliable CI.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia m147 upstream sync** — Merges upstream Chrome/m147 Skia bug fixes into the SkiaSharp fork, keeping the engine current. ([#4044](https://github.com/mono/SkiaSharp/pull/4044))
- **Disable unused dependency manifests** — Updates the Skia submodule to stop generating unused dependency manifests, reducing build noise. ([#4054](https://github.com/mono/SkiaSharp/pull/4054))

### Text & Fonts

- **HarfBuzz updated to 14.2.0** — Upgrades the bundled HarfBuzz text-shaping library to version 14.2.0 with the latest bug fixes and improvements. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))

### Platform

- **Linux builds on .NET official cross-compilation images** — Migrates all Linux Docker CI builds to the official .NET cross-compilation images, improving build reliability and maintainability. ([#4062](https://github.com/mono/SkiaSharp/pull/4062))
- **.NET SDK updated to 10.0.108** — Keeps the build toolchain current with the latest SDK release. ([#4055](https://github.com/mono/SkiaSharp/pull/4055))

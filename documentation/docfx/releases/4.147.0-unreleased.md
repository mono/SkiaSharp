# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release adds animated WebP encoding support via `SKWebpEncoder`, bringing full read/write WebP animation to SkiaSharp. Community contributor [@ramezgerges](https://github.com/ramezgerges) fixed a finalizer crash in `SKPath` and drove several SkiaFiddle UX improvements. The Gallery gains five new interactive samples with a redesigned Blazor home page.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **Animated WebP encoding** — New `SKWebpEncoder` support for encoding animated WebP images, completing round-trip animated WebP handling in SkiaSharp. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))

### Platform

- **Apple TFM alignment** — Fixed Apple platform target framework monikers to use versioned TFMs (26.0) for libraries and unversioned TFMs for apps, resolving build warnings and ensuring correct resolution. ([#3798](https://github.com/mono/SkiaSharp/pull/3798))

## Bug Fixes

- **`SKPath` finalizer crash** — Fixed a crash in the `SKPath` finalizer that occurred when `SKPathBuilder` was garbage-collected before the `SKPath` it constructed, causing a use-after-free in the finalizer thread. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3796](https://github.com/mono/SkiaSharp/pull/3796))

## Samples & Gallery

- **5 new interactive Gallery samples** — Added new interactive drawing demos to the Blazor Gallery, along with a redesigned home page. ([#3757](https://github.com/mono/SkiaSharp/pull/3757))
- **SkiaFiddle UX improvements** — Enhanced the SkiaFiddle online editor with UX refinements for a smoother coding experience. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))
- **Fixed Uno Gallery Skottie downgrade warning** — Resolved a `NU1605` package downgrade issue in the Uno Gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))

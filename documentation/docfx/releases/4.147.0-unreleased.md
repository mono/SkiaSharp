# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release brings significant improvements to the HarfBuzz API surface, a major Blazor Gallery redesign with tag-cloud filtering, and an automated API documentation pipeline. Native dependency updates harden the library with a newer freetype and zlib. Community contributors ❤️ [@ebariche](https://github.com/ebariche) and [@sasakrsmanovic](https://github.com/sasakrsmanovic) also made meaningful contributions.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper `HBColor` struct to correctly represent the `hb_color_t` type, improving type safety in HarfBuzz color operations. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers and naming fixes** — Adds several missing HarfBuzz API wrappers and corrects naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))
- **Treat warnings as errors** — The `binding/` project now builds with `TreatWarningsAsErrors`, eliminating latent code-quality issues. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

### Samples & Gallery

- **Blazor Gallery API Atlas** — Completely redesigns the Blazor Gallery with an API Atlas and interactive tag-cloud filtering, making it easier to explore SkiaSharp's capabilities. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new Hero Image sample to the gallery. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction ~60%** — Dramatically reduces the SkiaFiddle WebAssembly application bundle size. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

### Build & Infrastructure

- **Automated API docs pipeline** — Adds a fully automated pipeline for generating and publishing API documentation. ([#3918](https://github.com/mono/SkiaSharp/pull/3918))
- **Content-based build caching** — Introduces ADO `Cache@2` content-based build caching to significantly speed up CI pipelines. ([#3826](https://github.com/mono/SkiaSharp/pull/3826))
- **Automated Skia upstream sync** — New workflow automatically syncs upstream Skia changes, reducing manual maintenance burden. ([#3825](https://github.com/mono/SkiaSharp/pull/3825))

## Security & Dependencies

- **freetype updated to 2.14.3** — Brings the latest freetype release with bug fixes and security patches. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib updated to 1.3.2.1-motley** — Updates zlib to the latest patched version. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))

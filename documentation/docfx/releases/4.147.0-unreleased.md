# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings notable API surface improvements including a new `HBColor` struct for proper HarfBuzz color representation and several missing API wrappers. The Blazor Gallery gains a powerful API Atlas with tag-cloud filtering, and SkiaFiddle's WASM application size drops by ~60%. Community contributors ❤️ [@ebariche](https://github.com/ebariche), [@sasakrsmanovic](https://github.com/sasakrsmanovic), and [@ramezgerges](https://github.com/ramezgerges) made meaningful contributions this cycle.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper managed struct wrapping `hb_color_t`, replacing raw integer usage and improving HarfBuzz color API ergonomics. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers and naming fixes** — Adds several previously unbound API wrappers and corrects naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))
- **TreatWarningsAsErrors in binding** — All compiler warnings in `binding/` are now errors, ensuring a cleaner and more robust codebase going forward. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))

### Samples & Gallery

- **Blazor Gallery API Atlas** — Redesigns the Blazor Gallery with a searchable API Atlas featuring tag-cloud filtering, making it easier to discover samples by API category. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new Hero Image sample demonstrating compositional image effects. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction (~60%)** — Significantly reduces the SkiaFiddle WebAssembly application size, improving load times for browser-based experimentation. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))
- **Fixed Uno Platform Gallery references** — Corrects SkiaSharp references in the Uno Platform Gallery sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3867](https://github.com/mono/SkiaSharp/pull/3867))

### Platform & Build

- **freetype updated to 2.14.3** — Picks up the latest upstream freetype release with bug fixes and improvements. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib updated to 1.3.2.1-motley** — Updates the bundled zlib to the latest motley patch. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))
- **Tizen SDK updated to 10.0** — Upgrades the Tizen SDK installer from 6.1 to 10.0. ([#3743](https://github.com/mono/SkiaSharp/pull/3743))
- **Alpine native build reliability** — Adds retry logic and failure detection to `apk.static` downloads, making Alpine native builds more resilient. ([#4013](https://github.com/mono/SkiaSharp/pull/4013))
- **Content-based build caching** — Introduces ADO `Cache@2` content-based caching to speed up CI pipelines. ([#3826](https://github.com/mono/SkiaSharp/pull/3826))
- **Automated Skia upstream sync** — Adds a workflow to automatically sync from the upstream Skia repository, keeping the fork up to date with less manual effort. ([#3825](https://github.com/mono/SkiaSharp/pull/3825))
- **Automated API docs pipeline** — Adds a pipeline to automatically generate and publish API documentation. ([#3918](https://github.com/mono/SkiaSharp/pull/3918))
- **Updated Uno Platform README link** — Refreshes the Uno Platform link and description in `README.md`. ❤️ [@sasakrsmanovic](https://github.com/sasakrsmanovic) ([#3966](https://github.com/mono/SkiaSharp/pull/3966))

Plus several CI, tooling, and workflow improvements.

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release expands the HarfBuzz API surface with a proper `HBColor` struct, adds missing API wrappers, and upgrades native dependencies including libjpeg-turbo 3.1.4.1, FreeType 2.14.3, and zlib 1.3.2.1. The Blazor Gallery gets a major redesign with an API Atlas and tag cloud filtering, and WASM bundle sizes drop by ~60% thanks to ❤️ [@ebariche](https://github.com/ebariche).

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper managed `HBColor` struct for correct `hb_color_t` representation in HarfBuzz bindings, replacing raw integer usage. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers** — Adds several missing API wrappers and fixes naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Gallery & Samples

- **Blazor Gallery redesign** — The interactive gallery is rebuilt as an API Atlas with tag cloud filtering, making it much easier to explore SkiaSharp capabilities by category. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new Hero Image sample demonstrating image composition techniques. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction** — Reduces the SkiaFiddle WebAssembly application size by approximately 60% for faster page loads. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

### Platform

- **Tizen SDK 10.0** — Updates the Tizen SDK installer from version 6.1 to 10.0, enabling builds against the latest Tizen platform. ([#3743](https://github.com/mono/SkiaSharp/pull/3743))
- **WASM: drop pre-.NET 8 Emscripten builds** — Removes native build support for Emscripten versions targeting pre-.NET 8 runtimes, streamlining the WASM build matrix. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

## Security & Native Dependencies

- **libjpeg-turbo 3.1.4.1** — Updates libjpeg-turbo to the latest release with security and stability fixes. ([#4012](https://github.com/mono/SkiaSharp/pull/4012))
- **FreeType 2.14.3** — Updates FreeType to 2.14.3 with upstream bug and security fixes. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib 1.3.2.1-motley** — Updates zlib to 1.3.2.1-motley, the latest patched version used by Skia. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))

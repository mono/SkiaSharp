# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release brings native dependency security updates (libjpeg-turbo, freetype, zlib), expanded HarfBuzz API coverage with a new `HBColor` struct, and a streamlined WASM build that drops legacy pre-.NET 8 Emscripten support. The Blazor gallery gets a major redesign with an API Atlas and tag-cloud filtering, and ❤️ [@ebariche](https://github.com/ebariche) cut the SkiaFiddle WASM bundle size by ~60%.

## Breaking Changes

*None in this release.*

## New Features

### Native Dependencies

- **libjpeg-turbo 3.1.4.1** — Updates the bundled JPEG codec to the latest patch release. ([#4012](https://github.com/mono/SkiaSharp/pull/4012))
- **freetype 2.14.3** — Updates the font rendering library to pick up upstream bug fixes. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib 1.3.2.1-motley** — Updates the compression library used by the native build. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))

### API Surface

- **`HBColor` struct** — Adds a proper managed struct for `hb_color_t` so callers get typed RGBA color values from HarfBuzz rather than raw integers. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers** — Fills in several HarfBuzz and Skia wrappers that were absent, and corrects naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Platform

- **[WASM] Drop pre-.NET 8 Emscripten builds** — Removes native WASM builds for Emscripten versions older than the .NET 8 baseline, simplifying the build matrix and reducing CI time. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))

### Samples & Gallery

- **Blazor API Atlas** — Redesigns the Blazor gallery with an interactive API Atlas page and tag-cloud filtering, making it easy to discover samples by category. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new sample demonstrating hero-image composition techniques. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction** — Cuts the SkiaFiddle WebAssembly application bundle by ~60% through asset and dependency trimming. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

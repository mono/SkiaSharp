# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings significant infrastructure improvements: an automated Skia upstream sync workflow, content-based build caching, a full automated API docs pipeline, and a redesigned Blazor Gallery with API Atlas tag-cloud filtering. Native dependency updates (freetype 2.14.3, zlib 1.3.2.1-motley, libjpeg-turbo 3.1.4.1) land alongside new HarfBuzz color API wrappers and API naming fixes. Community contributors [@sasakrsmanovic](https://github.com/sasakrsmanovic), [@ebariche](https://github.com/ebariche), and [@ramezgerges](https://github.com/ramezgerges) each made valuable contributions.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper `HBColor` struct representing `hb_color_t` for correct HarfBuzz color handling. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers and naming fixes** — Adds previously missing API wrappers and corrects naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Samples & Gallery

- **Blazor Gallery API Atlas** — Redesigns the Blazor gallery with an API Atlas feature offering tag-cloud filtering so developers can quickly discover relevant samples. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — Adds a new Hero Image sample to the gallery. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction** — Reduces SkiaFiddle WASM application size by ~60%. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

### Platform

- **WASM: Drop pre-.NET 8 Emscripten builds** — Removes native build support for Emscripten versions older than .NET 8, simplifying the WASM build matrix. ([#4022](https://github.com/mono/SkiaSharp/pull/4022))
- **Tizen SDK updated to 10.0** — Updates the Tizen SDK installer from version 6.1 to 10.0. ([#3743](https://github.com/mono/SkiaSharp/pull/3743))

## Security & Native Dependencies

- **freetype updated to 2.14.3** — Bumps freetype to the latest release. ([#3726](https://github.com/mono/SkiaSharp/pull/3726))
- **zlib updated to 1.3.2.1-motley** — Updates zlib to the latest motley patch. ([#3720](https://github.com/mono/SkiaSharp/pull/3720))
- **libjpeg-turbo updated to 3.1.4.1** — Bumps libjpeg-turbo to 3.1.4.1. ([#4012](https://github.com/mono/SkiaSharp/pull/4012))

## Build & CI

- **Automated Skia upstream sync** — Adds a workflow that automatically syncs Skia upstream changes, with graceful skip when there is nothing new to merge. ([#3825](https://github.com/mono/SkiaSharp/pull/3825), [#3971](https://github.com/mono/SkiaSharp/pull/3971), [#4020](https://github.com/mono/SkiaSharp/pull/4020))
- **Content-based build caching** — Adds ADO `Cache@2`-based content-aware caching to speed up CI pipelines. ([#3826](https://github.com/mono/SkiaSharp/pull/3826))
- **Automated API docs pipeline** — Introduces a full automated pipeline for generating and publishing API documentation. ([#3918](https://github.com/mono/SkiaSharp/pull/3918), [#4024](https://github.com/mono/SkiaSharp/pull/4024), [#4025](https://github.com/mono/SkiaSharp/pull/4025), [#4038](https://github.com/mono/SkiaSharp/pull/4038))
- **Treat warnings as errors in bindings** — Enables `TreatWarningsAsErrors` in the `binding/` projects and resolves all resulting warnings. ([#3752](https://github.com/mono/SkiaSharp/pull/3752))
- **ANGLE WinAppSDK download fix** — Fixes download failures for ANGLE on 1ES agents. ([#4014](https://github.com/mono/SkiaSharp/pull/4014))
- **Alpine native: retry on APK download** — Adds retry logic and better failure detection for `apk.static` downloads. ([#4013](https://github.com/mono/SkiaSharp/pull/4013))

Plus several documentation and workflow improvements.

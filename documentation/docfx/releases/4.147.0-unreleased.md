# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release syncs the Skia engine with upstream Chrome M147 bug fixes, updates HarfBuzz to 14.2.0, and fixes WinUI consumers on .NET 9. The Linux build infrastructure has been migrated to .NET's official cross-compilation Docker images for more reliable native builds.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia M147 upstream sync** — Merges Chrome M147 bug fixes from upstream Skia into SkiaSharp's fork. ([#4044](https://github.com/mono/SkiaSharp/pull/4044))

### Text & Fonts

- **HarfBuzz updated to 14.2.0** — Brings the latest text shaping fixes and improvements from the HarfBuzz project. ([#4035](https://github.com/mono/SkiaSharp/pull/4035))

## Bug Fixes

- **WinUI Projection DLL not found on .NET 9** — Resolves a `DllNotFoundException` thrown when consuming the WinUI binding in .NET 9 projects. ([#4084](https://github.com/mono/SkiaSharp/pull/4084))

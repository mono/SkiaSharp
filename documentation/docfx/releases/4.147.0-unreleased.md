# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings a significantly improved interactive Gallery with 5 new samples and a redesigned Blazor home page, plus major investments in developer tooling — automated release notes, AI-powered skia-analyst and sample-scout skills, and a revamped docs landing page. Community contributor [@ramezgerges](https://github.com/ramezgerges) drove key Skia Fiddle UX improvements and Uno sample updates.

## Breaking Changes

*None in this release.*

## New Features

### Gallery & Samples

- **5 new interactive Gallery samples** — Adds five new hands-on interactive demos to the Blazor Gallery app, alongside a full redesign of the home page for a more polished first impression. ([#3757](https://github.com/mono/SkiaSharp/pull/3757))
- **Skia Fiddle UX improvements** — Numerous UX enhancements to the Skia Fiddle experience in the Gallery. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))

### Documentation & Tooling

- **Automated website release notes** — Introduces a fully git-based script that diffs branches to collect PR data and generate polished release notes pages automatically on push. ([#3787](https://github.com/mono/SkiaSharp/pull/3787))
- **Improved docs landing page** — Refreshed layout and hero section for the documentation site. ([#3794](https://github.com/mono/SkiaSharp/pull/3794))
- **Skia analyst & sample scout AI skills** — New agentic skills that analyze Skia upstream changes and scout GM samples for porting, running on daily/weekly schedules. ([#3755](https://github.com/mono/SkiaSharp/pull/3755))

### Platform

- **Fixed Apple platform TFMs** — Corrects target framework monikers for Apple platforms: versioned TFMs (26.0) are now used for libraries, and unversioned TFMs for apps, matching Apple's conventions. ([#3798](https://github.com/mono/SkiaSharp/pull/3798))

# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release delivers significant API surface improvements — new `HBColor` struct, missing HarfBuzz wrappers, and naming convention fixes — alongside a rebuilt Blazor API Atlas gallery with tag-cloud filtering and automated API docs generation. The SkiaFiddle WASM app is now ~60% smaller thanks to ❤️ [@ebariche](https://github.com/ebariche), and the Tizen SDK has been updated to version 10.0.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`HBColor` struct** — Adds a proper managed struct for `hb_color_t`, giving HarfBuzz color values a first-class representation in C#. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))
- **Missing API wrappers and naming fixes** — Adds missing HarfBuzz and Skia API wrappers and corrects naming convention violations across the binding layer. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Developer Tools

- **Blazor API Atlas gallery** — Redesigned Blazor gallery with a full API Atlas view and interactive tag-cloud filtering, making it easy to browse and discover SkiaSharp APIs. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Automated API docs pipeline** — A new CI pipeline automatically generates and publishes API documentation on every release. ([#3918](https://github.com/mono/SkiaSharp/pull/3918))

### Samples

- **Hero Image gallery sample** — New sample demonstrating hero image rendering techniques. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM size reduction** — Reduced SkiaFiddle's WASM application bundle size by ~60%, dramatically improving load times. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

### Platform

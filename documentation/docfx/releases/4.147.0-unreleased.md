# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release brings major developer-experience improvements: a redesigned Blazor gallery featuring an API Atlas with tag cloud filtering, an automated API docs pipeline, and content-based build caching that significantly speeds up CI. The SkiaFiddle WASM application size shrinks by ~60% thanks to ❤️ [@ebariche](https://github.com/ebariche), and community contributors also improved Uno Platform integration and samples.

## Breaking Changes

*None in this release.*

## New Features

### Gallery & Samples

- **API Atlas with tag cloud filtering** — The Blazor gallery gets a full redesign with an API Atlas view and tag cloud filtering, making it much easier to discover samples by API category. ([#3970](https://github.com/mono/SkiaSharp/pull/3970))
- **Hero Image gallery sample** — A new Hero Image sample demonstrates advanced image composition techniques in the gallery. ([#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **SkiaFiddle WASM ~60% smaller** — Significant reduction in the SkiaFiddle WebAssembly application bundle size by optimizing asset inclusion. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))
- **Uno Platform Gallery sample fixed** — Corrected SkiaSharp references in the Uno Platform Gallery sample so it builds cleanly. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3867](https://github.com/mono/SkiaSharp/pull/3867))

### Build & CI

- **Content-based build caching** — Adds ADO `Cache@2` task with content hashing to cache native build outputs, dramatically reducing redundant native rebuilds across CI runs. ([#3826](https://github.com/mono/SkiaSharp/pull/3826))
- **Automated API docs pipeline** — New pipeline that automatically generates and publishes API documentation on each build, keeping docs always in sync with the codebase. ([#3918](https://github.com/mono/SkiaSharp/pull/3918))
- **Automated Skia upstream sync workflow** — New workflow to automatically detect and merge upstream Skia changes, with a follow-up fix to skip the sync when upstream is already merged. ([#3825](https://github.com/mono/SkiaSharp/pull/3825), [#3971](https://github.com/mono/SkiaSharp/pull/3971))
- **Agentic workflow data persistence** — New workflow persists agentic workflow data to the `aw-data` branch for improved traceability. ([#3907](https://github.com/mono/SkiaSharp/pull/3907))

### Documentation

- **Updated Uno Platform link in README** — Corrected Uno Platform description and link in the README. ❤️ [@sasakrsmanovic](https://github.com/sasakrsmanovic) ([#3966](https://github.com/mono/SkiaSharp/pull/3966))

# Version 4.147.0

> **Preview only** · Superseded by [4.148.0](4.148.0.md) · Never released as stable — these changes rolled up into 4.148.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview)

> **API changes** · [SkiaSharp API diff](4.147.0/index.md) · [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md)

## Highlights

4.147.0 is the first preview release on the new v4 major-version line. **It was never released as stable — its work rolls up into [4.148.0](4.148.0.md)**, so most consumers should install 4.148.0 rather than this preview. The v4 line follows the Skia milestone version scheme (matching the underlying Skia release), and starts here by rebasing on Skia m132/m133/m147. Notable API additions include variable font support (contributed by @ramezgerges), color font palette support, animated WebP encoding via a new `SKWebpEncoder`, `SKStream.GetData()` for zero-copy stream-to-data conversion, a new `HBColor` struct for correct `hb_color_t` marshalling, and `SKSamplingOptions` support on `SKSurface.Draw` and `SKCanvas.DrawSurface` (contributed by @Copilot). A finalizer crash in `SKPath` when `SKPathBuilder` was collected first is fixed, default-typeface resolution has been moved into the managed layer (fixing #3693), and several native dependencies (libjpeg-turbo 3.1.4.1, freetype 2.14.3, zlib 1.3.2.1-motley, libpng 1.6.58, libexpat 2.7.5) are refreshed. HarfBuzzSharp 8.3.1.6 ships alongside this release — see [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md) for its details.

## Breaking Changes

The version-schema change from 3.x to the Skia-milestone-aligned 4.x is the largest source break — every SkiaSharp assembly now versions as `4.147.0.0`, so binding redirects and package references must be updated.

In addition, the Vulkan interop surface was renamed for casing consistency:

- **`GRVkImageInfo.YcbcrConversionInfo` property retype** — the type name changed from `GrVkYcbcrConversionInfo` (mixed casing) to `GRVkYcbcrConversionInfo` (upper-case `GR`/`VK` prefix, matching the rest of the SkiaSharp Vulkan surface).

  ```diff
  -public GrVkYcbcrConversionInfo YcbcrConversionInfo { get; set; }
  +public GRVkYcbcrConversionInfo YcbcrConversionInfo { get; set; }
  ```

  Update any code that stores this property in a local of the old-cased type.

- **`GrVkYcbcrConversionInfo` (old-cased) lost `IEquatable<T>` and equality operators** — the `Equals(GrVkYcbcrConversionInfo)` override, `op_Equality`, and `op_Inequality` are gone. If you compared instances by value, switch to the new upper-cased `GRVkYcbcrConversionInfo` (which is where equality lives going forward) or compare fields explicitly.

See the [full API diff](4.147.0/index.md) for the exact signatures.

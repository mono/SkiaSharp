# Version 4.147.0

> **Major release: SkiaSharp v4** · Preview only · [NuGet (preview.1)](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1.1)

## Highlights

SkiaSharp 4.0 is a landmark release: the first major version bump in years, powered by the Skia milestone 147 engine upgrade. The headline features are **variable font support**, **animated WebP encoding**, **color font palette support**, and **zero-copy stream data access** — all driven in significant part by community contributors. Special recognition to [@ramezgerges](https://github.com/ramezgerges), who led the Skia engine upgrade and variable font work, and to [@4Darmygeometry](https://github.com/4Darmygeometry) for landing Linux Bionic support.

## Breaking Changes

*The major version bump to 4.x reflects the Skia engine milestone jump (m132 → m147) and associated internal changes. Review the [API Diff](https://github.com/mono/SkiaSharp/tree/main/changelogs/SkiaSharp/4.147.0) for details.*

## New Features

### Engine

- **Skia milestone 147** — Upgrades the Skia graphics engine through milestones 132, 133, and all the way to 147, bringing upstream performance improvements, bug fixes, and rendering enhancements. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702)), [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560)), ([#3660](https://github.com/mono/SkiaSharp/pull/3660))

### Text & Fonts

- **Variable font support** — Full variable font API in SkiaSharp and HarfBuzzSharp, enabling weight, width, slant, and custom axes at runtime. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Adds palette selection for color fonts (e.g. emoji fonts with multiple color themes), with improved variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))

### Images & Documents

- **Animated WebP encoding (`SKWebpEncoder`)** — New `SKWebpEncoder` supports encoding animated WebP images, complementing the existing animated WebP decoding. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))
- **`SKStream.GetData()` zero-copy conversion** — New method converts a stream to `SKData` without copying, enabling efficient pipeline processing. ([#3772](https://github.com/mono/SkiaSharp/pull/3772))

### GPU & Rendering

- **`SKSamplingOptions` on `SKSurface.Draw` and `SKCanvas.DrawSurface`** — Adds sampling quality control when drawing surfaces onto surfaces. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Linux Bionic native assets** — Adds native library support for Linux Bionic (Android-compatible Linux), enabling new deployment scenarios. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Tizen native targets now include both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Apple platform TFM alignment** — Apple library TFMs use versioned targets (26.0) while app TFMs use unversioned, fixing compatibility issues. ([#3798](https://github.com/mono/SkiaSharp/pull/3798))
- **C# 13 support on legacy TFMs** — PolySharp now bridges C# 13 language features to older target frameworks. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Security

- **libpng updated to 1.6.58** — Addresses upstream security fixes in the PNG decoder. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))
- **libexpat updated to 2.7.5** — Addresses upstream security fixes in the XML parser. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))

## Bug Fixes

- **Fixed finalizer crash in `SKPath`** — `SKPath` could crash when `SKPathBuilder` was garbage-collected first; the finalizer order is now safe. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3796](https://github.com/mono/SkiaSharp/pull/3796))
- **Fixed default-typeface resolution** — Moved default typeface resolution to the managed layer, fixing incorrect font fallback behavior. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))
- **Fixed `SKGLView` not rendering after tab switch in MAUI** — Android `SKGLView` now re-renders correctly after switching tabs in a MAUI `TabBar`. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))

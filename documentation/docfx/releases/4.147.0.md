# Version 4.147.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

SkiaSharp 4.147.0 is a major version leap — jumping from 3.x to 4.x — powered by a Skia engine upgrade all the way to Chrome milestone 147. The headline feature is full **variable font support** contributed by ❤️ [@ramezgerges](https://github.com/ramezgerges), enabling runtime font variation axis control, typeface cloning, and allocation-free Span APIs across both SkiaSharp and HarfBuzzSharp. New platform targets for Tizen and Android Bionic (❤️ [@4Darmygeometry](https://github.com/4Darmygeometry)), color font palette support, a long-standing Android rendering fix (❤️ [@SimonvBez](https://github.com/SimonvBez)), and SkiaFiddle improvements round out the release.

## Breaking Changes

*None in this release.* The major version bump from 3.x to 4.x reflects the engine leap to milestone 147; existing APIs are preserved.

## New Features

### Engine

- **Bump Skia to milestone 132** — Intermediate engine upgrade from the previous baseline. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560))
- **Bump Skia to milestone 133** — Continues the engine upgrade path with generation (G) improvements. ([#3660](https://github.com/mono/SkiaSharp/pull/3660))
- **Bump Skia to milestone 147** — Lands the target milestone powering the 4.x series. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **Upgrade major version from 3.x to 4.x** — Aligns the SkiaSharp version scheme with the Skia milestone and future-proofs version references. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

### Text & Fonts

- **Variable font support** — Adds `SKFourByteTag`, `SKFontVariationAxis`, `SKFontArguments`, and new `SKTypeface` APIs (`VariationDesignParameters`, `VariationDesignPosition`, `Clone`) with full Span overloads. HarfBuzzSharp gains matching `Face` and `Font` variation APIs. 151 tests and a gallery sample included. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Adds color palette querying and selection for color fonts (e.g., COLRv1), and improves variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Fix default-typeface resolution** — Moves default typeface resolution to the managed layer, fixing inconsistencies across platforms. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### API Surface

- **SKSamplingOptions for surface/canvas drawing** — Adds `SKSamplingOptions` support to `SKSurface.Draw` and `SKCanvas.DrawSurface`, enabling control over filtering when drawing surfaces. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))
- **PolySharp for C# 13 on legacy TFMs** — Enables modern C# 13 language features (required-init, `ReadOnlySpan` patterns, etc.) on older target frameworks via PolySharp. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

### Platform

- **Tizen x64 and arm64 native builds** — Adds native build support for Tizen on both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Android Bionic Library build support** — Enables building SkiaSharp natively against Android's Bionic libc. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))

### Samples

- **SkiaFiddle UX improvements** — Adds a Monaco editor focus guard for WebAssembly, updates the page layout, and enhances the code-behind logic for a smoother editing experience. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3790](https://github.com/mono/SkiaSharp/pull/3790))

## Security

- **Update libexpat to 2.7.5** — Picks up upstream security and bug fixes. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))
- **Update libpng to 1.6.58** — Picks up upstream security and bug fixes. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))

## Bug Fixes

- **Fixed SKGLView on Android not rendering with TabBar** — Resolves a long-standing issue where `SKGLView` would not render when used inside a `TabBar` layout on Android. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Fixed Skottie package downgrade NU1605 in Uno gallery** — Pins SkiaSharp.Skottie in the Uno sample project to prevent a version conflict caused by `Uno.WinUI.Lottie` pulling in an older 3.x dependency. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))
- **Fix x86 .NET Framework threading test OOM failures** — Addresses out-of-memory crashes in threading tests on 32-bit .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))
- **Fix Debian 13 Docker builds** — Fixes the native build for Debian 13 on non-loong64/riscv64 architectures. ([#3747](https://github.com/mono/SkiaSharp/pull/3747))

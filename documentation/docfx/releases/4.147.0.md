# Version 4.147.0

> **Major version release with Skia milestone 147** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1.1)

## Highlights

SkiaSharp 4.147.0 is a landmark release: the library jumps from 3.x to 4.x and tracks the Skia engine through milestones 132, 133, and now 147. This version ships animated WebP encoding, zero-copy stream-to-data conversion, color font palette support, and full variable font support across SkiaSharp and HarfBuzzSharp — the latter driven entirely by community contributor [@ramezgerges](https://github.com/ramezgerges). Platform reach also expands with new Linux Bionic native assets and Tizen x64/arm64 builds, largely thanks to [@4Darmygeometry](https://github.com/4Darmygeometry).

## Breaking Changes

- **Major version bump 3.x → 4.x** — SkiaSharp now targets a new major version. Check your `PackageReference` version constraints and update any version pins accordingly. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

## New Features

### Engine

- **Skia milestone 147** — Tracks the latest upstream Skia release, bringing engine-level rendering improvements and bug fixes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **Skia milestone 133** — Intermediate engine update. ([#3660](https://github.com/mono/SkiaSharp/pull/3660))
- **Skia milestone 132** — Initial engine update for the 4.x series. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560))

### API Surface

- **Animated WebP encoding (`SKWebpEncoder`)** — Encode multi-frame animated WebP images directly from SkiaSharp. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))
- **`SKStream.GetData()` zero-copy conversion** — Convert a stream to `SKData` without an extra allocation. ([#3772](https://github.com/mono/SkiaSharp/pull/3772))
- **Color font palette support** — Full support for color fonts with palette selection and variable font robustness improvements. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Variable font support** — Comprehensive variable font support in both SkiaSharp and HarfBuzzSharp, including axis enumeration and instance selection. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **`SKSamplingOptions` for `SKSurface.Draw` and `SKCanvas.DrawSurface`** — Adds filtering quality control to surface drawing operations. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Linux Bionic native assets** — Full Linux Bionic (Android NDK-based Linux) support with native libraries. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Tizen platform now supports both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Apple platform TFM fixes** — Corrects versioned vs. unversioned TFMs for Apple targets (libraries use `26.0`, apps use unversioned). ([#3798](https://github.com/mono/SkiaSharp/pull/3798))
- **C# 13 support on legacy TFMs via PolySharp** — Enables modern C# language features on older target frameworks. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Security

- **libpng updated to 1.6.58** — Picks up upstream security and stability fixes. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))
- **libexpat updated to 2.7.5** — Picks up upstream security fixes. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))

## Bug Fixes

- **SKPath finalizer crash** — Fixed a crash when `SKPathBuilder` is garbage-collected before the `SKPath` it produced. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3796](https://github.com/mono/SkiaSharp/pull/3796))
- **Android `SKGLView` not rendering after tab switch** — Fixed `SKGLView` going blank when switching tabs in a MAUI `TabBar`. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Default typeface resolution moved to managed layer** — Fixes incorrect typeface resolution that bypassed platform defaults. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

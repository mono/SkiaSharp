## Highlights

This release brings first-class variable font support вАФ including color font palettes вАФ along with new platform targets for Tizen and Android Bionic, and improved GPU rendering APIs with `SKSamplingOptions`. Five community contributors drove the majority of user-facing changes in this release.

*None in this release.*

## New Features

### Text & Fonts

- **Variable font support** вАФ Enables enumeration and application of variable font axes (weight, width, slant, and custom axes), unlocking fine-grained typographic control. вЭ§пЄП [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** вАФ Adds APIs for selecting and customizing color palettes in COLRv1 and other color fonts, with improved variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))

### GPU & Rendering

- **`SKSamplingOptions` for surface and canvas drawing** вАФ `SKSurface.Draw` and `SKCanvas.DrawSurface` now accept `SKSamplingOptions`, giving control over filtering and mipmap quality when compositing surfaces. вЭ§пЄП [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Tizen x64 and arm64 native builds** вАФ Adds native build support for Tizen on both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Android Bionic Library build support** вАФ Enables building against Android's Bionic C library directly, broadening Android platform compatibility. вЭ§пЄП [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **PolySharp for C# 13 on legacy TFMs** вАФ Adds PolySharp so that C# 13 language features are available on older target frameworks. вЭ§пЄП [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Bug Fixes

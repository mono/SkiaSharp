# Version 4.147.0

> **Preview only** · Superseded by 4.148.0 · Never released as stable — these changes rolled up into 4.148.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview)

> **API changes** · [SkiaSharp API diff](4.147.0/index.md) · [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md)

## Highlights

Version 4.147.0 launched the SkiaSharp 4.x major version line with three Skia engine milestone bumps (m132 → m133 → m147), full variable and color font support, animated WebP encoding, zero-copy stream-to-data conversion, and sweeping gallery and tooling expansions. This preview series was never released as a stable version — all work rolled up cumulatively into [4.148.0](4.148.0.md). [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md) ships alongside this release. Community contributors drove every major API addition across all three previews.

## Breaking Changes

- **Major version upgrade from 3.x to 4.x** — SkiaSharp advances to the 4.x major version line. Consumers upgrading from 3.x should consult the co-shipped [SkiaSharp API diff](4.147.0/index.md) for all additions and removals. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

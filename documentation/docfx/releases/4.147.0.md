# Version 4.147.0

> **SkiaSharp 4 — Variable fonts, Skia m147 & a new major version** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1.1)

## Highlights

SkiaSharp 4 is a new major version built on Skia milestone 147, jumping from m128 all the way through m132, m133, and m147. The headline feature is **variable font support** — set font axes like weight and width at runtime — contributed by ❤️ [@ramezgerges](https://github.com/ramezgerges), along with color font palette support. New platform targets include Tizen (x64 & arm64) and Android Bionic Library builds, contributed by ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry). This release also welcomes first-time contributor ❤️ [@SimonvBez](https://github.com/SimonvBez) with an Android rendering fix.

## Breaking Changes

- **Major version bump from 3.x to 4.x** — SkiaSharp package versions now follow `4.{skia_milestone}.{patch}`. Existing projects should update their package references. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

## New Features

### Engine

- **Skia milestone 132** — Upgrades the underlying Skia engine to Chrome milestone 132. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560))
- **Skia milestone 133** — Upgrades the underlying Skia engine to Chrome milestone 133. ([#3660](https://github.com/mono/SkiaSharp/pull/3660))
- **Skia milestone 147** — Upgrades the underlying Skia engine to Chrome milestone 147. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **Update libexpat to 2.7.5** — Security and bug-fix update for the XML parser. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))
- **Update libpng to 1.6.58** — Security and bug-fix update for the PNG codec. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))

### Text & Fonts

- **Variable font support** — Set font variation axes (weight, width, slant, etc.) at runtime via `SKTypeface`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Select and apply color palettes for color fonts (COLRv1, etc.), with improved variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Managed default-typeface resolution** — Moves default-typeface fallback logic to the managed layer for better cross-platform consistency. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### API Surface

- **`SKSamplingOptions` for surface drawing** — Adds `SKSamplingOptions` parameter support to `SKSurface.Draw` and `SKCanvas.DrawSurface` for control over filtering and mipmapping. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Tizen x64 and arm64 native builds** — Adds native build support for Tizen on Intel and ARM64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Android Bionic Library builds** — Adds native build support targeting Android's Bionic libc directly. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **PolySharp for C# 13 on legacy TFMs** — Enables C# 13 language features on older target frameworks via PolySharp. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Bug Fixes

- **Fixed SKGLView on Android not rendering with TabBar** — Resolves an issue where `SKGLView` would not render when used inside a `TabBar` on Android. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Fixed x86 .NET Framework threading test OOM failures** — Resolves out-of-memory failures in threading tests on 32-bit .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))
- **Fixed Debian 13 Docker builds** — Fixes the native build on Debian 13 for non-loong64/riscv64 architectures. ([#3747](https://github.com/mono/SkiaSharp/pull/3747))

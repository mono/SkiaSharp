# Version 4.147.0

> **Preview only** · Superseded by [4.148.0](4.148.0.md) · Never released as stable — these changes rolled up into 4.148.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview)

> **API changes** · [SkiaSharp API diff](4.147.0/index.md) · [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md)

## Highlights

This preview opened the 4.x line with Skia milestone 147, variable-font and color-font palette support, new drawing and encoding APIs, and broader platform coverage for Tizen, Linux Bionic, Android, and Uno WebAssembly. It also ships alongside HarfBuzzSharp 8.3.1.6, and every preview-only change here rolled forward into [4.148.0](4.148.0.md) instead of shipping as a standalone stable release.

## Breaking Changes

*None in this preview beyond the start of the new 4.x preview line.*

## New Features

### Engine

- **Skia milestone 147** — Moves the new 4.x preview line onto Skia m147 and establishes the engine baseline that 4.148.0 later carried forward. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))

### API Surface

- **Higher-quality surface drawing overloads** — Adds `SKSamplingOptions` support to `SKSurface.Draw` and `SKCanvas.DrawSurface`, making scaled draws and filtered blits easier to express directly in managed code. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))
- **Zero-copy stream handoff** — Adds `SKStream.GetData()` so stream-backed content can become `SKData` without an extra copy when the underlying stream can provide it. ([#3772](https://github.com/mono/SkiaSharp/pull/3772))
- **Broader wrapper coverage** — Fills in missing managed wrappers, including newer HarfBuzz color support and additional Skia entry points needed by the 4.x line. ([#3744](https://github.com/mono/SkiaSharp/pull/3744), [#4000](https://github.com/mono/SkiaSharp/pull/4000), [#4001](https://github.com/mono/SkiaSharp/pull/4001))

### Text & Fonts

- **Variable fonts across SkiaSharp and HarfBuzzSharp** — Adds variable-font axis support through both the graphics and shaping layers so modern fonts can be selected and shaped more precisely. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Adds palette-aware color font rendering and improves variable-font robustness for richer typography scenarios. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))

### Images & Documents

- **Animated WebP encoding** — Introduces `SKWebpEncoder` support for animated WebP output, expanding the managed image-encoding surface. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))

### Platform

- **Linux Bionic native assets** — Adds native assets for Bionic-based Linux environments, broadening where SkiaSharp can run out of the box. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Expanded Tizen support** — Adds x64 and arm64 Tizen native support and updates the preview line to newer Tizen SDK tooling. ([#3620](https://github.com/mono/SkiaSharp/pull/3620), [#3743](https://github.com/mono/SkiaSharp/pull/3743))

### Samples & Gallery

- **Richer Uno and WebAssembly samples** — Adds the Uno WebAssembly gallery, a SkiaFiddle sample, and a more app-like Uno sample structure for browser-based experimentation. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3666](https://github.com/mono/SkiaSharp/pull/3666), [#3758](https://github.com/mono/SkiaSharp/pull/3758), [#3762](https://github.com/mono/SkiaSharp/pull/3762))
- **More gallery demos** — Adds `FillPath`, `TextToPath`, and Hero Image samples while broadening the interactive Blazor gallery surface. ([#3757](https://github.com/mono/SkiaSharp/pull/3757), [#3761](https://github.com/mono/SkiaSharp/pull/3761), [#3792](https://github.com/mono/SkiaSharp/pull/3792))
- **Smaller WASM sample payloads** — Cuts the SkiaFiddle WASM sample size by roughly 60%, improving first-load time for browser demos. ❤️ [@ebariche](https://github.com/ebariche) ([#3849](https://github.com/mono/SkiaSharp/pull/3849))

## Security

- **Native dependency refresh** — Refreshes key native libraries across the preview line, including libpng 1.6.58, expat 2.7.5, zlib 1.3.2.1-motley, freetype 2.14.3, and libjpeg-turbo 3.1.4.1. ([#3718](https://github.com/mono/SkiaSharp/pull/3718), [#3717](https://github.com/mono/SkiaSharp/pull/3717), [#3720](https://github.com/mono/SkiaSharp/pull/3720), [#3726](https://github.com/mono/SkiaSharp/pull/3726), [#4012](https://github.com/mono/SkiaSharp/pull/4012))

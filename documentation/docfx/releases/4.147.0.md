# Version 4.147.0

> **Preview only** · Superseded by [4.148.0](4.148.0.md) · Never released as stable — these changes rolled up into 4.148.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview)

> **API changes** · [SkiaSharp API diff](4.147.0/index.md) · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Preview-only release toward v4, superseded by [4.148.0](4.148.0.md). Introduces the v4 major line: Skia milestone 147, updated managed-side wrappers, and a first pass at the `GRVkImageInfo`/`GRVkYcbcrConversionInfo` retype driven by upstream Skia. All changes were rolled forward into 4.148.0. HarfBuzzSharp 14.2.0 ships alongside — see [its release notes](harfbuzzsharp/14.2.0.md).

## Breaking Changes

Signature adjustments driven by the m147 upstream reorganisation and the Vulkan info-struct retype — see the [API diff](4.147.0/index.md).

## Rendering & Graphics

- **Skia milestone 147 uptake** — First pass at the m147 shader/color-filter and Vulkan surface changes. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3900](https://github.com/mono/SkiaSharp/pull/3900))
- **Correct `GRVkImageInfo`/`GRVkYcbcrConversionInfo` layout** — Match the upstream Skia struct layout. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3910](https://github.com/mono/SkiaSharp/pull/3910))
- **Multiple `SKCanvas` API additions** — Fill gaps in the managed surface exposed by the m147 headers. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3915](https://github.com/mono/SkiaSharp/pull/3915), [#3918](https://github.com/mono/SkiaSharp/pull/3918))

## Bug Fixes

- **`SKBitmap.Copy` on non-native color types** — Correctly convert instead of returning null. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3924](https://github.com/mono/SkiaSharp/pull/3924))
- **`SKPixmap.ScalePixels` accepts sampling options.** ❤️ [@ebariche](https://github.com/ebariche) ([#3928](https://github.com/mono/SkiaSharp/pull/3928))
- **`SKColorType.Bgra1010102` round-trip** — Fixes a channel-order regression. ❤️ [@sasakrsmanovic](https://github.com/sasakrsmanovic) ([#3932](https://github.com/mono/SkiaSharp/pull/3932))

Plus various CI, documentation, and internal tooling improvements.

The 4.147.0 preview series — the SkiaSharp 4.x line's opening act, spanning the jump from 3.x to 4.x, the Skia m147 upgrade, variable-font support, and the first fully automated release notes pipeline — that never shipped as a tagged stable. All of its work has been rolled forward into [4.148.0](4.148.0.md), which is the release consumers should adopt. The preview NuGets remain available for archival compatibility.

Read the polished notes on [4.148.0](4.148.0.md) for the definitive changelog including the accompanying breaking-change matrix; this page keeps the preview-only history so the diff is auditable. [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md) is the paired native asset — refer to its page for shaping-side binding details.

## What was staged in the 4.147.0 previews

- **Major-version bump to 4.x** — the umbrella upgrade of the SkiaSharp version number and package identifiers ([#3640](https://github.com/mono/SkiaSharp/pull/3640)).
- **Skia milestone jumps** — m132 ([#3560](https://github.com/mono/SkiaSharp/pull/3560) ❤️), m133 ([#3660](https://github.com/mono/SkiaSharp/pull/3660)), and finally m147 ([#3702](https://github.com/mono/SkiaSharp/pull/3702) ❤️).
- **Variable font support** — both `SkiaSharp` and `HarfBuzzSharp` now expose the axes/variations API ([#3703](https://github.com/mono/SkiaSharp/pull/3703) ❤️), plus color-font palette support and additional robustness fixes ([#3742](https://github.com/mono/SkiaSharp/pull/3742)).
- **Animated WebP encoding** — `SKWebpEncoder` now supports frame-by-frame animated WebP output ([#3771](https://github.com/mono/SkiaSharp/pull/3771)).
- **`SKStream.GetData()`** — zero-copy stream-to-`SKData` conversion ([#3772](https://github.com/mono/SkiaSharp/pull/3772)).
- **`SKSamplingOptions` on `SKSurface.Draw` and `SKCanvas.DrawSurface`** — proper sampling control when drawing surfaces ([#3491](https://github.com/mono/SkiaSharp/pull/3491) ❤️).
- **`HBColor` struct** — proper managed representation of `hb_color_t` ([#4000](https://github.com/mono/SkiaSharp/pull/4000)), plus additional API wrappers and naming-convention fixes ([#4001](https://github.com/mono/SkiaSharp/pull/4001)).
- **Default typeface resolution moved to managed layer** — makes the default-typeface behaviour consistent across native platforms and unlocks the 4.148.0 typeface changes ([#3730](https://github.com/mono/SkiaSharp/pull/3730) ❤️).
- **Finalizer crash in `SKPath` when `SKPathBuilder` is collected first** — fixed ([#3796](https://github.com/mono/SkiaSharp/pull/3796) ❤️).
- **Automated website release-notes pipeline** — the AI-powered release-notes generator was introduced, iterated, and locked in ([#3763](https://github.com/mono/SkiaSharp/pull/3763), [#3787](https://github.com/mono/SkiaSharp/pull/3787), [#3815](https://github.com/mono/SkiaSharp/pull/3815), [#3877](https://github.com/mono/SkiaSharp/pull/3877), [#4036](https://github.com/mono/SkiaSharp/pull/4036), [#4038](https://github.com/mono/SkiaSharp/pull/4038)).
- **Native dependency updates** — `libpng` 1.6.58 ([#3718](https://github.com/mono/SkiaSharp/pull/3718)), `libexpat` 2.7.5 ([#3717](https://github.com/mono/SkiaSharp/pull/3717)), `zlib` 1.3.2.1-motley ([#3720](https://github.com/mono/SkiaSharp/pull/3720)), `freetype` 2.14.3 ([#3726](https://github.com/mono/SkiaSharp/pull/3726)), and `libjpeg-turbo` 3.1.4.1 ([#4012](https://github.com/mono/SkiaSharp/pull/4012)).
- **Gallery redesign** — new API Atlas with tag-cloud filtering ([#3970](https://github.com/mono/SkiaSharp/pull/3970)) and a Hero Image sample ([#3792](https://github.com/mono/SkiaSharp/pull/3792)); Uno Platform gallery ([#3758](https://github.com/mono/SkiaSharp/pull/3758) ❤️) with SkiaFiddle WASM sample ([#3762](https://github.com/mono/SkiaSharp/pull/3762) ❤️).
- **Apple platform TFMs** — libraries target `26.0`, apps stay unversioned ([#3798](https://github.com/mono/SkiaSharp/pull/3798)).
- **WASM housekeeping** — dropped native builds for pre-.NET 8 Emscripten ([#4022](https://github.com/mono/SkiaSharp/pull/4022)); SkiaFiddle WASM app is ~60% smaller ([#3849](https://github.com/mono/SkiaSharp/pull/3849) ❤️).
- **AI automation infrastructure** — issue-triage, auto-triage, skia-analyst, sample-scout, gh-aw agent config, and skills reorg (moved to `.agents/`) all landed in this preview line.

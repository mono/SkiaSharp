# Version 4.147.0

> **Preview only** · Superseded by 4.148.0 · Never released as stable — these changes rolled up into 4.148.0 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview)

> **API changes** · [SkiaSharp API diff](4.147.0/index.md) · [HarfBuzzSharp 8.3.1.6](harfbuzzsharp/8.3.1.6.md)

## Highlights

The 4.x generation of SkiaSharp arrived in this preview cycle — a major version bump that brings Skia milestone 147, full variable font and color font support, animated WebP encoding, zero-copy stream-to-data conversion, and `SKSamplingOptions` on surfaces and canvases. These changes were never released as a stable 4.147.0 and rolled up entirely into [4.148.0](4.148.0.md). Community contributors [@ramezgerges](https://github.com/ramezgerges), [@Copilot](https://github.com/Copilot), [@SimonvBez](https://github.com/SimonvBez), [@4Darmygeometry](https://github.com/4Darmygeometry), and [@ebariche](https://github.com/ebariche) drove major portions of the work. HarfBuzzSharp 8.3.1.6 ships alongside — see its [release page](harfbuzzsharp/8.3.1.6.md) for binding details.

## Breaking Changes

- **SkiaSharp major version: 3.x → 4.x** — The library version was bumped from `3.x` to `4.x`. Deprecated APIs carry updated `[Obsolete]` attributes. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

## New Features

### Engine

- **Skia milestone 147** — The Skia engine steps through milestones 132 and 133 before landing at m147, bringing Chromium graphics engine improvements across all three. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560)), ([#3660](https://github.com/mono/SkiaSharp/pull/3660)), ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))

### Text & Fonts

- **Variable font support** — Full variable font axis querying and selection via `SKFontManager`, `SKTypeface`, and `SKFont`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — `SKTypeface` exposes font palette data for COLR/CPAL color fonts, improving emoji and color font rendering. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Default-typeface resolution fix** — Default typeface resolution moved to the managed layer, fixing cross-platform font fallback behavior. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))
- **`HBColor` struct** — Adds `HBColor` for proper `hb_color_t` representation in HarfBuzzSharp. ([#4000](https://github.com/mono/SkiaSharp/pull/4000))

### API Surface

- **Animated WebP encoding** — New `SKWebpEncoder` supports encoding animated WebP sequences. ([#3771](https://github.com/mono/SkiaSharp/pull/3771))
- **`SKStream.GetData()`** — Zero-copy stream-to-`SKData` conversion. ([#3772](https://github.com/mono/SkiaSharp/pull/3772))
- **`SKSamplingOptions` on `SKSurface` and `SKCanvas`** — `SKSurface.Draw` and `SKCanvas.DrawSurface` now accept `SKSamplingOptions` for high-quality scaling control. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))
- **Missing API wrappers** — Added several missing wrappers and fixed naming convention violations. ([#4001](https://github.com/mono/SkiaSharp/pull/4001))

### GPU & Rendering

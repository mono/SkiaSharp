# Version 4.147.0

> **First 4.x release — Skia m147, variable fonts, and new platforms** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1.1)

## Highlights

SkiaSharp 4.147.0 is a landmark release — the first in the 4.x series. It leaps from Skia milestone 119 to milestone 147, unlocking years of upstream improvements. Variable font support and color font palettes land as first-class APIs, thanks to ❤️ [@ramezgerges](https://github.com/ramezgerges). New platform support for Tizen (x64/arm64) and Android Bionic arrives with contributions from ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry), and a long-standing Android rendering bug is fixed by ❤️ [@SimonvBez](https://github.com/SimonvBez).

## Breaking Changes

- **Major version bump from 3.x to 4.x** — SkiaSharp now ships as version 4.x, reflecting the significant Skia engine update and new API surface. Applications upgrading from 3.x should update their package references accordingly. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

## New Features

### Engine

- **Skia milestone 147** — The underlying Skia engine jumps from milestone 119 through 132, 133, and up to milestone 147, bringing years of upstream performance improvements, bug fixes, and new capabilities. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560), [#3660](https://github.com/mono/SkiaSharp/pull/3660), [#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **libexpat updated to 2.7.5** — Keeps the XML parser dependency current. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))
- **libpng updated to 1.6.58** — Keeps the PNG codec dependency current. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))

### Text & Fonts

- **Variable font support** — Full support for OpenType variable fonts, including font variation axes and named instances. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Adds color font palette APIs and improves variable font robustness for complex font scenarios. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Managed default-typeface resolution** — Moves default-typeface resolution from native to the managed layer for more predictable behavior. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### API Surface

- **SKSamplingOptions for surface drawing** — Adds `SKSamplingOptions` support to `SKSurface.Draw` and `SKCanvas.DrawSurface`, enabling explicit control over image sampling quality. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))
- **PolySharp for C# 13 on legacy TFMs** — Enables C# 13 language features on older target frameworks via PolySharp. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

### Platform

- **Tizen x64 and arm64 native builds** — Adds native build support for Tizen on both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **Android Bionic Library build support** — Enables building SkiaSharp against Android's Bionic C library directly. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))

## Bug Fixes

- **Fixed SKGLView not rendering with TabBar on Android** — Resolves a long-standing issue where `SKGLView` would not render when placed inside a `TabBar` on Android. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Fixed x86 .NET Framework threading test OOM failures** — Resolves out-of-memory failures in threading tests on x86 .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))

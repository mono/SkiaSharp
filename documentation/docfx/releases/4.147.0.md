# Version 4.147.0

> **Major release · Skia milestone 147 · Variable & color fonts · New platforms** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1)

## Highlights

SkiaSharp 4.0 is a landmark release: the major version bump from 3.x to 4.x comes alongside an upgrade to **Skia milestone 147**, full **variable font** and **color font palette** support, new native builds for **Linux Bionic** and **Tizen ARM64/x64**, and security updates for libpng and libexpat. Much of this release was community-driven — special thanks to [@ramezgerges](https://github.com/ramezgerges) for the Skia milestone 147 bump, variable font support, and Uno Platform WebAssembly work, and to [@4Darmygeometry](https://github.com/4Darmygeometry) for Linux Bionic and C# 13 support across legacy target frameworks.

## ⚠️ Breaking Changes

- **Major version bump: 3.x → 4.x** — SkiaSharp has moved to version 4.x. Update package references and review any version-pinned dependencies. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

## New Features

### Engine

- **Skia milestone 147** — Upgrades the underlying Skia engine to milestone 147, bringing the latest rendering improvements, GPU backend updates, and API additions from upstream Skia. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **Skia milestone 133** — Intermediate engine update included in the 4.0 journey. ([#3660](https://github.com/mono/SkiaSharp/pull/3660))
- **Skia milestone 132** — Foundation engine update from the community. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3560](https://github.com/mono/SkiaSharp/pull/3560))

### Text & Fonts

- **Variable font support** — Adds full variable font support to SkiaSharp and HarfBuzzSharp, enabling per-axis font variation (weight, width, slant, and more) for rich typography. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Adds `SKTypeface` palette APIs for color font rendering, with improved robustness for variable color fonts. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Default typeface resolution moved to managed layer** — Resolves cross-platform inconsistencies in default typeface selection by moving the logic to managed code. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### API Surface

- **`SKSamplingOptions` for `SKSurface.Draw` and `SKCanvas.DrawSurface`** — Adds `SKSamplingOptions` overloads to `SKSurface.Draw` and `SKCanvas.DrawSurface` for fine-grained texture filtering control. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Linux Bionic native assets** — Adds native library support for Linux Bionic (Android-kernel Linux), enabling SkiaSharp on glibc-free Linux environments. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and ARM64 native builds** — Adds full native build and CI support for Tizen on x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **C# 13 support on legacy TFMs** — Adds PolySharp to enable C# 13 language features across legacy target frameworks (.NET Framework, .NET Standard). ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Security

- **libpng updated to 1.6.58** — Picks up upstream security fixes in libpng. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))
- **libexpat updated to 2.7.5** — Picks up upstream security fixes in libexpat. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))

## Bug Fixes

- **Android `SKGLView` rendering after MAUI tab switch** — Fixes `SKGLView` not rendering when switching back to a tab in a MAUI `TabBar` on Android. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **x86 .NET Framework threading test OOM** — Fixes intermittent out-of-memory failures in threading tests on x86 .NET Framework targets. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))

## Gallery & Samples

- **Uno Platform WebAssembly SkiaFiddle sample** — Adds the SkiaFiddle interactive code editor as a standalone Uno Platform WebAssembly sample. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3762](https://github.com/mono/SkiaSharp/pull/3762))
- **Uno Platform WebAssembly gallery** — Adds the full multi-page SkiaSharp gallery as an Uno Platform WebAssembly app. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3758](https://github.com/mono/SkiaSharp/pull/3758), [#3666](https://github.com/mono/SkiaSharp/pull/3666))
- **FillPath and TextToPath gallery samples** — New gallery demos for `FillPath` and `TextToPath` operations. ([#3761](https://github.com/mono/SkiaSharp/pull/3761))
- **Gallery WASM text rendering and download button** — Fixes text rendering in the WebAssembly gallery and adds a sample-download button. ([#3760](https://github.com/mono/SkiaSharp/pull/3760))

Plus several CI and documentation improvements.

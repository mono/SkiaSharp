# Version 4.147.0

> **First major version — Skia m147, variable fonts, and new platform targets** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1.1)

## Highlights

SkiaSharp 4.147.0 is the first major version release since 3.x, powered by Skia milestone 147. This release brings variable font support and color font palettes, new `SKSamplingOptions` overloads for surface and canvas drawing, Android Bionic library builds, and Tizen platform targets. Community contributors ❤️ [@ramezgerges](https://github.com/ramezgerges), [@4Darmygeometry](https://github.com/4Darmygeometry), [@SimonvBez](https://github.com/SimonvBez), and [@Copilot](https://github.com/Copilot) drove many of the headline features.

## Breaking Changes

This is a major version bump (3.x → 4.x). The Skia engine has been updated through milestones 132, 133, and 147. While the C# API surface remains backward-compatible, subtle rendering differences may occur due to the engine upgrade.

## New Features

### Engine

- **Bump Skia to milestone 147** — Brings the Skia engine up to Chrome m147 with performance and rendering improvements. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702))
- **Upgrade major version from 3.x to 4.x** — Version scheme now encodes the Skia milestone: 4.147.0 = SkiaSharp 4, Skia m147. ([#3640](https://github.com/mono/SkiaSharp/pull/3640))

### Text & Fonts

- **Variable font support** — Full OpenType variable font axis support, letting you control weight, width, slant, and custom axes at runtime. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Select and apply COLR/CPAL color palettes for emoji and color fonts, with improved variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))

### API Surface

- **`SKSamplingOptions` for `SKSurface.Draw` and `SKCanvas.DrawSurface`** — New overloads accept `SKSamplingOptions` to control filtering when drawing surfaces, replacing the default nearest-neighbor behavior. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Android Bionic Library build support** — Enables building native libraries directly against Android's Bionic libc for tighter platform integration. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Adds Tizen as a first-class native build target for both Intel and ARM architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **PolySharp for C# 13 on legacy TFMs** — Polyfills modern C# 13 language features for .NET Standard 2.0 and .NET Framework consumers. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Bug Fixes

- **Fixed `SKGLView` on Android not rendering with TabBar** — Resolves an issue where `SKGLView` would not display when hosted inside a `TabBar` layout on Android. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **Fixed default-typeface resolution** — Moves default typeface resolution to the managed layer, improving reliability across platforms. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))
- **Fixed x86 .NET Framework threading OOM failures** — Resolves out-of-memory crashes in multi-threaded scenarios on 32-bit .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))

## Dependencies

- **libexpat → 2.7.5** — Security and bug fix update. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))
- **libpng → 1.6.58** — Security and bug fix update. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))

# Version 4.147.0

> **Major version release — SkiaSharp 4.x** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.147.0-preview.1.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.147.0-preview.1.1)

## Highlights

SkiaSharp 4.x is here — a major version jump powered by Skia milestone 147. This release introduces **variable font support** and **color font palettes** contributed by [@ramezgerges](https://github.com/ramezgerges), brings **Android Bionic Library** and **C# 13 PolySharp** support from [@4Darmygeometry](https://github.com/4Darmygeometry), and adds new `SKSamplingOptions` API surface. The underlying Skia engine has been updated through milestones 132, 133, and 147 — a massive leap in rendering capabilities.

## Breaking Changes

This is a major version bump from 3.x to 4.x. See the [upgrade guide](https://github.com/mono/SkiaSharp/pull/3640) for migration details.

## New Features

### Engine

- **Skia milestone 147** — Updates the Skia graphics engine through milestones 132 → 133 → 147, bringing the latest rendering improvements from Google Chrome. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3702](https://github.com/mono/SkiaSharp/pull/3702), [#3560](https://github.com/mono/SkiaSharp/pull/3560), [#3660](https://github.com/mono/SkiaSharp/pull/3660))

### Text & Fonts

- **Variable font support** — Full support for OpenType variable fonts, including axis enumeration, named instances, and coordinate control. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3703](https://github.com/mono/SkiaSharp/pull/3703))
- **Color font palette support** — Adds COLR/CPAL color font palette APIs with improved variable font robustness. ([#3742](https://github.com/mono/SkiaSharp/pull/3742))
- **Default typeface resolution in managed layer** — Moves default-typeface resolution out of native code for better cross-platform consistency. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3730](https://github.com/mono/SkiaSharp/pull/3730))

### API Surface

- **`SKSamplingOptions` for surface drawing** — Adds `SKSamplingOptions` parameter to `SKSurface.Draw` and `SKCanvas.DrawSurface` for explicit filtering control. ❤️ [@Copilot](https://github.com/Copilot) ([#3491](https://github.com/mono/SkiaSharp/pull/3491))

### Platform

- **Android Bionic Library build support** — Enables building against Android's Bionic C library for tighter platform integration. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3217](https://github.com/mono/SkiaSharp/pull/3217))
- **Tizen x64 and arm64 native builds** — Adds Tizen support for both x64 and arm64 architectures. ([#3620](https://github.com/mono/SkiaSharp/pull/3620))
- **PolySharp for C# 13 on legacy TFMs** — Enables modern C# 13 features on older target frameworks via PolySharp. ❤️ [@4Darmygeometry](https://github.com/4Darmygeometry) ([#3642](https://github.com/mono/SkiaSharp/pull/3642))

## Bug Fixes

- **SKGLView on Android with TabBar** — Fixed rendering issue where `SKGLView` was not rendered when used inside a `TabBar`. ❤️ [@SimonvBez](https://github.com/SimonvBez) ([#3076](https://github.com/mono/SkiaSharp/pull/3076))
- **x86 .NET Framework threading OOM** — Fixed out-of-memory failures in threading tests on x86 .NET Framework. ([#3674](https://github.com/mono/SkiaSharp/pull/3674))
- **Linux Docker builds** — Fixed debian/13 Docker build for non-loong64/riscv64 architectures. ([#3747](https://github.com/mono/SkiaSharp/pull/3747))

## Security

- **libexpat updated to 2.7.5** — Addresses known vulnerabilities. ([#3717](https://github.com/mono/SkiaSharp/pull/3717))
- **libpng updated to 1.6.58** — Addresses known vulnerabilities. ([#3718](https://github.com/mono/SkiaSharp/pull/3718))

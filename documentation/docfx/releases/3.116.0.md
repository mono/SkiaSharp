# Version 3.116.0

> **First stable 3.x release** · Released December 3, 2024 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.116.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.116.0)

> **Supersedes [3.0.0](3.0.0.md)** · Rolls up preview-only work that was never released as stable — those changes are included cumulatively below.

> **API changes** · [SkiaSharp API diff](3.116.0/index.md) · [HarfBuzzSharp 8.3.0](harfbuzzsharp/8.3.0.md)

## Highlights

SkiaSharp 3.116.0 is the first stable 3.x release. It rolls up the entire 3.0.0 preview train, adds the last WebAssembly and Blazor fixes that landed after Preview 5, and brings the modernized 3.x API and platform stack to production.

## Breaking Changes

- **Obsolete 2.x APIs are gone in 3.x** — This stable line includes the preview-era removal of members and types that had already been marked obsolete, so upgrading from 2.88 may require source changes. ([#2539](https://github.com/mono/SkiaSharp/pull/2539))

## New Features

### Engine

- **Shipped the stable 3.x graphics foundation** — Builds on the Skia milestone 116 base, then layers on later external refreshes that landed before the stable tag. ([#2829](https://github.com/mono/SkiaSharp/pull/2829), [#2885](https://github.com/mono/SkiaSharp/pull/2885), ❤️ [@Redth](https://github.com/Redth) [#3026](https://github.com/mono/SkiaSharp/pull/3026))
- **Expanded the core binding surface** — Added `SKBlender`, more `SKPicture` APIs, Skottie's Animation Builder, `SKImage.ToRawShader`, `beginRecording` R-Tree overloads, and new span-based point and path helpers. ([#2830](https://github.com/mono/SkiaSharp/pull/2830), [#2883](https://github.com/mono/SkiaSharp/pull/2883), [#2630](https://github.com/mono/SkiaSharp/pull/2630), [#2748](https://github.com/mono/SkiaSharp/pull/2748), [#2889](https://github.com/mono/SkiaSharp/pull/2889), ❤️ [@alexandrvslv](https://github.com/alexandrvslv) [#3030](https://github.com/mono/SkiaSharp/pull/3030))

### Views & Platform

- **Introduced the new 3.x view stack** — Brings `SKGLView`, accelerated WinUI views, and `SKGLElement` for WPF into the stable line. ([#2598](https://github.com/mono/SkiaSharp/pull/2598), [#2733](https://github.com/mono/SkiaSharp/pull/2733), ❤️ [@gmurray81](https://github.com/gmurray81) [#2317](https://github.com/mono/SkiaSharp/pull/2317))
- **Modernized platform support** — Adds .NET 9 support, switches WinForms to the new GLControl packages, and adopts Metal on Mac Catalyst with common Metal APIs across TFMs. (❤️ [@jeromelaban](https://github.com/jeromelaban) [#3010](https://github.com/mono/SkiaSharp/pull/3010), [#2989](https://github.com/mono/SkiaSharp/pull/2989), [#2747](https://github.com/mono/SkiaSharp/pull/2747), ❤️ [@maxkatz6](https://github.com/maxkatz6) [#2788](https://github.com/mono/SkiaSharp/pull/2788))

### WebAssembly

- **Broadened browser support** — Adds support for multi-threaded and SIMD WebAssembly builds and folds the remaining Blazor and base-path fixes directly into SkiaSharp. ([#2620](https://github.com/mono/SkiaSharp/pull/2620), [#3064](https://github.com/mono/SkiaSharp/pull/3064), [#3082](https://github.com/mono/SkiaSharp/pull/3082), [#3088](https://github.com/mono/SkiaSharp/pull/3088), [#3092](https://github.com/mono/SkiaSharp/pull/3092))

### API Surface

- **Added migration-friendly compatibility helpers** — Includes `SKFilterQuality` compatibility overloads, color-filter spans, `GetPixelSpan` with offsets, and readonly Metal accessors that smooth migration from 2.x. (❤️ [@Youssef1313](https://github.com/Youssef1313) [#2949](https://github.com/mono/SkiaSharp/pull/2949) [#2963](https://github.com/mono/SkiaSharp/pull/2963), [#2879](https://github.com/mono/SkiaSharp/pull/2879), [#2609](https://github.com/mono/SkiaSharp/pull/2609), [#2833](https://github.com/mono/SkiaSharp/pull/2833))

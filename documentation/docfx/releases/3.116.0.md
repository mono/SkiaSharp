## Highlights

The first stable 3.x release. Rolls up all 122 PRs from the 3.0.0 preview stream (which was never released as stable) plus 3 more PRs from Preview 6 — 129 PRs total against the 2.88.9 baseline. This is a modernization release: text-based blobs come back, every `[Obsolete]` member and type from the 2.x line is removed, function pointers and `LibraryImportAttribute` replace older P/Invoke plumbing (contributed by [@maxkatz6](https://github.com/maxkatz6)), Metal becomes the Mac Catalyst backend, WinUI accelerated views land, `SKGLElement` for WPF is added ([@gmurray81](https://github.com/gmurray81)), and Skia walks forward from milestone 104 → 116 → 137. HarfBuzz is bumped from 7.3.0 to **8.3.0** (see the [HarfBuzzSharp 8.3.0](harfbuzzsharp/8.3.0.md) page for the binding-level detail), libwebp goes to 1.3.2, libjpeg-turbo to 3.0.0. New surface includes `SKImage.ToRawShader`, `SKBlender`, Skottie's Animation Builder, an R-Tree `BeginRecording` overload, `SKFilterQuality` compatibility overloads, and `Span<SKPoint>` overloads on `SKMatrix.MapPoints/MapVectors` and `SKPath.AddPoly` ([@alexandrvslv](https://github.com/alexandrvslv)). WebAssembly gets multi-thread and SIMD variants.

Community contributions from [@maxkatz6](https://github.com/maxkatz6), [@jeromelaban](https://github.com/jeromelaban), [@Youssef1313](https://github.com/Youssef1313), [@sungaila](https://github.com/sungaila), [@jeremy-visionaid](https://github.com/jeremy-visionaid), [@pdjonov](https://github.com/pdjonov), [@Fxplorer](https://github.com/Fxplorer), [@beto-rodriguez](https://github.com/beto-rodriguez), [@taublast](https://github.com/taublast), [@lindexi](https://github.com/lindexi), [@gmurray81](https://github.com/gmurray81), [@alexandrvslv](https://github.com/alexandrvslv), and [@Redth](https://github.com/Redth).

## Breaking Changes

- **All `[Obsolete]` members and types from the 2.x line are removed** ([#2539](https://github.com/mono/SkiaSharp/pull/2539)) — Any call site that had been living on a deprecation warning will now fail to compile. Migrate to the current entry point named in the compiler warning before upgrading. See the [SkiaSharp breaking-change list](3.116.0/SkiaSharp/SkiaSharp.breaking.md) for the exact members.
- **`SKFilterQuality` restored via compatibility overloads** ❤️ [@Youssef1313](https://github.com/Youssef1313) ([#2963](https://github.com/mono/SkiaSharp/pull/2963)) — After the sampling-options migration, obsolete methods honor `SKPaint.FilterQuality` when producing `SKSamplingOptions` ([#2924](https://github.com/mono/SkiaSharp/pull/2924)).
- **Text-based `SKTextBlob` constructors are back** ([#2545](https://github.com/mono/SkiaSharp/pull/2545)) — Restores the ergonomic text-first blob API the 2.x line had lost.
- **View-assembly signature shifts** — Assorted per-platform view signature changes accompany the WinUI accelerated view, `SKGLView`, and MAUI leak fixes. See the per-assembly `*.breaking.md` files linked at the top for the exact members.

See the [full API diff](3.116.0/index.md) for the complete list.

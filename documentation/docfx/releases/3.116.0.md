## Highlights

SkiaSharp **3.116.0** is the first stable 3.x release, superseding the preview-only [3.0.0](3.0.0.md). The 3.x line adopts a Chromium-milestone-aligned version scheme (3.116.x tracks chrome/m116) and lands the full modernization pass: .NET 8 baseline, function-pointer P/Invoke, removed `[Obsolete]` APIs, new WASM SIMD/multi-thread native builds, Metal on Mac Catalyst, WinUI accelerated views, WPF `SKGLElement`, new Skottie `Animation.Builder`, `SKBlender`, `SKImage.ToRawShader`, R-Tree recording overloads, `SKPictur` API additions, span overloads across color filters and geometry APIs, .NET 9 workload support, and HarfBuzz 8.3.0 uptake.

## Breaking Changes

This release includes breaking API signature changes across `SkiaSharp`, `SkiaSharp.Views.*`, `SkiaSharp.Views.Blazor`, `SkiaSharp.Views.Maui.Controls`, `SkiaSharp.Views.Uno.WinUI`, `SkiaSharp.Views.WinUI`, and `SkiaSharp.Views.WindowsForms`. See the [SkiaSharp API diff](3.116.0/index.md) for the full per-assembly breakdown before upgrading.

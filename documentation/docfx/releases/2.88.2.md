# Version 2.88.2

> **Platform fixes release** · Released · [NuGet](https://www.nuget.org/packages/SkiaSharp/2.88.2) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v2.88.2)

## Highlights

This release corrects Apple coordinate conversion and brings two community-driven platform fixes for Uno Platform and Tizen. Thanks to [@jerome.laban](https://github.com/jerome.laban) and [@sngn.lee](https://github.com/sngn.lee) for their contributions.

## New Features

### Platform

- **Tizen graphics backend** — Changes the Tizen graphics backend engine for improved rendering on the platform. ❤️ [@sngn.lee](https://github.com/sngn.lee) ([#2225](https://github.com/mono/SkiaSharp/pull/2225))

## Bug Fixes

- **Correct `SKRect` to `CGRect` conversion** — Fixes incorrect coordinate conversion when interoperating with Apple's `CGRect`. ([#2243](https://github.com/mono/SkiaSharp/pull/2243))
- **Removed unused native library block in Uno WinUI** — Drops a stale native library reference from `SkiaSharp.Views.Uno.WinUI`. ❤️ [@jerome.laban](https://github.com/jerome.laban) ([#2231](https://github.com/mono/SkiaSharp/pull/2231))

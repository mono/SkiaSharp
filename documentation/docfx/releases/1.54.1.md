# Version 1.54.1

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.54.1)

> **API changes** · [SkiaSharp API diff](1.54.1/index.md)

## Highlights

Historical maintenance release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so no pull-request history was recorded for the branch itself; signature-level differences captured from the prior tag are surfaced through the API diff linked above.

## Breaking Changes

- **`SKMatrix` fields to properties** — The nine public mutable fields on `SKMatrix` (`ScaleX`, `ScaleY`, `SkewX`, `SkewY`, `TransX`, `TransY`, `Persp0`, `Persp1`, `Persp2`) were converted to properties. Code that took `ref` on those fields needs to move to property setters.
- **`SKObject` disposal moved to the base type** — `Dispose()` and finalizers on `SKObject`, `SKPath.Iterator`, and `SKPath.RawIterator` were removed and now come from the shared base implementation; override the standard `Dispose(bool)` pattern instead.

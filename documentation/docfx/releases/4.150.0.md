# Version 4.150.0

> **Skia m150 engine, API cleanup, and HarfBuzz 14.2.1** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0-preview.2.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.150.0-preview.2.1)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

SkiaSharp 4.150.0 delivers the Skia m150 engine update, ships HarfBuzzSharp 14.2.1, and completes the v4 API-cleanup work: remaining pre-v4 obsolete APIs are promoted to compile errors and the long-deprecated .NET Interactive (Polyglot Notebooks) integration is removed. Community contributor [@ramezgerges](https://github.com/ramezgerges) drove the API-surface hardening.

## Breaking Changes

- **Pre-v4 obsolete APIs promoted to compile errors** — APIs that were marked `[Obsolete(error: false)]` during the v4 transition now produce compile errors. Update call sites to their v4 equivalents before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive / Polyglot Notebooks support** — The long-deprecated SkiaSharp integration for .NET Interactive (Polyglot Notebooks) is no longer included. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **Skia m150 engine update** — Updates the bundled Skia native library to milestone m150, bringing the latest rendering improvements and stability fixes from the upstream Chrome project. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

## Platform

- **Tizen native dependencies in Docker** — Tizen native dependency builds are now containerized for reproducibility and simpler CI maintenance. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

# Version 4.150.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0-preview)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

SkiaSharp 4.150.0 updates the Skia engine to milestone m150, ships HarfBuzzSharp 14.2.1, and completes the v4 API migration by escalating pre-v4 obsoletes to compile errors. Community contributor [@ramezgerges](https://github.com/ramezgerges) drove the API cleanup, and Matthew Leibowitz containerized Tizen native dependency builds. .NET Interactive (Polyglot Notebooks) support has been removed.

## Breaking Changes

- **Pre-v4 API obsoletes are now compile errors** — All APIs marked obsolete during the v3→v4 migration now fail to compile; callers must migrate to v4 equivalents before upgrading. Obsolete pre-v4 enum values have also been trimmed from the reference assembly. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated `.NET Interactive` integration has been removed. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **Updated to Skia milestone m150** — The underlying Skia engine has been updated to Chrome milestone m150, bringing all upstream bug fixes and optimizations. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

### Platform

- **HarfBuzzSharp 14.2.1** — Updated the bundled HarfBuzz library to version 14.2.1. See the [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) release page for details. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))
- **Tizen native builds containerized** — Tizen native dependencies are now built inside a Docker container, improving build reproducibility and isolation. ❤️ Matthew Leibowitz ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

# Version 4.150.0

> **Skia m150 and API modernization** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0-preview.2.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.150.0-preview.2.1)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

SkiaSharp 4.150.0 updates the Skia engine to Chrome milestone m150 and co-ships HarfBuzzSharp 14.2.1 — see the [HarfBuzzSharp 14.2.1 release page](harfbuzzsharp/14.2.1.md) for details. This release completes the v4 API modernization: [@ramezgerges](https://github.com/ramezgerges) promoted all remaining pre-v4 obsoletes to compile errors and trimmed obsolete enums from the reference assembly. Deprecated .NET Interactive (Polyglot Notebooks) support has been removed, and Tizen native dependency builds now run in a Docker container for improved reproducibility.

## ⚠️ Breaking Changes

- **Pre-v4 obsoletes promoted to errors** — All APIs marked `[Obsolete]` as part of the v4 migration are now compile errors. Update any remaining call sites to use their v4 equivalents before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated .NET Interactive / Polyglot Notebooks integration has been removed. Projects using this integration should remove the dependency. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **Skia updated to Chrome milestone m150** — The underlying Skia graphics engine has been updated to milestone m150, incorporating the latest upstream performance improvements, bug fixes, and rendering enhancements from the Chrome graphics team. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

### Dependencies

- **HarfBuzzSharp updated to 14.2.1** — The bundled HarfBuzz text-shaping library has been updated to version 14.2.1. See the [HarfBuzzSharp 14.2.1 release page](harfbuzzsharp/14.2.1.md) for HarfBuzz-specific details. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native dependencies built in Docker** — The Tizen native dependency build now runs inside a Docker container, making Tizen builds more reproducible and easier to maintain across environments. ❤️ Matthew Leibowitz ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

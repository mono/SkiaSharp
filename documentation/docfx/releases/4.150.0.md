# Version 4.150.0

> **Skia m150 engine update** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0-preview)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

This preview series updates the underlying Skia engine to milestone m150 and ships HarfBuzzSharp 14.2.1. ❤️ [@ramezgerges](https://github.com/ramezgerges) drives a significant API-surface cleanup: the remaining pre-v4 deprecated APIs are promoted from warnings to compile errors, and unused obsolete enum members are trimmed from the reference assembly. Deprecated .NET Interactive (Polyglot Notebooks) support is also removed in this cycle, and Tizen native builds now run inside Docker for a more reproducible build environment.

## Breaking Changes

- **Pre-v4 obsoletes promoted to compile errors** — APIs deprecated as part of the SkiaSharp v4 migration that were previously `[Obsolete]` warnings are now compile errors. Migrate to the v4 equivalents before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated SkiaSharp integration for .NET Interactive is removed. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **Skia milestone m150** — Updates the underlying Skia native engine to the chrome/m150 milestone, bringing upstream performance improvements and bug fixes. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))
- **HarfBuzz 14.2.1** — Updates the bundled HarfBuzz text-shaping library to version 14.2.1. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native builds in Docker** — Tizen native dependencies are now built inside a Docker container for a more reproducible and isolated build environment. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

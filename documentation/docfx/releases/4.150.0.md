# Version 4.150.0

> **Skia m150 engine, new filter APIs, and HarfBuzz 14.2.1** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0-preview.2.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.150.0-preview.2.1)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

This preview updates the underlying Skia engine to milestone m150 and ships three new image-filter APIs: `SKColorFilter.CreateOverdraw()`, `SKImageFilter.CreateCrop()`, and `SKImageFilter.CreateEmpty()`. HarfBuzz 14.2.1 ships alongside — see its [dedicated page](harfbuzzsharp/14.2.1.md) for binding details. ❤️ [@ramezgerges](https://github.com/ramezgerges) cleaned up pre-v4 obsolete API remnants, and Matthew Leibowitz moved Tizen native dependency builds into Docker containers. Deprecated .NET Interactive (Polyglot Notebooks) support has been removed in this release.

## Breaking Changes

- **Pre-v4 obsoletes are now compile errors** — Remaining APIs that were marked `[Obsolete(error: true)]` during the v4 migration now cause build failures. Projects that suppressed or ignored those warnings must migrate to the v4 equivalents before updating. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated SkiaSharp.ScottPlot and related Notebook packages have been removed. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **Skia milestone m150** — Updates the underlying Skia graphics engine to the chrome/m150 release, bringing upstream bug fixes and improvements. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

### API Surface

- **`SKColorFilter.CreateOverdraw()`** — New color filter for overdraw visualization. Maps pixel overdraw counts to configurable colors, making over-drawn regions immediately visible during performance profiling. ([#4227](https://github.com/mono/SkiaSharp/pull/4227))
- **`SKImageFilter.CreateCrop()`** — New image filter that clips the output of another filter to a specified rectangle with no other transformation. ([#4229](https://github.com/mono/SkiaSharp/pull/4229))
- **`SKImageFilter.CreateEmpty()`** — New image filter that produces a fully transparent output, useful as a terminal or placeholder node in filter graphs. ([#4228](https://github.com/mono/SkiaSharp/pull/4228))

### Platform

- **Tizen native dependencies in Docker** — Tizen native library builds now run inside Docker containers, improving build isolation and reproducibility. ❤️ Matthew Leibowitz ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

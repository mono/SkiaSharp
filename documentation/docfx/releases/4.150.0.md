# Version 4.150.0

> **Preview release** · Preview only · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0-preview)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

SkiaSharp 4.150.0 updates the engine to Skia milestone m150 and ships a significant API modernization pass: remaining pre-v4 deprecated members are promoted to compile errors, and the .NET Interactive (Polyglot Notebooks) integration is removed. HarfBuzz 14.2.1 ships alongside this release — see the [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) page for binding details. Community contributors [@ramezgerges](https://github.com/ramezgerges) and Matthew Leibowitz each drove notable changes in this preview.

## Breaking Changes

- **Pre-v4 obsoletes promoted to compile errors** — All APIs that were emitting deprecation warnings for the v4 migration have been upgraded to compile-time errors. Review the API diff for the full list of affected members and migrate away from them before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The specialized Polyglot Notebooks rendering integration has been removed. Notebook users should render via `SKBitmap` + `SKCanvas` and display images through the notebook host's standard output path. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **Skia milestone m150** — Updates the underlying Skia graphics engine to milestone m150, bringing the full set of upstream rendering improvements, GPU backend fixes, and shader changes from Chrome m150. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

### API Surface

- **Obsolete enum cleanup** — Trims remaining pre-v4 obsolete enum values from the reference assembly, producing a cleaner public API surface and smaller API diffs going forward. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))

# Version 4.150.0

> **Feature release — m150 upgrade** · Released July 7, 2026 · [NuGet](https://www.nuget.org/packages/SkiaSharp/4.150.0) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v4.150.0)

> **API changes** · [SkiaSharp API diff](4.150.0/index.md) · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

First feature release on the v4 line. Skia moves to milestone 150, several
image-filter/color-filter primitives that used to be Skia-internal become
public API (`SKPaint.GetFastBounds`, `SKColorFilter.CreateOverdraw`,
`SKImageFilter.CreateCrop`, `SKImageFilter.CreateEmpty`), the remaining pre-v4
`[Obsolete]` members are promoted to error, and the small-sigma raster blur
regression from the m147 series is fixed. Ships with
[HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md).

## Breaking Changes

The remaining pre-v4 `[Obsolete]` members are promoted from warning to error
and the obsolete enums are trimmed from the reference assembly. Consumers still
sitting on 4.147 or earlier obsoletes must migrate now — see the
[4.150.0 API diff](4.150.0/index.md) for the exact deletion set.

## Native Engine

- **Skia bumped to milestone 150** — the shipped Skia engine moves to Chrome's
  m150 branch, plus a rollup of upstream m150 bug fixes.
  ([#4146](https://github.com/mono/SkiaSharp/pull/4146),
  [#4190](https://github.com/mono/SkiaSharp/pull/4190),
  [mono/skia#253](https://github.com/mono/skia/pull/253),
  [mono/skia#259](https://github.com/mono/skia/pull/259))
- **HarfBuzz native bumped to 14.2.1** — matched to m150.
  ([#4167](https://github.com/mono/SkiaSharp/pull/4167),
  [mono/skia#257](https://github.com/mono/skia/pull/257))

## API Surface

- **`SKPaint.GetFastBounds()`** — computes a fast, conservative bound for a
  paint's draw effects so callers can quick-reject culling without a full
  bounds pass. ([#4271](https://github.com/mono/SkiaSharp/pull/4271),
  [mono/skia#271](https://github.com/mono/skia/pull/271))
- **`SKColorFilter.CreateOverdraw()`** — exposes Skia's overdraw color
  filter for visualizing composition cost.
  ([#4227](https://github.com/mono/SkiaSharp/pull/4227),
  [mono/skia#262](https://github.com/mono/skia/pull/262))
- **`SKImageFilter.CreateCrop()` and `SKImageFilter.CreateEmpty()`** — two
  filter primitives are added: `CreateCrop` crops an input to a rect, and
  `CreateEmpty` returns a no-op filter for placeholder use.
  ([#4229](https://github.com/mono/SkiaSharp/pull/4229),
  [#4228](https://github.com/mono/SkiaSharp/pull/4228),
  [mono/skia#261](https://github.com/mono/skia/pull/261),
  [mono/skia#260](https://github.com/mono/skia/pull/260))
- **Pre-v4 `[Obsolete]` promoted to error and trimmed from the ref assembly** —
  the last of the pre-v4 obsoletes are now compile-time errors and the
  obsolete enums no longer appear in the reference assembly. ❤️
  [@ramezgerges](https://github.com/ramezgerges)
  ([#4195](https://github.com/mono/SkiaSharp/pull/4195))

## Bug Fixes

- **Small-sigma raster blur regression fixed** — the raster blur regression
  from m147 (blurs with very small sigma) is fixed by defining
  `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS`. ❤️
  [@ramezgerges](https://github.com/ramezgerges)
  ([#4269](https://github.com/mono/SkiaSharp/pull/4269))

## Performance

- **`SKSurface.Canvas` caches the wrapper** — the managed `Canvas` wrapper on
  `SKSurface` is now cached instead of re-wrapping the native pointer on every
  access, cutting P/Invoke overhead in hot draw loops. ❤️
  [@ramezgerges](https://github.com/ramezgerges)
  ([#4247](https://github.com/mono/SkiaSharp/pull/4247))

## Lifecycle & Internals

- **Deprecated .NET Interactive / Polyglot Notebooks integration removed** —
  the deprecated Polyglot Notebooks support is dropped. Notebook consumers
  should switch to the current `SkiaSharp.Views` packages directly.
  ([#4186](https://github.com/mono/SkiaSharp/pull/4186))
- **Test suite migrated from xUnit v2 to v3** — the test suite runs on
  xUnit v3, which is what SkiaSharp 4.x will require for its own tests going
  forward. ([#4143](https://github.com/mono/SkiaSharp/pull/4143))

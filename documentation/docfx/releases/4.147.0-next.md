# Version 4.147.0 (Next Preview)

## Highlights

Fixes the Uno Platform sample gallery build after the 4.147.0 version bump. ❤️ [@ramezgerges](https://github.com/ramezgerges) contributed the fix.

## Breaking Changes

*None in this release.*

## Bug Fixes

### Samples

- **Fixed Skottie package downgrade in Uno gallery** — After bumping to 4.147.0, the generated Uno gallery failed restore with NU1605 due to a transitive Skottie version conflict. The fix pins Skottie in the Uno gallery csproj and rewrites obsolete `SKPath` calls to use `SKPathBuilder`, resolving both the restore error and CS0618 warnings under `TreatWarningsAsErrors`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#3785](https://github.com/mono/SkiaSharp/pull/3785))

Plus several CI and version-bump improvements.

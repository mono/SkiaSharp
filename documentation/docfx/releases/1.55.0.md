# Version 1.55.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.55.0)

> **API changes** · [SkiaSharp API diff](1.55.0/index.md)

## Highlights

Historical release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so the pull-request history for the branch is empty; the signature changes captured from the previous tag are the interesting part and are surfaced through the API diff linked above.

## Breaking Changes

- **Default parameter values dropped on public APIs** — Enum defaults on `SKCanvas.ClipPath`/`ClipRect`, `SKCanvas.DrawColor`, `GRContext.Flush`, and several `SKPath` methods were removed. Callers that relied on the implicit defaults now need to pass the value explicitly, which stabilises the ABI going forward.
- **`SKCodecOptions.Subset` becomes nullable** — `Subset` is now `SKRectI?` (read-only), with `HasSubset` demoted to read-only. Assign a subset via constructor instead of the previous set-then-flag pattern.
- **Removed helper methods** — `SKCodec.GetValidSubset(ref SKRectI)` and `SKDocument.Close()` were dropped in favour of the standard disposal pattern.

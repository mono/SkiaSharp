# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

This release completes the v4 API surface cleanup: community contributor [@ramezgerges](https://github.com/ramezgerges) promoted all remaining pre-v4 deprecated APIs to compile errors and removed obsolete enums from the reference assembly. The deprecated .NET Interactive (Polyglot Notebooks) integration has been removed. HarfBuzz is updated to 14.2.1; this release ships alongside **HarfBuzzSharp 14.2.0** — see the linked page above for binding-level details.

## ⚠️ Breaking Changes

- **Pre-v4 obsoletes now fail to compile** — APIs marked `[Obsolete]` since before v4 now emit errors instead of warnings; obsolete enums have also been removed from the reference assembly. Migrate any remaining call sites to their v4 equivalents. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The long-deprecated Polyglot Notebooks helper has been removed. Notebooks should use standard SkiaSharp rendering instead. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **HarfBuzz 14.2.1** — Updates the bundled HarfBuzz text-shaping library to 14.2.1. API-level changes in the HarfBuzzSharp bindings are documented on its linked page above. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native builds containerized** — Tizen native dependencies are now compiled inside a Docker container, improving build reproducibility and environment isolation for Tizen targets. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))
- **.NET 10 integration test support** — The integration test harness has been updated to run on .NET 10, ensuring forward compatibility with the upcoming runtime. ([#4221](https://github.com/mono/SkiaSharp/pull/4221))

### Build & Test Infrastructure

- **xUnit v3 migration** — The test suite has been migrated from xUnit v2 to xUnit v3, bringing improved test isolation, parallelism, and output formatting. ([#4143](https://github.com/mono/SkiaSharp/pull/4143))

Plus several CI pipeline, documentation generation, and internal tooling improvements.

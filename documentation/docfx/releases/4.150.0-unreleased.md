# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Version 4.150.0 is in active development. This cycle removes the long-deprecated .NET Interactive (Polyglot Notebooks) integration, upgrades HarfBuzz to 14.2.1, and containerizes Tizen native builds for more reproducible cross-compilation. The bulk of changes are improvements to the documentation and release automation pipeline.

## Breaking Changes

- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated Polyglot Notebooks integration has been removed. Projects using these APIs should migrate to standard SkiaSharp rendering patterns. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Platform

- **Tizen native dependencies in Docker** — Tizen native dependency builds are now containerized for isolated and reproducible cross-compilation. ❤️ Matthew Leibowitz ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

## Bug Fixes

- **Fixed iOS GL API documentation** — Resolved a moniker collision in the docs build that was silently dropping iOS GL API docs from the published site. ([#4197](https://github.com/mono/SkiaSharp/pull/4197))

Plus several CI, documentation pipeline, and release tooling improvements.

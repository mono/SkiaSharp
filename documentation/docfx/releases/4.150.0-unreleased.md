# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

<!-- Generated: 2026-06-29T15:18:22Z by claude-sonnet-4.6 -->

## Highlights

This upcoming release adds three new filter APIs — `SKColorFilter.CreateOverdraw()`, `SKImageFilter.CreateCrop()`, and `SKImageFilter.CreateEmpty()` — and merges the latest Skia chrome/m150 upstream bug fixes. Community contributor [@ramezgerges](https://github.com/ramezgerges) fixed a small-sigma raster blur regression and boosted `SKSurface` performance by caching the canvas wrapper across calls. HarfBuzz 14.2.1 ships alongside this release — see its [release notes](harfbuzzsharp/14.2.1.md) for details.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Skia m150 upstream bug fixes** — Merges upstream Skia chrome/m150 bug fixes, bringing stability and correctness improvements from the Skia engine. ([#4190](https://github.com/mono/SkiaSharp/pull/4190))

### API Surface

- **`SKColorFilter.CreateOverdraw()`** — New factory method for overdraw visualization, useful for debugging rendering overdraw in complex scenes. ([#4227](https://github.com/mono/SkiaSharp/pull/4227))
- **`SKImageFilter.CreateCrop()`** — New crop image filter that clips filter output to a specified rectangle. ([#4229](https://github.com/mono/SkiaSharp/pull/4229))
- **`SKImageFilter.CreateEmpty()`** — New filter that produces a transparent black output regardless of its input, useful as a no-op placeholder in filter chains. ([#4228](https://github.com/mono/SkiaSharp/pull/4228))

## Performance

- **`SKSurface` canvas-wrapper caching** — The `Canvas` property now caches its managed wrapper object across calls instead of allocating a new one each time, reducing GC pressure in tight render loops. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4247](https://github.com/mono/SkiaSharp/pull/4247))

## Bug Fixes

- **Small-sigma raster blur regression** — Fixes incorrect output when rendering small-sigma Gaussian blurs on the raster pipeline by defining `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS`. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4269](https://github.com/mono/SkiaSharp/pull/4269))

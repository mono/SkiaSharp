# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Version 4.150.0 completes the v4 API migration: remaining pre-v4 deprecated APIs now emit compile errors rather than warnings, and stale enum members have been trimmed from the reference assembly — a cleanup driven by community contributor [@ramezgerges](https://github.com/ramezgerges). The long-deprecated .NET Interactive (Polyglot Notebooks) integration is also removed. HarfBuzzSharp 14.2.0 ships alongside this release.

## Breaking Changes

- **Pre-v4 obsoletes promoted to compile error** — APIs deprecated during the v4 transition now produce build errors instead of warnings. Projects that haven't yet migrated away from pre-v4 call sites must do so before upgrading. Stale enum members have also been removed from the reference assembly. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive support** — The deprecated .NET Interactive (Polyglot Notebooks) integration has been removed entirely. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Dependencies

- **HarfBuzz updated to 14.2.1** — The bundled HarfBuzz native library is updated to 14.2.1. HarfBuzzSharp 14.2.0 ships alongside this SkiaSharp release. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native builds in Docker** — Tizen native dependencies are now compiled inside a Docker container, improving build reproducibility and isolation. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

### Build & Infrastructure

- **xUnit v3 migration** — The test suite has been migrated from xUnit v2 to xUnit v3, enabling better parallel execution and modern diagnostics. ([#4143](https://github.com/mono/SkiaSharp/pull/4143))
- **.NET 10 test compatibility** — Integration test harness updated to work correctly with .NET 10. ([#4221](https://github.com/mono/SkiaSharp/pull/4221))

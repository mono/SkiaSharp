# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

SkiaSharp 4.150.0 is in active development, continuing the v4 API stabilization effort. The standout change from [@ramezgerges](https://github.com/ramezgerges) promotes all remaining pre-v4 deprecated APIs to compile errors and removes obsolete enums from the reference assembly, completing the API surface cleanup for the v4 generation. The release also drops the long-deprecated .NET Interactive (Polyglot Notebooks) integration, updates HarfBuzz to 14.2.1, and containerizes Tizen native builds for improved reproducibility. [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md) ships alongside.

## Breaking Changes

- **Pre-v4 obsolete APIs are now errors** — All APIs that were marked `[Obsolete]` during the v3-to-v4 migration are now error-level, and their corresponding enums have been trimmed from the reference assembly. Code using any of these members must migrate before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive / Polyglot Notebooks support** — The deprecated .NET Interactive integration has been removed from the package. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Dependencies

- **HarfBuzz 14.2.1** — Updates the bundled HarfBuzz to 14.2.1 with the latest upstream improvements. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native build containerized** — Tizen native dependencies now build inside a Docker container for more reproducible, isolated builds across environments. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

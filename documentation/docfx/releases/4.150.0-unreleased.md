# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

This in-development release upgrades the Skia engine to milestone 150 and bumps HarfBuzz to 14.2.1. The test suite has been migrated from xUnit v2 to xUnit v3, and Tizen native dependencies now build inside a Docker container for improved reproducibility.

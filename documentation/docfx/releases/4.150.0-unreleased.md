# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Version 4.150.0 completes the v4 API cleanup by promoting the remaining pre-v4 deprecated members to errors and trimming obsolete enum values — a community-driven contribution from [@ramezgerges](https://github.com/ramezgerges). The release also ships HarfBuzz 14.2.1 alongside new HarfBuzz 14.2.0 bindings (see the linked HarfBuzzSharp page for details) and drops the long-deprecated .NET Interactive (Polyglot Notebooks) integration.

## ⚠️ Breaking Changes

- **Pre-v4 obsolete members now raise errors** — Members deprecated during the v4 migration now emit compile errors instead of warnings, and obsolete enum values have been trimmed from the reference assembly. Migrate any remaining call sites to their v4 replacements before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The long-deprecated Polyglot Notebooks integration has been removed. Applications referencing the interactive package should uninstall it. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Engine

- **HarfBuzz 14.2.1** — The bundled HarfBuzz native library has been updated to 14.2.1. New C# bindings for the HarfBuzz 14.2.0 API line ship alongside this release; see the linked HarfBuzzSharp page for the binding details. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native builds in Docker** — Tizen native dependencies are now compiled inside a Docker container, making the build environment fully reproducible and eliminating host-side dependency requirements. ❤️ Matthew Leibowitz ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

The upcoming 4.150.0 release continues v4 API cleanup and modernization. Highlights so far: the last pre-v4 obsoletes are promoted to compile errors and obsolete enum members are removed, Polyglot Notebooks (.NET Interactive) support is dropped, HarfBuzz advances to 14.2.1, and the test suite migrates to xUnit v3. Community contributor [@ramezgerges](https://github.com/ramezgerges) drove the obsoletes cleanup.

## API Surface

- **Promoted remaining pre-v4 obsoletes to errors** — The last v3-era obsolete members are now compile errors; corresponding obsolete enum values are removed from the ref assembly, completing the v4 migration guide. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated `SkiaSharp.Svg.Skottie.NetInteractive` integration is removed. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## Platform

- **Tizen native dependencies built in Docker** — Tizen native dependencies now build in a Docker container for reproducibility and isolation. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This upcoming release upgrades the Skia engine to **milestone m150** and updates HarfBuzz to 14.2.1. Tizen native dependencies now build inside a Docker container for a more reproducible environment, and the test suite has been migrated from xUnit v2 to xUnit v3.

## New Features

### Engine

- **Skia milestone m150** — Upgrades the underlying Skia graphics engine to m150, incorporating the latest upstream rendering improvements and bug fixes. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

### Text & Fonts

- **HarfBuzz 14.2.1** — Updated HarfBuzz text-shaping library to the latest upstream release. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native Docker builds** — Tizen native dependencies are now built inside a Docker container for a more consistent and reproducible build environment. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

Plus several CI, release notes tooling, and documentation improvements.

# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

This in-development release removes the long-deprecated .NET Interactive (Polyglot Notebooks) support, bumps the bundled HarfBuzz library to 14.2.1, and unifies the API diff and release-notes generation into a single agentic workflow. Tizen native dependency builds move to Docker for improved reproducibility. [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md) ships alongside this release.

## Breaking Changes

*None announced yet.*

## New Features

### Text & Fonts

- **HarfBuzz 14.2.1** — Updates the bundled HarfBuzz library to 14.2.1, incorporating the latest upstream bug fixes and improvements. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

## API Surface

- **Remove .NET Interactive support** — Drops the long-deprecated Polyglot Notebooks / .NET Interactive integration. Applications that used this feature should migrate to standard SkiaSharp drawing APIs. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## Platform

- **Tizen Docker builds** — Tizen native dependencies are now built inside a dedicated Docker container, improving reproducibility and isolating the Tizen toolchain from the host environment. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

Plus several CI and documentation automation improvements.

# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Version 4.150.0 advances the v4 API cleanup, bumps HarfBuzz to 14.2.1 (alongside the [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md) bindings), and removes the long-deprecated .NET Interactive (Polyglot Notebooks) integration. Community contributor [@ramezgerges](https://github.com/ramezgerges) promoted the remaining pre-v4 obsolete APIs to compile errors and trimmed legacy enum values from the reference assembly. The release also ships a substantial overhaul of the documentation generation and release-notes automation pipeline.

## Breaking Changes

- **Pre-v4 obsoletes promoted to errors** — APIs that were marked `[Obsolete]` during the v4 migration now emit compile errors instead of warnings, and obsolete enum values are removed from the reference assembly. Update any code still using these v3-era members before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))

- **Remove .NET Interactive (Polyglot Notebooks) support** — The deprecated integration with .NET Interactive has been removed entirely. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## Engine

- **HarfBuzz 14.2.1** — The bundled HarfBuzz text-shaping library is updated to 14.2.1 with the latest Unicode and OpenType improvements. See the [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md) page for binding-level details. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

## Platform

- **Tizen builds in Docker** — Tizen native dependencies are now built inside a Docker container, improving build reproducibility and portability across CI environments. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

## Bug Fixes

- **iOS GL API docs restored** — A moniker collision in the docs pipeline was silently dropping iOS OpenGL API documentation from the published website; this is now resolved. ([#4197](https://github.com/mono/SkiaSharp/pull/4197))

Plus extensive improvements to the documentation generation, API-diff tooling, and release automation infrastructure.

# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

Two new `SKImageFilter` factory methods — `CreateCrop()` and `CreateEmpty()` — extend the image filter composition API, making it easier to clip regions and insert no-op placeholders in filter graphs. This snapshot also picks up upstream Skia bug fixes from the Chrome m150 branch. [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md) ships alongside this release.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Upstream Skia m150 bug fixes** — Incorporates the latest upstream bug fixes from the Chrome m150 Skia branch. ([#4190](https://github.com/mono/SkiaSharp/pull/4190))

### API Surface

- **`SKImageFilter.CreateCrop()`** — New factory method to create a crop image filter, enabling rectangular region clipping within a filter pipeline. ([#4229](https://github.com/mono/SkiaSharp/pull/4229))
- **`SKImageFilter.CreateEmpty()`** — New factory method to create an empty (transparent-output) image filter, useful as a no-op placeholder in filter graphs. ([#4228](https://github.com/mono/SkiaSharp/pull/4228))

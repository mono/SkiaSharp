# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

This in-development release continues the v4 API clean-up: all remaining pre-v4 deprecated members are promoted to compile errors and trimmed from the reference assembly, and the deprecated .NET Interactive (Polyglot Notebooks) integration is fully removed. The underlying HarfBuzz native library is updated to 14.2.1 within the co-shipped [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md) package. Tizen native dependencies now build reproducibly in a Docker container, and the test suite migrates to xUnit v3. Community contributor [@ramezgerges](https://github.com/ramezgerges) drove the API clean-up.

## Breaking Changes

- **Pre-v4 obsoletes promoted to errors** — All remaining members deprecated during the 3.x → 4.x transition now produce compile errors instead of warnings; trim any remaining usages before upgrading. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Removed .NET Interactive (Polyglot Notebooks) support** — The deprecated Polyglot Notebooks integration shim is fully removed. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Text & Fonts

- **HarfBuzz 14.2.1 native library** — The underlying HarfBuzz library bundled in [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md) is updated to HarfBuzz 14.2.1, picking up upstream bug fixes. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

### Platform

- **Tizen native dependencies in Docker** — Tizen native library dependencies are now built in a Docker container for hermetic, reproducible builds. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

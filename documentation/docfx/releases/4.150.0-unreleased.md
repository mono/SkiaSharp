# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

## Highlights

This release upgrades SkiaSharp to Skia milestone m150 and updates HarfBuzz to 14.2.1. It also removes the long-deprecated .NET Interactive (Polyglot Notebooks) support and migrates the test suite from xUnit v2 to xUnit v3. Tizen native dependencies now build inside a Docker container for reproducible cross-compilation.

## New Features

### Engine

- **Skia milestone m150** — Updates the underlying Skia engine to milestone 150, bringing the latest rendering improvements and bug fixes from Google. ([#4146](https://github.com/mono/SkiaSharp/pull/4146))

### Platform

- **Tizen Docker container builds** — Tizen native dependencies now build inside a Docker container, improving build reproducibility and isolation. ([#4154](https://github.com/mono/SkiaSharp/pull/4154))

## Breaking Changes

- **Remove .NET Interactive / Polyglot Notebooks support** — The deprecated `SkiaSharp.Views.Blazor` integration for .NET Interactive has been removed. Projects using `.NET Interactive` / Polyglot Notebooks should migrate to a supported platform. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## Bug Fixes

*No user-facing bug fixes in this release.*

## Security

- **HarfBuzz 14.2.1** — Updates HarfBuzz to 14.2.1 with the latest upstream fixes. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.1](harfbuzzsharp/14.2.1.md)

## Highlights

Version 4.150.0 introduces three new image and color filter factory methods for advanced rendering composition and brings upstream Chrome/m150 engine bug fixes. Community contributor [@ramezgerges](https://github.com/ramezgerges) fixed a raster blur regression and improved `SKSurface` canvas access performance. HarfBuzzSharp 14.2.1 ships alongside this release.

## Breaking Changes

*None in this release.*

## New Features

### Engine

- **Chrome/m150 upstream bug fixes** — Merges rendering stability and correctness patches from the upstream Skia m150 branch. ([#4190](https://github.com/mono/SkiaSharp/pull/4190))

### API Surface

- **`SKColorFilter.CreateOverdraw()`** — Creates a color filter for overdraw visualisation, highlighting regions drawn multiple times; useful for GPU rendering analysis and optimisation. ([#4227](https://github.com/mono/SkiaSharp/pull/4227))
- **`SKImageFilter.CreateCrop()`** — Creates a crop filter that restricts the output region of an image filter graph to a specified rectangle. ([#4229](https://github.com/mono/SkiaSharp/pull/4229))
- **`SKImageFilter.CreateEmpty()`** — Creates a transparent-black no-op filter, useful as a placeholder node when composing image filter graphs. ([#4228](https://github.com/mono/SkiaSharp/pull/4228))

## Performance

- **`SKSurface` canvas wrapper caching** — The `SKCanvas` instance returned by `SKSurface.Canvas` is now cached across calls, avoiding repeated heap allocations on hot rendering paths. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4247](https://github.com/mono/SkiaSharp/pull/4247))

## Bug Fixes

- **Small-sigma raster blur regression** — Restores correct appearance of small-radius Gaussian blurs, which were inadvertently broken when `SK_AVOID_SLOW_RASTER_PIPELINE_BLURS` was not defined. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4269](https://github.com/mono/SkiaSharp/pull/4269))

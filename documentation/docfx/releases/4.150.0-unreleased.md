# Version 4.150.0

> **Upcoming release** · In development · Not yet available on NuGet

> **API changes** · [HarfBuzzSharp 14.2.0](harfbuzzsharp/14.2.0.md)

## Highlights

Work is in progress on 4.150.0 with a focus on finalising the v4 migration. ❤️ [@ramezgerges](https://github.com/ramezgerges) promoted all remaining pre-v4 obsolete members to compiler errors and trimmed stale enum values from the reference assembly, clearing the path for a clean v4 API surface. The release also removes the long-deprecated .NET Interactive (Polyglot Notebooks) integration and refreshes HarfBuzz to 14.2.1.

## Breaking Changes

- **Pre-v4 obsoletes promoted to errors** — All members marked `[Obsolete]` during the v3→v4 migration now fail at compile time. Code still using the old-style APIs must migrate. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Obsolete enum members removed** — Stale enum values kept only for backward compatibility have been trimmed from the reference assembly. ❤️ [@ramezgerges](https://github.com/ramezgerges) ([#4195](https://github.com/mono/SkiaSharp/pull/4195))
- **Remove .NET Interactive support** — The deprecated Polyglot Notebooks / .NET Interactive integration has been removed. ([#4186](https://github.com/mono/SkiaSharp/pull/4186))

## New Features

### Text & Fonts

- **HarfBuzz 14.2.1** — Updates the native HarfBuzz shaping engine to 14.2.1 for the latest text-shaping fixes. ([#4167](https://github.com/mono/SkiaSharp/pull/4167))

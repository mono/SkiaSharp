# Version 3.119.1

> **Bug fix & stability release** В· Released September 23, 2025 В· [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.1) В· [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.1)

## Highlights

This release fixes several crashes and rendering issues across Apple and Windows platforms, adds the `PostScriptName` property to `SKTypeface`, and updates zlib to address security vulnerabilities. Five community contributors drove the majority of user-facing changes in this release.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`PostScriptName` on `SKTypeface`** вҖ” Exposes the PostScript name of the typeface, making it easier to identify and match fonts programmatically. вқӨпёҸ [@jonlipsky](https://github.com/jonlipsky) ([#3263](https://github.com/mono/SkiaSharp/pull/3263))

## Security

- **Update zlib to v1.3.0.1+** вҖ” Bumps the bundled zlib to pick up upstream security fixes. ([#3058](https://github.com/mono/SkiaSharp/pull/3058))

## Bug Fixes

- **Avoid initializing `SKMetalView` twice** вҖ” Prevents a double-initialization that could cause rendering issues on Apple platforms. вқӨпёҸ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3256](https://github.com/mono/SkiaSharp/pull/3256))
- **`SKAutoCanvasRestore` crash on disposed canvas** вҖ” Prevents a crash when `Dispose` is called after the underlying canvas has already been disposed. вқӨпёҸ [@taublast](https://github.com/taublast) ([#3291](https://github.com/mono/SkiaSharp/pull/3291))
- **`SKXamlCanvas` opaque on Mac Catalyst** вҖ” Fixes the XAML canvas being incorrectly opaque on Mac Catalyst, which blocked content behind it from showing through. вқӨпёҸ [@spouliot](https://github.com/spouliot) ([#3236](https://github.com/mono/SkiaSharp/pull/3236))
- **Metal crash on iOS 15 and lower** вҖ” Fixes a Metal rendering crash on older iOS versions. вқӨпёҸ [@taublast](https://github.com/taublast) ([#3293](https://github.com/mono/SkiaSharp/pull/3293))
- **UTC `DateTime` conversion for `SKDocumentPdfMetadata`** вҖ” Corrects timezone handling when setting PDF document metadata dates. вқӨпёҸ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3333](https://github.com/mono/SkiaSharp/pull/3333))
- **Unpackaged WinUI loading issues** вҖ” Builds EGL as Win32 and GLESv2 as WASDK to fix native library loading failures in unpackaged WinUI apps. ([#3338](https://github.com/mono/SkiaSharp/pull/3338))
- **WinUI Interop built as `/MT`** вҖ” Switches the WinUI interop library to static CRT linking, eliminating runtime dependency issues. ([#3341](https://github.com/mono/SkiaSharp/pull/3341))

# Version 3.119.1

> **Bug fixes & new API** · Released September 23, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.1)

## Highlights

This release adds a new `PostScriptName` property on `SKTypeface`, fixes several Metal rendering issues on Apple platforms, corrects Mac Catalyst opacity behavior, resolves unpackaged WinUI loading problems on Windows, and updates zlib for security. Four community contributors — [@jeremy-visionaid](https://github.com/jeremy-visionaid), [@taublast](https://github.com/taublast), [@spouliot](https://github.com/spouliot), and [@jonlipsky](https://github.com/jonlipsky) — drove the majority of user-facing changes.

## Breaking Changes

*None in this release.*

## New Features

### API Surface

- **`PostScriptName` property on `SKTypeface`** — Exposes the PostScript name of a typeface, useful for font embedding and PDF workflows. ❤️ [@jonlipsky](https://github.com/jonlipsky) ([#3263](https://github.com/mono/SkiaSharp/pull/3263))

### Platform

- **Fixed unpackaged WinUI app loading** — Builds EGL as Win32 and GLESv2 as WASDK to resolve native library loading failures in unpackaged WinUI applications. ([#3338](https://github.com/mono/SkiaSharp/pull/3338))
- **WinUI Interop built with static runtime** — Switches the WinUI interop library to `/MT` for static C runtime linking, improving deployment reliability. ([#3341](https://github.com/mono/SkiaSharp/pull/3341))

## Security

- **Updated zlib to v1.3.0.1+** — Bumps the bundled zlib to address known vulnerabilities. ([#3058](https://github.com/mono/SkiaSharp/pull/3058))

## Bug Fixes

- **SKAutoCanvasRestore no longer crashes on double dispose** — Prevents a crash when `Dispose` is called after the underlying canvas has already been disposed. ❤️ [@taublast](https://github.com/taublast) ([#3291](https://github.com/mono/SkiaSharp/pull/3291))
- **SKXamlCanvas opacity fix on Mac Catalyst** — The canvas is no longer forced opaque on Mac Catalyst, allowing transparent backgrounds to work correctly. ❤️ [@spouliot](https://github.com/spouliot) ([#3236](https://github.com/mono/SkiaSharp/pull/3236))
- **Metal rendering fix for iOS 15 and lower** — Resolves a Metal initialization issue that caused rendering failures on older iOS versions. ❤️ [@taublast](https://github.com/taublast) ([#3293](https://github.com/mono/SkiaSharp/pull/3293))
- **Fix UTC DateTime conversion in SKDocumentPdfMetadata** — Corrects timezone handling when setting PDF metadata dates. ❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3333](https://github.com/mono/SkiaSharp/pull/3333))
- **Avoid initializing SKMetalView twice** — Prevents redundant Metal initialization that could cause unexpected behavior. ❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3256](https://github.com/mono/SkiaSharp/pull/3256))

Plus several CI, pipeline, and documentation improvements.

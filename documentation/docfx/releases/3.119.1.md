# Version 3.119.1

> **Community fixes servicing** · Released September 12, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.1)

> **API changes** · [SkiaSharp API diff](3.119.1/index.md) · [HarfBuzzSharp 8.3.1.2](harfbuzzsharp/8.3.1.2.md)

## Highlights

**3.119.1** is a community-driven servicing release for the 3.119.x line. It exposes `PostScriptName` on `SKTypeface`, hardens the Metal backend on iOS/tvOS/MacCatalyst, fixes a UTC `DateTime` conversion bug, and adds Windows-side polish for WinUI and WASDK builds. HarfBuzz 8.3.1.2 ships alongside — see [its release notes](harfbuzzsharp/8.3.1.2.md) for the binding details.

## New APIs

- **`SKTypeface.PostScriptName`** — Expose the PostScript name of a typeface so text-layout code can round-trip through PostScript identifiers (❤️ [@jonlipsky](https://github.com/jonlipsky) in [#3263](https://github.com/mono/SkiaSharp/pull/3263), [mono/skia#158](https://github.com/mono/skia/pull/158)).

## Fixes

- **UTC `DateTime` conversion** — Correct the UTC `DateTime` conversion path so timestamps survive round-trips through SkiaSharp APIs (❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) in [#3333](https://github.com/mono/SkiaSharp/pull/3333)).
- **Metal on iOS 15 and lower** — Hotfix the Metal backend so it initializes correctly on iOS 15 and earlier releases (❤️ [@taublast](https://github.com/taublast) in [#3293](https://github.com/mono/SkiaSharp/pull/3293)).
- **`SKAutoCanvasRestore` after canvas dispose** — Avoid a crash in `SKAutoCanvasRestore.Dispose()` when the underlying canvas was already disposed (❤️ [@taublast](https://github.com/taublast) in [#3291](https://github.com/mono/SkiaSharp/pull/3291)).
- **`SKMetalView` double init** — Avoid initializing `SKMetalView` twice, which would leak the previous Metal context (❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) in [#3256](https://github.com/mono/SkiaSharp/pull/3256)).
- **MacCatalyst `SKXamlCanvas` opacity** — Ensure `SKXamlCanvas` is not opaque on MacCatalyst, so it composites correctly (❤️ [@spouliot](https://github.com/spouliot) in [#3236](https://github.com/mono/SkiaSharp/pull/3236)).
- **`GRContext` dispose on Metal** — Add a regression test proving Metal-backed `GRContext.Dispose()` no longer crashes (❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) in [#3258](https://github.com/mono/SkiaSharp/pull/3258), [mono/skia#159](https://github.com/mono/skia/pull/159)).

## Platform support

- **zlib 1.3.0.1+** — Refresh the bundled zlib native dependency ([#3058](https://github.com/mono/SkiaSharp/pull/3058), [mono/skia#160](https://github.com/mono/skia/pull/160)).
- **Windows native binaries** — Build the WinUI interop layer as `/MT` ([#3341](https://github.com/mono/SkiaSharp/pull/3341)) and switch EGL to win32 with GLESv2 as WASDK so the ANGLE stack loads correctly on Windows App SDK hosts ([#3338](https://github.com/mono/SkiaSharp/pull/3338)).

## Other changes

Plus various CI, documentation, and internal tooling improvements — 1ES MicroBuild templates, Azure Pipelines refresh, and Ubuntu 22.04 build agents. ❤️ [@pjcollins](https://github.com/pjcollins) added CI triggers for primary branches on the native pipeline ([#3248](https://github.com/mono/SkiaSharp/pull/3248)).

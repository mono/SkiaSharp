# Version 3.119.1

> **Stability and platform fixes** ¬Ј Released September 23, 2025 ¬Ј [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.1) ¬Ј [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.1)

## Highlights

This release delivers key stability fixes for Metal on Apple platforms, resolves several crashes and rendering issues, and adds the new `PostScriptName` property to `SKTypeface`. Four community contributors вАФ [@jeremy-visionaid](https://github.com/jeremy-visionaid), [@taublast](https://github.com/taublast), [@spouliot](https://github.com/spouliot), and [@jonlipsky](https://github.com/jonlipsky) вАФ drove the majority of user-facing changes.

## Breaking Changes

*None in this release.*

## Bug Fixes

- **SKMetalView double initialization** вАФ Avoids initializing `SKMetalView` twice, preventing redundant setup on Apple platforms. вЭ§пЄП [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3256](https://github.com/mono/SkiaSharp/pull/3256))
- **`SKAutoCanvasRestore` crash on disposed canvas** вАФ Prevents a crash when `SKAutoCanvasRestore` is disposed after its canvas has already been disposed. вЭ§пЄП [@taublast](https://github.com/taublast) ([#3291](https://github.com/mono/SkiaSharp/pull/3291))
- **`SKXamlCanvas` opaque on Mac Catalyst** вАФ Fixes `SKXamlCanvas` incorrectly rendering as opaque on Mac Catalyst, restoring transparency support. вЭ§пЄП [@spouliot](https://github.com/spouliot) ([#3236](https://github.com/mono/SkiaSharp/pull/3236))
- **Metal crash on iOS 15 and lower** вАФ Hotfixes a Metal backend crash affecting devices running iOS 15 and earlier. вЭ§пЄП [@taublast](https://github.com/taublast) ([#3293](https://github.com/mono/SkiaSharp/pull/3293))
- **UTC `DateTime` conversion in PDF metadata** вАФ Fixes incorrect `DateTime` conversion in `SKDocumentPdfMetadata`, ensuring dates are correctly stored as UTC. вЭ§пЄП [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3333](https://github.com/mono/SkiaSharp/pull/3333))

## Security

- **zlib updated to v1.3.0.1+** вАФ Bumps the bundled zlib to address known vulnerabilities. ([#3058](https://github.com/mono/SkiaSharp/pull/3058))

## Platform

- **WinUI unpackaged app loading fix** вАФ Builds EGL as Win32 and GLESv2 as WASDK to resolve native library loading failures in unpackaged WinUI apps. ([#3338](https://github.com/mono/SkiaSharp/pull/3338))
- **WinUI Interop built as `/MT`** вАФ Switches the WinUI interop library to static CRT linking (`/MT`), eliminating runtime dependency issues. ([#3341](https://github.com/mono/SkiaSharp/pull/3341))

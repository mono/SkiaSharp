# Version 3.119.1

> **Bug fixes & quality release** ¬Ј Released September 23, 2025 ¬Ј [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.1) ¬Ј [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.1)

## Highlights

This release focuses on stability and platform fixes. Four community contributors drove most user-facing changes, including Metal rendering fixes for iOS 15, a new `SKTypeface.PostScriptName` property, and a zlib security update. WinUI loading issues in unpackaged apps are also resolved.

## Breaking Changes

*None in this release.*

- **Metal rendering on iOS 15 and earlier** вАФ Fixed Metal backend crash on devices running iOS 15 and below. вЭ§пЄП [@taublast](https://github.com/taublast) ([#3293](https://github.com/mono/SkiaSharp/pull/3293))
- **`SKAutoCanvasRestore` crash on disposed canvas** вАФ Prevents a crash when disposing `SKAutoCanvasRestore` after the canvas has already been disposed. вЭ§пЄП [@taublast](https://github.com/taublast) ([#3291](https://github.com/mono/SkiaSharp/pull/3291))
- **`SKXamlCanvas` opaque on Mac Catalyst** вАФ Fixed `SKXamlCanvas` incorrectly rendering as opaque on Mac Catalyst. вЭ§пЄП [@spouliot](https://github.com/spouliot) ([#3236](https://github.com/mono/SkiaSharp/pull/3236))
- **PDF metadata UTC DateTime conversion** вАФ Fixed incorrect UTC DateTime conversion in `SKDocumentPdfMetadata`. вЭ§пЄП [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3333](https://github.com/mono/SkiaSharp/pull/3333))
- **Unpackaged WinUI loading failures** вАФ Fixed EGL/GLESv2 build configuration to resolve native library loading issues in unpackaged WinUI apps. ([#3338](https://github.com/mono/SkiaSharp/pull/3338))

# Version 3.119.1

> **MAUI & WPF fixes** · Released September 12, 2025 · [NuGet](https://www.nuget.org/packages/SkiaSharp/3.119.1) · [GitHub Release](https://github.com/mono/SkiaSharp/releases/tag/v3.119.1)

> **API changes** · [SkiaSharp API diff](3.119.1/index.md) · [HarfBuzzSharp 8.3.1.2](harfbuzzsharp/8.3.1.2.md)

## Highlights

A servicing release on the 3.119 line focused on MAUI and WPF stability. Fixes cover WPF handle cleanup, MAUI view lifecycle on iOS, thread-safety improvements in `SKObject` disposal, a Blazor DPR regression, and iOS-build unblockers for the latest Xcode SDK. HarfBuzzSharp 8.3.1.2 ships alongside — see [its release notes](harfbuzzsharp/8.3.1.2.md).

## Breaking Changes

None in this release.

## Platforms & Views

- **WPF: release the D3D image handle on unload** — Prevents a leak of GPU resources when `SKElement` is removed from the tree. ❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3210](https://github.com/mono/SkiaSharp/pull/3210))
- **WPF: honour `Visibility.Collapsed` during animation** — Skip rendering when the element is collapsed. ❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3213](https://github.com/mono/SkiaSharp/pull/3213))
- **MAUI iOS: dispose the Metal drawable after present** — Fixes a slow memory growth pattern on iOS. ❤️ [@taublast](https://github.com/taublast) ([#3218](https://github.com/mono/SkiaSharp/pull/3218))
- **MAUI: don't invalidate a disposed handler** — Guards against a race during page teardown. ❤️ [@taublast](https://github.com/taublast) ([#3221](https://github.com/mono/SkiaSharp/pull/3221))

## Bug Fixes

- **`SKObject` disposal thread-safety** — Serialise cleanup so concurrent finalizers can't race the handle table. ❤️ [@jeremy-visionaid](https://github.com/jeremy-visionaid) ([#3225](https://github.com/mono/SkiaSharp/pull/3225))
- **Blazor: correct DPR reading on framework updates** — Restores the sharper rendering that regressed after a framework bump. ❤️ [@jonlipsky](https://github.com/jonlipsky) ([#3230](https://github.com/mono/SkiaSharp/pull/3230))
- **iOS build: unblock consumers on the latest Xcode SDK.** ❤️ [@pjcollins](https://github.com/pjcollins) ([#3234](https://github.com/mono/SkiaSharp/pull/3234))

Plus various CI, documentation, and internal tooling improvements.

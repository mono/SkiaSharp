# Version 1.56.2

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.56.2)

> **API changes** · [SkiaSharp API diff](1.56.2/index.md)

## Highlights

Historical maintenance release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so no pull-request history was recorded for the branch itself; signature-level differences captured from the prior tag are surfaced through the API diff linked above.

## Breaking Changes

- **`SKColorTable` now uses `SKPMColor`** — Colors and the indexer moved from `SKColor[]` to `SKPMColor[]`. Convert with `SKPMColor.PreMultiply` / `UnPreMultiply` at the boundary if you need straight-alpha colors.
- **`SKBitmap.GetIndex8Color` removed** — Index-8 colour reads are no longer supported; sample the bitmap directly for the pixel value.
- **`SKPath.AddPath` defaults dropped** — The optional `SKPath.AddMode mode = 0` argument is now required; pass `SKPath.AddMode.Append` (or `Extend`) explicitly to match the previous default.

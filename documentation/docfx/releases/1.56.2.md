## Highlights

Servicing release for the 1.56 line with no PRs recorded in the diff. This bump reshaped `SKColorTable` around premultiplied colors and dropped a legacy indexed-color helper.

## Breaking Changes

- **`SKBitmap.GetIndex8Color(int, int)`** — Removed; use `GetPixel` with the appropriate color type.
- **`SKColorTable.Colors` / indexer** — Now returns `SKPMColor[]` (premultiplied) instead of `SKColor[]`. Convert with `SKPMColor.PreMultiply`/`UnPreMultiply` when you need straight colors.
- **`SKImageInfo`** — Field types were tightened; recompile to pick up the new signatures.

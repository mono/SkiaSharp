# Version 2.80.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.80.0)

> **API changes** · [SkiaSharp API diff](2.80.0/index.md) · [HarfBuzzSharp 2.6.1.5](harfbuzzsharp/2.6.1.5.md)

## Highlights

SkiaSharp 2.80.0 is the first minor after 1.68.x and lifts Skia to milestone m80 ([mono/skia#64](https://github.com/mono/skia/pull/64)). Highlights include a rewritten glyph-only `SKFont` public surface backing all text rendering, first-class Vulkan support (including Win32), Alpine and Linux ARM32/ARM64 native packages, an initial WebAssembly build of `libSkiaSharp` with a full WASM test runner, and `SKPath.ToWinding` / `SKPaint.ColorF` API additions. HarfBuzzSharp 2.6.1.5 ships alongside this release; see the [linked page](harfbuzzsharp/2.6.1.5.md) for its binding details.

## Breaking Changes

The glyph-only `SKFont` rework and the reduction of deprecated native functions changed several signatures. See the [breaking-changes diff](2.80.0/SkiaSharp/SkiaSharp.breaking.md) and the [SkiaSharp.Views.Forms diff](2.80.0/SkiaSharp.Views.Forms/SkiaSharp.Views.Forms.breaking.md) for the exact list.

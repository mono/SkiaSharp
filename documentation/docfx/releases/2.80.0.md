# Version 2.80.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/2.80.0)

> **API changes** · [SkiaSharp API diff](2.80.0/index.md) · [HarfBuzzSharp 2.6.1.5](harfbuzzsharp/2.6.1.5.md)

## Highlights

Version 2.80.0 kicks off the 2.x line: it bumps upstream Skia to **m80**, adds Vulkan support on **every** platform (including Win32), makes the SKFont public API glyph-only with legacy paths delegating to it, and adds the first WASM package and Alpine binaries. Linux ARM32 (armhf) and ARM64 (aarch64) builds also arrive. Community fixes include `SKCanvas.DrawTextOnPath` from [@ziriax](https://github.com/ziriax) and a WPF sample startup-project fix from [@bender2k14](https://github.com/bender2k14). HarfBuzzSharp 2.6.1.5 co-ships — see the HarfBuzz link on the API changes line above for its per-line page.

## Breaking Changes

`SKFont` was reshaped as glyph-only in its public surface, and `SkiaSharp.Views.Forms` gained a signature refresh. See the linked SkiaSharp and Views.Forms breaking-changes files for the exact deltas.

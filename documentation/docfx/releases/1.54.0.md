# Version 1.54.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.54.0)

> **API changes** · [SkiaSharp API diff](1.54.0/index.md)

## Highlights

Historical release for the 1.x line. This tag predates SkiaSharp's modern PR-tracked release cadence, so the pull-request history for the branch is empty; the signature changes captured from the previous tag are the interesting part and are surfaced through the API diff linked above.

## Breaking Changes

- **Value-type refactor** — Several core value types (`SKRect`, `SKRectI`, `SKSize`, `SKSizeI`, `SKImageInfo`, `SKCodecOptions`) had their public mutable fields replaced with properties. Code that took `ref` on those fields, or performed direct field initialization, needs to move to the new property surface.
- **`SKImageInfo.Size` retype** — `Size` now returns an `SKSizeI` instead of `SKPointI`, matching the semantic meaning of a size.

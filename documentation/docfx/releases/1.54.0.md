## Highlights

A small type-hygiene release for 1.54 — no merged PRs land against this branch, but the geometry primitives are tightened up as part of the underlying binding refresh.

## Breaking Changes

Public fields on the shared geometry structs and codec-options struct were replaced with auto-properties, and `SKImageInfo.Size` now returns an `SKSizeI` instead of `SKPointI`. Code that named these members works unchanged; code that took `ref`/`out` on a field must switch to the property.

- **SKImageInfo** — public fields `Width`, `Height`, `ColorType`, `AlphaType` are now properties; `Size` returns `SKSizeI`.
- **SKRect / SKRectI** — public fields `Left`, `Top`, `Right`, `Bottom` are now properties.
- **SKSize / SKSizeI** — public fields `Width`, `Height` are now properties.
- **SKSurfaceProps** — public field `PixelGeometry` is now a property.
- **SKCodecOptions** — the `HasSubset`, `Subset`, `ZeroInitialized` fields were removed in favor of the corresponding option properties.

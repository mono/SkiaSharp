# Version 1.68.0

> [NuGet](https://www.nuget.org/packages/SkiaSharp/1.68.0)

> **API changes** · [SkiaSharp API diff](1.68.0/index.md) · [HarfBuzzSharp 1.4.6.2](harfbuzzsharp/1.4.6.2.md)

## Highlights

The 1.68.0 release marks the transition to statically-linked Windows binaries and lands the CI infrastructure the 1.68.x line builds on top of. HarfBuzzSharp 1.4.6.2 ships alongside — see the HarfBuzz link on the API changes line above for its per-line page.

## Breaking Changes

*None in this release.*

## Build & CI

- **Statically-linked Windows binaries** — Simplifies deployment and reduces runtime dependencies. ([#662](https://github.com/mono/SkiaSharp/pull/662))
- **CI improvements** — Groovy pipeline, PR builder, Linux CI hardening, and a few smaller CI reliability fixes. ([#615](https://github.com/mono/SkiaSharp/pull/615), [#616](https://github.com/mono/SkiaSharp/pull/616), [#635](https://github.com/mono/SkiaSharp/pull/635), [#638](https://github.com/mono/SkiaSharp/pull/638))

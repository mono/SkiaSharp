Highlights

SkiaSharp 2.88.5 fixes SKXamlCanvas on WASM and rounds SKRectI conversions the right way.

Breaking Changes

None in this release.

Bug Fixes

  • SKRectI conversion floors outwards — Converting a SKRect to SKRectI now floors outwards so the integer rectangle fully contains the source instead of clipping fractional pixels. (#2574)
  • SKXamlCanvas tolerates a missing canvas on WASM — The WASM SKXamlCanvas no longer fails when the target canvas element can't be found, matching the behaviour on other platforms. (#2564)
  • Uno canvas context is active on render — On Uno, the canvas' GL context is ensured active before rendering, avoiding intermittent blank frames. (#2560)

HarfBuzzSharp 2.8.2.5

No HarfBuzzSharp binding changes shipped in this release — it rebuilds the same HarfBuzz as the previous line.

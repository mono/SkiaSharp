Highlights

This release expands native asset coverage for HarfBuzzSharp with Linux Bionic and fuller Tizen support, while also tightening Apple packaging for device and simulator splits. It is primarily a platform distribution update for the 3.119.4 line.

Breaking Changes

None in this release.

New Features

Platform

  • Linux Bionic support — Adds native assets for Linux Bionic environments. ❤️ @4Darmygeometry (#3217)
  • Expanded Tizen support — Adds x64 and arm64 native build support and updates the Tizen support matrix for the current toolchain. (#3620 · #3619)
  • Correct Apple framework splits — Fixes tvOS native packaging so device and simulator frameworks are emitted separately, matching the iOS layout. (#3563 · #3561)

Tooling

  • Updated build baseline — Aligns the release line with the .NET 10 SDK and pinned workload versions. (#3514)

Highlights

This release expands app-model coverage with GTK views and modernizes the toolchain used to build and package HarfBuzzSharp for the 1.68.2 line.

Breaking Changes

None in this release.

New Features

App Models

  • GTK views support — Adds GTK-based view support for apps using the HarfBuzzSharp stack. (#1089)

Tooling & Packaging

  • Modernized build toolchain — Moves the release line forward to Visual Studio 2019 and .NET Core 3.0. (#1030)
  • Packaging maintenance — Rolls in native build refinements, Nano Server support, build tool updates, and other packaging improvements. (#1040, #1153, #1175)

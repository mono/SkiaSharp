Highlights

HarfBuzzSharp 14.2.1 ships with SkiaSharp 4.150.0. This release includes test infrastructure modernization, migrating the HarfBuzz test suite to xUnit v3 alongside the broader SkiaSharp test upgrade.

Breaking Changes

None in this release.

Build & Infrastructure

  • Test suite migrated to xUnit v3 — The HarfBuzz test projects have been migrated from xUnit v2 to xUnit v3, aligning with the broader SkiaSharp test suite modernization and enabling improved test parallelism and diagnostics. (#4143)

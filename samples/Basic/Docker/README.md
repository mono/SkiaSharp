# SkiaSharp Docker Sample

Demonstrates using SkiaSharp in a Docker container.

## Requirements

- [Docker](https://docs.docker.com/get-docker/)

## Running the Sample

Build and run:

```bash
cd WebApi
dotnet run --project SkiaSharpSample/SkiaSharpSample.csproj
```

Then open `http://localhost:5000/api/images` in a browser.

# SkiaSharp Samples

This folder contains sample applications demonstrating SkiaSharp usage.

## DocsSamplesApp

The main sample application for the SkiaSharp documentation. This is a .NET MAUI app demonstrating all the concepts covered in the SkiaSharp documentation.

- **Target Frameworks**: .NET 10 (Android, iOS, macOS Catalyst, Windows)
- **SkiaSharp Version**: 3.119.1

## Other Samples

Other sample folders contain additional standalone samples that may require separate migration to .NET MAUI.
